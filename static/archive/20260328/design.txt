**Overall Aesthetic and Architecture**
The website utilizes a "Cyber-Blacksmith" aesthetic, perfectly blending the brutalist, heavy textures of a medieval forge with the sleek, luminescent elements of modern cyberpunk. The background is a deep, textured obsidian (#0B0C10) that creates a parallax effect over a subtly glowing, animated core of molten server racks. Instead of traditional dust particles, the site features a custom JavaScript animation of falling "digital embers"—cascading green and blue hex codes that turn into glowing orange sparks when the user's cursor interacts with them, simulating the hammering of raw data. The typography relies on a heavy, metallic slab-serif for headers (resembling stamped iron) and a clean, monospaced font for body text (evoking terminal output). The custom cursor is styled as a glowing digital hammer, and every click produces a brief, satisfying visual burst of CSS sparks. 

**Hero Section: The Anvil**
At the top center sits "The Anvil," a prominent, elevated header section bordered by darkened steel textures and glowing orange rivets. Dead center, stamped in thick, white-hot typography, is the developer’s name: **Geva Kipper**. Beneath it, etched in a glowing cyan monospace font, sits a single sentence: "Forging scalable infrastructure and cutting-edge solutions as a Software Engineer at Google." Right below this, mimicking a constantly updating server terminal, is the dynamic subtitle that reads: "[SYSTEM NOTICE]: This digital armory is re-designed each day by AI; its form is ever-changing, forged anew every 24 hours."

**The Grid: The Tool Wall**
Below the hero section lies the main content, organized in an asymmetrical, masonry-style CSS grid. The grid consists of seven dark, iron-cast "Ingots" (boxes/sections). When resting, these boxes appear as cold, raw metal etched with faint snippets of backend code. Upon mouse hover, a "heating up" CSS transition triggers—the box's edges glow bright orange and yellow, casting an inner-shadow that makes the content inside look fully forged and illuminated. Inside each box, the links are styled heavily, resembling polished metal levers or glowing terminal buttons. 

**Box 1: GlglStats – The Compressor**
Designed to look like a pressure gauge or compression vice. It presents the Python project measuring the repetitiveness of Israeli pop-chart songs using LZ77 compression and pattern visualization. The box features two prominent glowing buttons: one leading to the live tool (https://glglstats.geekip.dev) and another styled like a cracked-open monitor panel containing an embedded or linked YouTube video of the PyData TLV presentation (https://www.youtube.com/watch?v=jrKAtOsj1Lo).

**Box 2: Ahuzat Dibuk – The Heatmap Plaque**
This box takes on the visual of a thermal scanner or radar plate. It describes the Python scraping and data visualization project that tracks Tel-Aviv parking lot occupancy. As a visual flair, an animated CSS heat-map gradient slowly shifts in the background of this specific card. It houses a polished array of external links: two "News Archive" buttons linking to the channel 13 (https://13tv.co.il/item/news/domestic/internal/parking-902956407/) and Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm) coverage, and a "Community Log" button leading to the Secret Tel-Aviv Facebook post (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).

**Box 3: 2D Part Orienting – The Blueprint**
Styled like an illuminated wireframe or blueprint set on a drafting table. It introduces the C++ algorithm project built into an open-source computational geometry library, designed to construct a "push plan" to orient polygons. A faint, glowing geometric polygon continuously rotates in the corner of this card. A sturdy, mechanical-looking button links directly to the detailed technical breakdown and code (https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/).

**Box 4: The Turing Jest – The Arena**
This box embodies a gladiatorial cage made of fiber-optic bars. It presents the LMSYS-style arena where LLMs compete in making and explaining puns for user ratings. The typography here is slightly erratic and playful, contrasting the heavy iron theme. A flashing "Enter the Arena" button directs users to the live skirmish (https://jest.geekip.dev).

**Box 5: Rhymes of Israel – The Scribe**
Themed like an automated, steam-powered printing press. It features a description of the Chrome extension that utilizes an LLM to rewrite serious news headlines into highschool yearbook-style comedic rhymes. The call-to-action is a bold button shaped like an interlocking gear, compelling the user to install the extension directly from the Chrome Web Store (https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb).

**Box 6: The Meta-Forge – Self Reference**
Designed as an intricate mirror or a recursive tunnel of glowing lines. This section winks at the user, introducing the very portfolio site they are currently browsing as a distinct project. It features a self-referencing "View Source Code / Live Site" button linking directly back to its own domain (https://geekip.dev), triggering a screen-shake or spark-burst effect when clicked.

**Box 7: The Maker’s Mark (Socials)**
Placed at the bottom or taking up a wider span across the grid, this box acts as the blacksmith's signature. It contains three heavy, metallic icons (GitHub, LinkedIn, Facebook) that illuminate in their respective brand colors when "struck" by the mouse cursor. They link firmly to respective profiles: github.com/gevak, linkedin.com/in/gevakip, and facebook.com/geva.kipper.
