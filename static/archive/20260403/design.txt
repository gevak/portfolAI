The overall aesthetic of the website is a modern, interactive interpretation of a 1980s retro text adventure game, styled as a glowing CRT terminal. The layout is constrained to a single, full-screen viewport. The background is a stark, deep charcoal-black, overlaid with subtle, animated CSS scanlines, a faint screen flicker, and a slight vignette vignette to simulate the curvature of an old monitor. Typography relies exclusively on a crisp, modern monospaced font (such as Fira Code or VT323) illuminated in a high-contrast, luminescent phosphor green or amber. Despite the retro motif, the UI must feel elegant and responsive, utilizing fluid transitions and hover states rather than clunky, outdated mechanics.

At the top of the terminal, the header functions as the game's "boot sequence." It displays the developer's name as an ASCII-art banner: **GEVA KIPPER**. Immediately beneath this is the character's class/status: "Currently weaving code and driving innovation as a Software Engineer at Google." Below this, a slowly pulsing, distinctively colored subtitle reads: "[SYSTEM NOTICE]: Reality dynamically re-compiled and redesigned each day by AI. This environment is ever-changing." 

The core interaction blends a traditional parser with modern web UX. At the bottom of the screen rests a blinking command prompt (`What would you like to do? > _`). The main content area functions as the game's narrative log. The user can either type actions on their keyboard or, for accessibility and ease, simply click on highlighted, glowing keywords embedded in the text. Clicking these keywords auto-fills the command prompt and simultaneously unpacks the corresponding section, styled as floating "inventory items" or "dungeon rooms" framed by delicate, CSS-drawn ASCII borders.

The content is divided into seven distinct, interactive narrative "boxes," presented as quests or artifacts the user encounters:
1. **The GlglStats Artifact:** Described as an ancient dictionary deciphering repetitions. It presents a Python project measuring the repetitiveness of Israeli pop-chart songs using LZ77 compression and visualizing the patterns. This box houses a glowing hyperlink to `https://glglstats.geekip.dev` and features a stylized, text-bordered embed or pop-out linking to the PyData TLV conference presentation: `https://www.youtube.com/watch?v=jrKAtOsj1Lo`.
2. **The Dibuk Radar:** A surveillance room narrative detailing 'Ahuzat Dibuk'—a Python scraping and data visualization project that generated heat-maps of Tel-Aviv parking lot occupancies. This box acts as a "dossier" featuring external "intel feeds" linking to the news broadcasts (`https://13tv.co.il/item/news/domestic/internal/parking-902956407/` and `https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm`), and the original grassroots broadcast on Facebook (`https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/`).
3. **The Geometry Engine:** Styled as a workshop where the user examines "2D Part Orienting". It describes a C++ implementation within an open-source computational geometry library, detailing an efficient algorithm designed to construct a 'push plan' for orienting given polygons. It contains a direct terminal link to `https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/`.
4. **The Turing Colosseum:** A narrative arena box presenting 'The Turing Jest'. Here, the text explains the LMSYS-like battleground where LLMs compete at crafting and explaining puns, rated by human spectators. A portal link is provided to enter the fray: `https://jest.geekip.dev`.
5. **The Rhyme Modifier:** Described as an augment or inventory mod, "Rhymes of Israel" is presented as a Chrome extension utilizing an LLM to hijack Israeli news headlines, rewriting them into comedic, high-school-yearbook-style rhymes. A clickable "Equip Mod" button links to: `https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb`.
6. **The Mirror of Inception:** A meta, self-referential room presenting the very portfolio website the user is navigating. The text muses on the paradox of a website describing its own creation, providing a self-referential loop link to `https://geekip.dev`.
7. **Player Social Stats:** The final box is the player’s character sheet. It features minimalist, CSS-drawn retro icons next to outgoing communication links to GitHub (`github.com/gevak`), LinkedIn (`linkedin.com/in/gevakip`), and Facebook (`facebook.com/geva.kipper`). 

To complete the retro-OS immersion, the footer acts as a persistent, fixed status bar spanning the bottom edge of the screen. On the left side of this bar, styled in muted grey-green text, is the metadata credit: "WORLD_BUILDER_AI: Designed by an Artificial Intelligence." On the far right is a live, ticking JavaScript stopwatch displaying the format `[ T-DELTA: HH:MM:SS ]`, calculating the exact runtime passed since the epoch timestamp: `2026-04-03 01:49:27`.
