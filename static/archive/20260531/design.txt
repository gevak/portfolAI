The website is designed as a "Glassmorphism OS," an elegant, ultra-modern desktop environment overlaying a slow-moving, fluid mesh gradient background that suggests a living, breathing system. The aesthetics combine a sleek, frosted-glass interface with smooth, physics-based drag-and-drop interactions. The layout consists of a top un-intrusive menu bar, a free-flowing desktop area containing draggable folder icons, and a floating bottom dock or taskbar. When a user clicks a folder, a sharply designed, frosted-glass "application window" smoothly scales open in the center of the screen, complete with native-looking minimize and close buttons.

The Top Menu Bar functions as the primary header, styled as a thin, translucent strip at the top edge of the screen. On the far left, in a crisp sans-serif font, is the developer’s name: Geva Kipper. Next to it reads a subtle system-status-style sentence: "Currently a Software Engineer building innovative solutions at Google." On the right side of the menu bar, displaying like a system notification, is the italicized subtitle: *Environment re-designed daily by AI; experience is ever-changing*. 

The Desktop Area holds eight unique, draggable icons that users can toss around the screen with satisfying snap-and-release physics. Each icon is a minimalist, single-color vector graphic floating above a subtle shadow, paired with a clean text label. Clicking these opens their respective "Application Windows":

1. **GlglStats**: The icon is a soundwave compressed into a ZIP folder. The opened window features a dark-mode, terminal-like aesthetic. It explains the Python project, the LZ77 compression algorithm, and the song repetition visualizations. The window includes an embedded, playable YouTube player for the PyData TLV presentation (https://www.youtube.com/watch?v=jrKAtOsj1Lo) and a sleek, glowing primary button linking to the live site at https://glglstats.geekip.dev.

2. **Ahuzat Dibuk**: The icon is a stylized parking sign merged with a heat map. The window has a high-contrast layout detailing the Python scraping and data visualization of Tel-Aviv parking lots. Below the description is a "Press Mentions" grid featuring elegant external link cards for the media coverage (https://13tv.co.il/item/news/domestic/internal/parking-902956407/ and https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm) alongside a distinct Facebook icon linking to the original secret-tel-aviv post (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).

3. **2D Part Orienting**: The icon is an abstract geometric polygon. The window design is highly technical and academic, structured like a blueprint. It outlines the C++ computational geometry algorithms and the "push plan" concepts. A bold call-to-action button, styled like a command-line input, directs the user to the documentation and code at https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/.

4. **The Turing Jest**: The icon is an AI jester hat. The window resembles a split-screen fighting game arena. It describes the LMSYS-like competition where LLMs explain puns and users rate them. At the center of the split screen is a vibrant "Enter Arena" button linking to https://jest.geekip.dev.

5. **Rhymes of Israel**: The icon is a graduation cap blended with the Google Chrome logo. The window uses a colorful, yearbook-style layout to explain the LLM-powered Chrome extension formatting Israeli news into comedic rhymes. A prominent, standard "Add to Chrome" store button links directly to https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb.

6. **Hatul Basak**: The icon is a pixel-art cat holding a puzzle piece. The window presents a playful, trivia-card interface explaining the Wikipedia-based trivia port, the AI-generated daily challenges, and user-created themes. A large, playful button acts as a portal to play and create puzzles at https://hatulbasak.geekip.dev.

7. **Portfolio**: The icon is a tiny, recursive "desktop-within-a-desktop" monitor. The window breaks the fourth wall with a meta, self-referential joke about being inside the specific UI you are currently looking at. It features a single, minimalist button linking to https://geekip.dev.

8. **System Directory (Socials)**: The icon looks like a high-tech ID badge or Rolodex. The window is a clean, hyper-minimalist profile card. It displays three large, beautifully animated social icons: GitHub (github.com/gevak), LinkedIn (linkedin.com/in/gevakip), and Facebook (facebook.com/geva.kipper) that hover and glow on interaction.

The Bottom Taskbar is a floating, frosted-glass pill at the base of the screen. On the left side, formatted like a system copyright, it reads: "Architecture and design generated by Gemini (AI)." On the right side, resembling the system clock, is a live-running red digital stopwatch ticking up in real-time (HH:MM:SS), calculating the exact time elapsed since the precise timestamp of 2026-05-31 02:48:38.
