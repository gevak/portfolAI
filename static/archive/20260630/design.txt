The core concept for this single-page portfolio is the "Hex-Driven Canvas," where color is not just aesthetic, but the primary user interface and sorting mechanism. The entire background is a deep, elegant dark mode (e.g., #0F0F0F) that acts as a stage. In the center of the viewport, upon loading, sits an oversized, interactive hexadecimal input field functioning as a dynamic color picker, surrounded by a minimalist, glowing UI halo. As the user drags the hue wheel or types a hex code (e.g., #FF4500), ambient light washes across the page, dynamically updating global CSS variables. 

The filtering mechanism relies on a "color distance" algorithm. Each of the eight project cards is assigned a hidden baseline hex color. When the user interacts with the color picker, the JavaScript calculates the color proximity of the chosen hex to each project's baseline color. Projects that closely match the selected color physically elevate on the Z-axis, scale up, increase in opacity, and emit a subtle drop-shadow glow matching the user's exact input color. Projects outside the selected color spectrum shrink, lose opacity, and blur slightly using a CSS backdrop-filter depth-of-field effect. This encourages the user to "paint" the page to discover different projects organically.

The header should be a sleek, glassmorphic panel pinned to the top left. The typography uses a modern sans-serif. It displays "Geva Kipper" in a bold weight. Beneath it, a single clean sentence: "Software Engineer at Google, currently building scalable solutions and shaping the future of web technologies." Beneath this is the glowing, italicized subtitle: "This digital canvas is re-designed daily by AI—an ever-evolving reflection of code and color." The text should subtly reflect the complementary color of whatever hex value the user currently has selected.

The layout for the eight sections should be a fluid, asymmetrical masonry grid that floats freely below the central hex picker. 
Card 1 (GlglStats) features a subtle background animation of LZ77-inspired compression audio waves. It includes a sleek typography layout describing the Python repetitive pop-chart project, with a primary button linking to https://glglstats.geekip.dev and an embedded thumbnail/play icon linking to the PyData TLV presentation on YouTube (https://www.youtube.com/watch?v=jrKAtOsj1Lo). 
Card 2 (Ahuzat Dibuk) utilizes a dark, stylized heatmap background. It describes the Python Tel-Aviv parking scraper, featuring three elegant news/social icons linking respectively to the 13tv story, the Mako article, and the original Secret Tel-Aviv Facebook post.
Card 3 (2D Part Orienting) uses wireframe geometric polygons rotating in the background. It explains the C++ open-source computational geometry push-plan algorithm, providing a prominent "View Source & Info" link leading to the CGL TAU project page.
Card 4 (The Turing Jest) features a playful split-screen motif. It outlines the LMSYS-like LLM pun arena and includes a bold "Enter the Arena" link to https://jest.geekip.dev.

Card 5 (Rhymes of Israel) mimics a stylized, retro high-school yearbook or newspaper clipping. It explains the LLM Chrome extension that converts Israeli news headlines into comedic rhymes, including an official "Available on Chrome Web Store" badge linking to the extension page. 
Card 6 (Hatul Basak) incorporates a fragmented Wikipedia puzzle-piece motif. It describes the AI-generated daily Hebrew Trivia challenges and community puzzles, featuring a direct call-to-action link to play and create at https://hatulbasak.geekip.dev. 
Card 7 (The Meta Portfolio) features a recursive, "picture-in-picture" wireframe graphic. It playfully references its own existence as a meta-project, providing a self-referential link back to https://geekip.dev. 
Card 8 (The Connect Card) acts as the anchor of the grid, stripped of project descriptions, featuring only three large, minimalist, interactive outline icons linking directly to github.com/gevak, linkedin.com/in/gevakip, and facebook.com/geva.kipper.

The footer is anchored to the bottom right of the viewport to balance the header. It should feature an ultra-minimalist, monospaced digital aesthetic. The text reads: "Concept & Execution by AI." Just below it, a live Javascript-driven stopwatch ticks endlessly, displaying the hours, minutes, and seconds elapsed since the absolute timestamp of 2026-06-30 02:43:10. The digits of this stopwatch should glow brightly, directly inheriting the exact hex color the user currently has selected in the main interactive picker, tying the site's beginning and end together through color.
