The overall visual layout is a top-down, interactive 2D RPG environment rendered via an HTML5 Canvas or CSS Grid, combining the nostalgia of classic adventure games with sleek, modern web design. Instead of retro 8-bit graphics, the art style is elegant, using high-resolution, minimalist vector graphics with smooth animations, dynamic lighting, and soft shadows. The user navigates a central avatar through the environment using either keyboard controls (WASD/Arrows) or click-to-move mechanics. Interactions with the environment open modern, frosted-glass (glassmorphism) dialogue modals featuring crisp typography, embedded media, and subtle CSS transitions. 

The user spawns in the "Town Square." Engraved elegantly into the cobblestone ground is the developer's name: **Geva Kipper**. Standing nearby is the first NPC, the "Town Guide." Upon interaction, a sleek dialogue box slides into view with the text: "Greetings! Geva is currently crafting code and magic as a Software Engineer at Google." Below this, a glowing subtitle reads: "Beware the shifting terrain: this digital realm is re-designed each day by AI and is thus ever-changing." From this plaza, cobblestone paths branch out to six distinct structures and characters.

Following the lower-right path leads to "The Bard's Tavern," where a Musician NPC stands surrounded by rolling sheet music. Speaking to the Bard opens the "GlglStats" modal. The dialogue explains this Python project, which measures the repetitiveness of Israeli pop-chart songs based on LZ77 compression and visualizes repetition patterns in famous tracks. The UI features a stylized button linking to the live site at https://glglstats.geekip.dev, and an elegantly embedded YouTube player showcasing Geva presenting the project at the PyData TLV conference (https://www.youtube.com/watch?v=jrKAtOsj1Lo).

To the north sits the "City Watchtower," occupied by a Surveyor NPC peering over a glowing map of Tel-Aviv populated by tiny, moving carriages. Interacting with the Surveyor reveals the "Ahuzat Dibuk" project. The dialogue details the Python scraping and data visualization tool that collects Tel-Aviv car parking lot occupancy statistics and generates heat-maps for residents. The glass modal includes a trio of elegant parchment-style links extending to news features on Channel 13 (https://13tv.co.il/item/news/domestic/internal/parking-902956407/), Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), and the original viral publication on the Secret Tel Aviv Facebook group (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).

In the western quarter lies "The Blacksmith's Forge," where an Artisan NPC is repeatedly hammering a glowing geometric polygon into perfect alignment. Here, the player discovers "2D Part Orienting." The modal text describes a C++ implementation within an open-source computational geometry library, utilizing an efficient algorithm for constructing a 'push plan' to orient a given polygon. A sturdy, anvil-styled hyperlinked button directs the user to the additional info and code repository at https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/.

Nearby is the "Court Jester's Arena," featuring two robotic NPCs wearing jester hats, taking turns telling jokes to an interactive crowd. The Referee NPC introduces "The Turing Jest." The dialogue explains this is an LMSYS-like arena where LLMs compete at making and explaining puns, with answers rated by real-world users. A vibrant, theatrical link invites the user to judge the competition at https://jest.geekip.dev.

Strolling along the eastern path, the user encounters the "Town Crier" NPC, holding a scroll that magically transforms into a high-school yearbook. The Crier announces the "Rhymes of Israel" project. The text explains it as a Chrome extension utilizing an LLM to rewrite Israeli news site headlines into comedic rhymes in the style of senior quotes. A sleek Chrome-branded download badge links directly to the Web Store: https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb.

Tucked away in a mystical, glitching segment of the map is "The Architect." This NPC stands over a miniaturized, holographic version of the exact map the user is currently navigating. Interacting with the Architect brings up the Meta Portfolio box: "How meta! You are currently exploring the very web project I am describing." An infinite-loop icon acts as a self-referential hyperlink pointing back to https://geekip.dev.

Finally, at the map's boundary sits the "Courier's Guild," monitored by a winged Messenger NPC. The dialogue box here serves as the social connectivity hub, offering clean, minimalist icon links for GitHub (github.com/gevak), LinkedIn (linkedin.com/in/gevakip), and Facebook (facebook.com/geva.kipper). 

Anchoring the single-page experience is a fixed, modern Heads-Up Display (HUD) spanning the bottom edge of the screen like an RPG action bar. On the left side of this dark, translucent bar, elegant typography credits the creation of the realm: "World architecture and design generated by Claude 3.5 Sonnet." On the far right is the "Temporal Glitch Clock," a live, ticking stopwatch calculating the exact hours, minutes, and seconds that have continually passed since the epoch of 2026-03-10 01:22:31, visually reinforcing the site's AI-generated, ever-evolving lore.
