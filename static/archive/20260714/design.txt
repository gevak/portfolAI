**Global Aesthetic and Theme Execution**
The website utilizes a "Cyber-Brutalist" design language. The background is a very deep obsidian blue/black (#07090F). Upon initial load, the screen is covered in chaotic, vibrating, neon-red text representing "bugs" (glitchy syntax errors, infinite loops, and literal stylized insect icons formed from ASCII). 

**The Guillotine Animation (On Load)**
A heavy, sleek, metallic executioner’s blade—rendered as a high-density, glowing horizontal laser or physical dark-metal UI element—drops rapidly from the top of the viewport to the bottom. A sharp, satisfying metallic "shing" sound effect creates a tactile feel. As the blade passes over the screen, it digitally "severs" the red bugs. The top halves of the bugs dissolve into digital dust, while the bottom halves instantly compile and snap into perfectly formatted, luminous cyan and ivory code. This code then elegantly cross-fades into the actual UI of the portfolio interface. 

**Typography and Colors**
*   **Headers & Content**: A sleek geometric sans-serif (like *Space Grotesk* or *Inter*).
*   **Accents & Code Snippets**: A modern monospace (*Fira Code*).
*   **Palette**: Deep Obsidian background, Neon Cyan (#00E5FF) for primary highlights, Slate Gray for secondary text, and Crimson (#FF204E) reserved strictly for interactive hover states (a callback to the "sliced bugs").

**Header Section**
Centered at the top, bordered by a thin, glowing container that looks like a pristine code editor window.

**The Project Portfolio Layout (8-Box Masonry Grid)**
The content is divided into an 8-card asymmetric masonry grid. Each box initially looks like a terminal window displaying raw data, but on cursor hover, the "code" slides apart (like a mini-guillotine split) to reveal the rich text and project links within. The cards feature a subtle glowing drop-shadow.

*   **Box 1: GlglStats**
    *   *Visuals*: A dynamic background graphic showing compressed algorithmic patterns (nodding to LZ77).
    *   *Content*: Title "GlglStats". Description explaining the Python tool that measures the repetitiveness of Israeli pop-chart songs and visualizes pattern structures.
    *   *Links*: A prominent cyan "Launch Project" button pointing to `https://glglstats.geekip.dev`, and a secondary "Watch PyData TLV Presentation" link styling pointing to the provided YouTube URL.

*   **Box 2: Ahuzat Dibuk**
    *   *Visuals*: A stylized, overlaid heatmap graphic.
    *   *Content*: Title "Ahuzat Dibuk". Description of the Python web-scraping and data visualization project generating heatmap statistics for Tel-Aviv parking lot occupancy.
    *   *Links*: Three distinct "Press/Media" styled text links: *Channel 13 News Feature*, *Mako News Article*, and *Original Facebook Publication*.

*   **Box 3: 2D Part Orienting**
    *   *Visuals*: Abstract, slowly rotating 2D polygons being "pushed" into alignment.
    *   *Content*: Title "2D Part Orienting". Details the C++ implementation inside an open-source computational geometry library, utilizing an algorithm to generate optimal "push plans" for polygons.
    *   *Links*: "View Code & Documentation" pointing to the provided Tel Aviv University project link.

*   **Box 4: The Turing Jest**
    *   *Visuals*: Two robotic silhouettes exchanging speech bubbles containing question marks.
    *   *Content*: Title "The Turing Jest". Explains the LMSYS-style arena where Large Language Models compete to formulate and explain puns, crowd-rated by users.
    *   *Links*: A glowing button to `https://jest.geekip.dev`.

*   **Box 5: Rhymes of Israel**
    *   *Visuals*: A high school yearbook icon morphing into a newspaper.
    *   *Content*: Title "Rhymes of Israel". Highlights the Chrome Extension leveraging LLMs to automatically rewrite Israeli news headlines into comedic high-school-yearbook-style rhymes.
    *   *Links*: "Install Chrome Extension" button pointing to the Chrome Web Store link.

*   **Box 6: Hatul Basak**
    *   *Visuals*: A pixel art cat fishing over an open Wikipedia encyclopedia.
    *   *Content*: Title "Hatul Basak". Describes the Hebrew port of catfishing.net—a Wikipedia trivia game featuring AI-generated daily challenges and user-created theme puzzles.
    *   *Links*: "Play & Create Puzzles" pointing to `https://hatulbasak.geekip.dev`.

*   **Box 7: Meta Portfolio**
    *   *Visuals*: Infinite mirror or "Droste effect" graphic of the website itself. 
    *   *Content*: Title "Portfolio Website". Playful copy acknowledging the meta-nature of presenting the exact site the user is currently viewing.
    *   *Links*: Self-referential link pointing to `https://geekip.dev`.

*   **Box 8: Social Links Console**
    *   *Visuals*: Clean, icon-based layout.
    *   *Content*: "Connect / Compile".
    *   *Links*: Large, interactive hover-icons for GitHub (`github.com/gevak`), LinkedIn (`linkedin.com/in/gevakip`), and Facebook (`facebook.com/geva.kipper`). Hovering makes the icons glitch briefly before settling.
