**Global Aesthetic & Layout**
The website utilizes a modern, sleek terminal emulator aesthetic. The background is a deep, rich charcoal (`#0d1117`) with a subtle, barely perceptible CRT scanline overlay to ground the theme without sacrificing elegance. The typography strictly uses a crisp monospaced font (like Fira Code or JetBrains Mono). The color palette relies on bright, muted neon accents—cyan for system prompts, amber for active states, and a soft off-white for body text. The layout is a single, responsive full-screen console window with clean padding. There is no scrolling; instead, content elegantly swaps in and out via snappy, modern CSS fade-and-slide transitions when commands are executed. 

**Header & Identity Section**
At the top left of the screen, the interface opens with a simulated brief command prompt: `> whoami`. 
Beneath it, the output appears in bright cyan: "Geva Kipper — Software Engineer at Google, currently focused on building scalable infrastructure and advanced software solutions." 
Directly below this, an automated system log reads in soft amber: `[SYS_NOTICE] This terminal interface is completely re-designed each day by AI and is thus ever-changing.`

**Navigation & Interaction Mechanics**
The core UI relies on extremely brief navigation. At the bottom of the screen rests a persistently blinking cursor: `> _`. The user does not need to click; typing a single character ('1' through '8') instantly collapses the current view and expands the corresponding project box in the center of the screen. Alternatively, for mobile/mouse users, the eight sections are displayed as a minimalist, responsive grid of elegant wireframe blocks. Each block displays only a single bracketed digit (e.g., `[1]`) and a one-word command (e.g., `glgl`). Hovering over a block highlights it with a glowing border, and clicking it simulates the keystroke.
