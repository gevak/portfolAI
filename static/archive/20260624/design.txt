**Global Architecture & Aesthetic Strategy**
The design utilizes a full-viewport layout mimicking a modern dark-mode IDE (e.g., VS Code or a Jupyter Notebook). The background is deep obsidian (`#111116`), with grid-line watermarks. The typography relies on a clean, monospace font (like Fira Code or JetBrains Mono) for UI elements, transitioning to an elegant sans-serif (Inter or Roboto) for rendered informational content. Color palettes focus on syntax-highlighting accents: neon pinks, cyans, and bright yellows. Instead of simple scrolling, the page is divided into vertical "script block cells," where users must interactively "Execute" scripts to reveal beautifully formatted UI cards disguised as terminal outputs.

**Header Section (The Global Docstring)**
At the pinnacle of the screen, the header is formatted as a large Python/JavaScript block comment. 
It features syntax-highlighted text:
`Developer: Geva Kipper`
`Current Thread: Software Engineer at Google, engineering scalable platforms and shaping cutting-edge web infrastructure.`
Directly beneath this, wrapped in a blinking amber warning banner, is the subtitle: 
`// WARNING: This ecosystem is re-designed daily by generative AI and functions as an ever-changing, ephemeral interface.`

**The Interactive Project Cells (The 8 Boxes)**
The page flows downward with 8 collapsed "Code Cells". Each cell features a neon green "▶ RUN" button on the left edge, followed by an executable command snippet. When the user clicks "RUN," an output console slides open smoothly underneath, deploying the respective elegant content card.

**1. Cell Command:** `analyze_pop_compression.py`
*Execution Output:* A sleek card backed by an animated audio-waveform graphic. It details the **GlglStats** project, explaining the LZ77 compression metrics used to measure repetitiveness in Israeli pop-chart songs and visualize their patterns. The output integrates a large structural hyperlink to `https://glglstats.geekip.dev` and directly seamlessly embeds the PyData TLV YouTube iframe (`https://www.youtube.com/watch?v=jrKAtOsj1Lo`) within the console frame.

**2. Cell Command:** `scrape_tlv_parking.sh`
*Execution Output:* The rendered card features a pulsating, dark-mode heatmap map grid over Tel-Aviv as its background. It describes the **Ahuzat Dibuk** Python scraping and data visualization platform generating parking occupancy heat-maps. In the style of standard output logs, three glowing terminal buttons link to the news coverage on Channel 13 (`https://13tv.co.il/item/news/domestic/internal/parking-902956407/`), Mako (`https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm`), and the original Secret Tel-Aviv Facebook publication (`https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/`).

**3. Cell Command:** `make_2d_orient.cpp`
*Execution Output:* A crisp, geometry-themed box showing a wireframe polygon slowly rotating into place. The text outlines **2D Part Orienting**, specifically the C++ implementation of the "push plan" orienting algorithm open-source library. A glowing cyan path links seamlessly to the code and additional info (`https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/`).

**4. Cell Command:** `init_lmsys_jest() --arena`
*Execution Output:* An engaging dual-pane layout visually simulating two AI text bots generating puns. The description introduces **The Turing Jest** as an LMSYS-like arena where LLMs compete at explaining puns while users rate them. A stylized, leaderboard-aesthetic button anchors to the live platform at `https://jest.geekip.dev`.

**5. Cell Command:** `install_ext_rhymes.js`
*Execution Output:* The box transforms into a playful, retro-yearbook split-screen. On one side, a serious Israeli news headline; on the other, an LLM-translated comedic rhyme. The text details the **Rhymes of Israel** Chrome extension, with a distinct "Add to Chrome" badge linking directly to the web store (`https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb`).

**6. Cell Command:** `launch_catfishing_hebrew.py`
*Execution Output:* The card reveals a slightly retro, Wikipedia-styled trivia interface overlaying an artificial puzzle layout. It presents **Hatul Basak** as the Hebrew port of catfishing.net, boasting AI-generated daily challenges and user-created themed puzzles. Bold, interactive terminal text prompts users to 'Play and Construct' linking to `https://hatulbasak.geekip.dev`.

**7. Cell Command:** `render_portfolio(self)`
*Execution Output:* Executing this script creates an elegant CSS "Droste effect" / infinite mirror recursively showing a tiny, functioning version of the entire website within the output box. The stylized text points out the meta-nature of presenting the portfolio *within* the portfolio, ending with a self-reference command link tying back to `https://geekip.dev`.

**8. Cell Command:** `ping_social_nodes()`
*Execution Output:* A minimalist interactive network graph appears. It consists of three glowing pulsing nodes connected by data-path lines. The accompanying details map out to GitHub (`github.com/gevak`), LinkedIn (`linkedin.com/in/gevakip`), and Facebook (`facebook.com/geva.kipper`). Hovering over a node displays the hyperlink string, and clicking acts as direct navigation out of the application.
