**Global Aesthetic and Layout**
The website is a modern, single-page application utilizing a dynamic CSS Grid or Masonry layout, styled as an elegant, high-end "Chef’s Cookbook." The background features a subtle, warm off-white eggshell color with a very faint, textured noise filter to mimic matte recipe paper. The typography pairs a classic, high-contrast serif font (like Playfair Display) for headings to evoke editorial cookbook elegance, with a crisp, readable sans-serif (like Inter) for body text and technical details. Interactions are tactile: hovering over elements triggers a smooth, slight lift (drop-shadow) and reveals secondary information, mimicking the flipping of a recipe card. 

**Hero Section: The Title Page**
The top of the page acts as the cookbook's cover and preface. Centered at the top in large, elegant serif typography is the name: "Chef Geva Kipper." Directly underneath, styled as a refined sub-heading, is the sentence: "Currently cooking up scalable solutions as a Software Engineer at Google." Below this, a stylized, italicized "Chef’s Note" explains the unique nature of the site: "This digital menu is re-designed fresh each day by AI, making every visit a uniquely crafted, ever-changing experience." The tech stack used to build the projects is introduced visually as "Pantry Ingredients" (e.g., Python, C++, LLMs) floating lightly in the background as watermark-style icons.

**The Recipe Cards (Content Sections)**
The core of the portfolio is an asymmetric grid of eight distinct sections styled as individual "Recipe Cards." Each card features a "Dish Name" (Project Title), "Preparation Method" (Project Description), "Ingredients" (Tech stack/tags), and "Serving Suggestions" (External Links/Videos). 

*   **Card 1: GlglStats (The Compression Confection):** Features a burnt-orange accent color. The description details the measurement of Israeli pop-chart repetitiveness based on LZ77 compression and pattern visualization. The "Serving Suggestions" section includes two interactive buttons styled as rustic labels: one linking to the live experience (https://glglstats.geekip.dev) and another opening a minimalist modal or linking outwards to the PyData TLV presentation video (https://www.youtube.com/watch?v=jrKAtOsj1Lo).
*   **Card 2: Ahuzat Dibuk (The Traffic Jam Tart):** Styled with a muted sage-green accent. The description explains the Python scraping and data visualization for Tel-Aviv parking lot occupancy and resident heat-maps. The links are arranged as "Press Mentions & Reviews," featuring sleek, icon-based text links to news coverage on Channel 13 (https://13tv.co.il/item/news/domestic/internal/parking-902956407/), Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), and the original Facebook publication in Secret Tel Aviv (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).
*   **Card 3: 2D Part Orienting (Geometric Gastronomy):** Given a stark, minimalistic monochrome aesthetic reflecting its C++ nature. The text describes an efficient algorithm for constructing a 'push plan' to orient polygons, implemented in an open-source computational geometry library. The link to additional info and code (https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/) is styled as a "Secret Formula" button.
*   **Card 4: The Turing Jest (A Punny Pastry):** Uses a playful, slightly brighter yellow accent. Described as an LMSYS-like arena where LLMs compete in making and explaining puns, rated by users. The link to the arena (https://jest.geekip.dev) is styled as a call-to-action inviting the user to "Taste and Rate."
*   **Card 5: Rhymes of Israel (The Yearbook Soufflé):** Features a subtle nostalgic textbook motif superimposed on the recipe card. The text explains the Chrome extension that utilizes an LLM to rewrite Israeli news headlines into comedic, high-school yearbook-style rhymes. The "Install" link (https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb) is styled like an "Add to Pantry" button.
*   **Card 6: Hatul Basak (The Trivia Tagine):** Card features subtle Wikipedia-puzzle-piece watermarks. Described as a Hebrew port of catfishing.net, using AI to generate daily trivia challenges and user-created themed puzzles. The link to play and create (https://hatulbasak.geekip.dev) is presented as a bold, interactive plate icon.
*   **Card 7: The Sourdough Starter (Meta-Portfolio):** A self-referential, visually distinct card with a "perpetual loop" visual motif. The description humorously points out the meta nature of the very site the user is viewing. It contains a self-reference link back to itself (https://geekip.dev). 
*   **Card 8: The Chef's Pantry (Social Links):** A horizontal "shelf" box spanning the bottom of the grid rather than a traditional card. It visually mimics lined-up spice jars or chef's tools. Hovering over each item reveals the link: GitHub (github.com/gevak), LinkedIn (linkedin.com/in/gevakip), and Facebook (facebook.com/geva.kipper).

**The Footer (The Oven Timer)**
The bottom of the page grounds the single-page layout with a dark charcoal footer, representing an oven interface or a kitchen counter. In crisp, monospace HTML/JS text glowing like a digital display, credit is given: "Baked with artificial intelligence by Gemini [or Claude/ChatGPT, insert respective AI identity here]." Next to this is a live, ticking JS stopwatch formatted as `HH:MM:SS`, accompanied by the text: "Time left fermenting on the counter since 2026-06-19 03:40:12." The timer visually updates every second, adding an engaging, dynamic final touch to the modern layout.
