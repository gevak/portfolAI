**Global Aesthetic and Layout**
The website utilizes a "Cyberpunk High-Stakes Casino" theme, viewed from a slight top-down isometric perspective. The background is a rich, dark felt-green gradient overlaid with a subtle, moving digital grid. The site operates as a single, immersive fullscreen dashboard. Glassmorphism is used heavily, with translucent, smokey-black glass panels accented by glowing neon borders in gold, purple, and cyan. The typography is modern and sleek—sans-serif for body text, and an elegant, metallic serif for headings to mimic high-end casino branding.

**The Core Interaction: The AI Poker Game**
The center of the screen features an interactive Texas Hold'em interface. A glowing AI avatar (styled as a holographic dealer) sits across from the user. Rather than simply scrolling to view projects, the user must click a "Deal Hand" button to play a quick, automated poker hand against the AI using sleek, animated HTML/JS playing cards. Placed around the central betting area are seven facedown cards, securely locked with glowing neon padlock icons. Every time the user wins a hand against the AI, one of these cards flips over, breaks its padlock with a satisfying particle animation, and expands into a detailed "Box" representing a project or social link. 

**The Unlocked Boxes (The Seven Cards)**
When unlocked, the face of each card expands into a hovering, interactive glass panel containing the specific content. 

**1. GlglStats (The Ace of Spades)**
The visual theme features soundwaves morphing into binary code sequences. It details a Python project measuring the repetitiveness of Israeli pop-chart songs based on LZ77 compression, complete with animated data-visualization repetition patterns. 
* Interactive Elements: Two glowing casino-chip buttons. One links to the live site (`https://glglstats.geekip.dev`), and the other, designed with a YouTube play icon, links to a PyData TLV presentation (`https://www.youtube.com/watch?v=jrKAtOsj1Lo`).

**2. Ahuzat Dibuk (The King of Hearts)**
The background of this card displays a slowly pulsing, heat-mapped grid of Tel-Aviv streets. It describes a Python scraping and data visualization project that collects statistics on parking lot occupancy for residents. 
* Interactive Elements: Three "Press Pass" styled badges linking to the project's media coverage on Channel 13 (`https://13tv.co.il/item/news/domestic/internal/parking-902956407/`), Mako (`https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm`), and the original viral Facebook post (`https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/`).

**3. 2D Part Orienting (The Queen of Diamonds)**
This card features a blueprint visual style with slowly rotating geometric polygons. It presents a C++ implementation in an open-source computational geometry library of an efficient algorithm constructing a "push plan" to orient a given polygon. 
* Interactive Elements: A button styled like a university seal linking to the code and additional info at TAU CGL (`https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/`).

**4. The Turing Jest (The Jack of Clubs)**
The artwork features a high-tech court jester split down the middle into two dueling robotic halves. It presents an LMSYS-like arena where LLMs compete at making and explaining puns, rated by users. 
* Interactive Elements: A glowing, neon "Enter Arena" button leading to the site (`https://jest.geekip.dev`).

**5. Rhymes of Israel (The Ten of Spades)**
The design merges a vintage highschool yearbook page with a modern news ticker. It presents a Chrome extension that uses an LLM to rewrite Israeli news site headlines into comedic rhymes. 
* Interactive Elements: A prominent "Install Extension" button featuring the Chrome Web Store logo, linking to the download page (`https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb`).

**6. The Meta Portfolio (The Joker Card)**
This box carries a deeply meta, recursive theme. The background is an endless mirror-effect animation of the poker table itself. It acknowledges the website project the user is currently navigating. 
* Interactive Elements: A self-referential infinity-loop icon linking back to (`https://geekip.dev`).

**7. Socials Array (The Royal Flush)**
The final unlocked box acts as the developer's main contact and social hub. It is styled as a velvet-lined jewelry box or luxury card case. 
* Interactive Elements: Three intricately detailed, heavy metallic tokens acting as links. The first token links to GitHub (`github.com/gevak`), the second to LinkedIn (`linkedin.com/in/gevakip`), and the third to Facebook (`facebook.com/geva.kipper`).
