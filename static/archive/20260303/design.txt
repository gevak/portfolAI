**Overall Visual Concept and Atmosphere**
The website employs a sleek, modern, minimalist aesthetic—an expansive, soft off-white canvas (#F4F4F0) that feels both organic and high-tech. The typography pairs an elegant sans-serif (like Inter or Roobert) for primary reading with a sharp monospaced font (like Fira Code or JetBrains Mono) for technical details and links. The central visual and navigational mechanic is an animated SVG "ant trail." Using scroll-linked animations, tiny, stylized black dots (ants) march along an invisible looping bezier curve that weaves back and forth down the single page. As the user scrolls, the ants rush forward to gather around subtle, seemingly blank geometric shapes (representing digital "sugar cubes" or "leaves"). Only when the ants swarm these hidden nodes—or when the user hovers over them—do the shapes unfold into sleek, glassmorphic project boxes containing the hidden text and links.

**Header Section**
At the top center of the page, clean and bold typography introduces the developer: **Geva Kipper**. 
Directly beneath, a crisp subtitle reads: *Software Engineer shaping massive-scale systems and pushing the boundaries of search technology at Google.*
Separated by a hairline divider, a secondary, monospaced subtitle states: *“This digital landscape is uniquely re-designed each day by Artificial Intelligence—an ever-changing portfolio.”* 
At the bottom of the header, a single “scout ant” animates downwards, pulsing subtly to prompt the user to scroll and begin the journey.

**The Project Nodes (Boxes 1-6)**
As the ant trail meanders down the page, it stops at six distinct, staggered waypoints. Each waypoint initially appears as a minimalist icon (a data point, a geometric cluster, a stylized text node). Upon cursor hover or scroll intersection, the node expands via a fluid CSS transition into an elegant content box:

*   **Box 1 (GlglStats):** The ants form the shape of an audio wave that compresses into a tight block. The box reveals "GlglStats," a Python-based project utilizing LZ77 compression to measure and visualize the repetitiveness of Israeli pop-chart songs. 
    *Hidden Links Revealed:* A dynamic button to the interactive site (`https://glglstats.geekip.dev`) and an embedded link to the PyData TLV conference presentation (`https://www.youtube.com/watch?v=jrKAtOsj1Lo`).

*   **Box 2 (Ahuzat Dibuk):** The ants scatter to form a glowing heatmap grid. The box presents "Ahuzat Dibuk," a Python scraping and data visualization tool generating occupancy heat-maps of Tel-Aviv parking lots for local residents. 
    *Hidden Links Revealed:* "In the News" links for Channel 13 (`https://13tv.co.il/item/news/domestic/internal/parking-902956407/`) and Mako (`https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm`), alongside a link to the original viral Facebook publication (`https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/`).

*   **Box 3 (2D Part Orienting):** The trail outlines a chaotic polygon before smoothly rotating it into perfect alignment. The box details "2D Part Orienting," a complex C++ implementation within an open-source computational geometry library used to construct efficient "push plans" for polygon orientation.
    *Hidden Link Revealed:* A technical link to the project info and source code (`https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/`).

*   **Box 4 (The Turing Jest):** The ants arrange into two jester masks facing one another. This box unveils "The Turing Jest," a competitive LMSYS-style arena where Large Language Models battle by making and explaining puns, rated by human users.
    *Hidden Link Revealed:* A direct link to enter the arena (`https://jest.geekip.dev`).

*   **Box 5 (Rhymes of Israel):** The ants march in a rhythmic, stanza-like formation. The section introduces "Rhymes of Israel," an LLM-powered Chrome extension transforming mundane Israeli news headlines into comedic, high-school yearbook-style rhymes.
    *Hidden Link Revealed:* A sleek install button linking to the Chrome Web Store (`https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb`).

*   **Box 6 (The Meta Portfolio):** The trail forms a miniature, recursive outline of the website itself. This box cheekily presents the current portfolio website project—a meta-reference acknowledging the very architecture the user is navigating.
    *Hidden Link Revealed:* A self-referential loop linking back to the domain (`https://geekip.dev`).

**The Anthill Footer (Box 7 & Credits)**
At the bottom of the page, the SVG ant trail funnels into a sleek, minimalist geometric "anthill" (a dark, semi-transparent dome). As the user scrolls to the absolute bottom, the dome expands into Box 7, revealing bold, interactive typography linking to the developer's digital footprint: 
*   GitHub (`github.com/gevak`)
*   LinkedIn (`linkedin.com/in/gevakip`)
*   Facebook (`facebook.com/geva.kipper`)

Beneath the anthill, the layout shifts to deeply technical monospaced text for the footer. It reads: 
*Design entirely hallucinated and architected by an AI model.* 
Below this credit lies a live, ticking JavaScript stopwatch displaying the precise elapsed time (Hours : Minutes : Seconds) since the designated temporal anchor: **2026-03-03 01:29:03**, flashing softly like the pulse of a terminal cursor.
