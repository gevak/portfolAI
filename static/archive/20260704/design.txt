**Overall Aesthetic and Architecture**
The website is a single-page, cinematic experience designed as a digital, futuristic "locked vault" or interactive escape room. The visual theme utilizes a dark, elegant color palette—deep obsidian backgrounds with glassmorphism effects and shifting neon accents (cyan, amber, and ultraviolet). The layout does not resemble a traditional portfolio; rather, it appears as an encrypted command terminal. As a nod to the "AI daily redesign" theme, the background features subtle, procedurally generated CSS geometric patterns that slowly morph, giving the illusion of an ever-changing, living system. 

**The Header / Command Terminal**
At the center of the viewport upon loading is a glowing, sleek command console. It displays the developer's name, **Geva Kipper**, in a bold, monospaced modern typography (like Space Grotesk). Below it, a sleekly styled sentence reads: "Software Engineer at Google, engineering scalable, high-impact systems." 
Beneath this bio is a flickering, glitch-text subtitle that stabilizes into: *"SYSTEM ALARM: Grid architecture and UI morphology are re-designed daily by AI."* 
Surrounding this central console are eight metallic, locked "containment units" (representing the 8 sections). The user cannot simply read the portfolio; they must interact with these units to "unlock" the information inside via JavaScript-driven micro-puzzles.

**The Interactive Project Modules (The 8 Locked Boxes)**
Each of the 8 boxes begins as a minimal, cryptic square with a puzzle hint. Once the user solves the micro-interaction, the box triggers a smooth CSS 3D flip-and-expand animation, transforming into an elegant project card with the detailed text and glowing hyperlink buttons.

*   **Box 1 (GlglStats):** The lock screen shows a short, repeating pattern of colored dots. The user must click the dots to compress the sequence (a nod to LZ77). Upon unlocking, it reveals a neon audio-visualizer background presenting the 'GlglStats' Python project. It explains the repetitiveness measurement of Israeli pop-chart songs and includes a sleek button to the site (https://glglstats.geekip.dev) and a prominent icon linking to the PyData TLV presentation (https://www.youtube.com/watch?v=jrKAtOsj1Lo).
*   **Box 2 (Ahuzat Dibuk):** The lock screen displays a blank grid map of Tel Aviv. By hovering over the grid, the user finds a glowing red "heat-map" hotspot. Clicking the hotspot unlocks the box. Inside, a thermal-gradient aesthetic presents the Python scraping and data visualization project for car parking occupancy. Links are stylized as news-ticker buttons for Channel 13 (https://13tv.co.il/item/news/domestic/internal/parking-902956407/), Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), and the Secret Tel Aviv Facebook post (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).
*   **Box 3 (2D Part Orienting):** The lock screen features a scrambled 2D polygon. The user must drag and rotate it into a matching silhouette to unlock it. The revealed card utilizes a blueprint-style background, presenting the C++ implementation of the 'push plan' orientation algorithm. A button directs to the computational geometry library (https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/).
*   **Box 4 (The Turing Jest):** The lock screen presents a split A/B choice with two blurred punchlines. Clicking either "judges" the joke, unlocking the panel. The interior mimics a retro arcade versus-screen, presenting the LMSYS-like LLM pun arena, complete with a primary link to play (https://jest.geekip.dev).
*   **Box 5 (Rhymes of Israel):** The lock screen displays a scrambled, serious news headline. A draggable slider "translates" the text, causing it to magically morph into a comedic rhyme. Unlocking this reveals a playful, yearbook-style card presenting the Chrome extension, alongside a direct download button (https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb).
*   **Box 6 (Hatul Basak):** The lock screen shows a classic Wikipedia-style link chain missing a middle node (e.g., Cat -> ? -> Internet). Clicking the blank space auto-fills the answer and unlocks the box. The interior reveals a trivia-themed layout presenting the Hebrew port of catfishing.net, emphasizing the AI daily challenges and user-created puzzles. A button takes users to play (https://hatulbasak.geekip.dev).
*   **Box 7 (The Meta Portfolio):** The lock screen features a visual recursive loop—an icon of the box itself. The user must double-click it, which creates an infinite CSS zoom effect until the project card materializes. The text playfully breaks the fourth wall, presenting the very portfolio website they are currently hacking. The self-reference link points to (https://geekip.dev).
*   **Box 8 (Social Links):** The lock screen acts as a digital combination safe. The user swipes three vertical dials to align three connection nodes. Once aligned, the box shutters open to reveal an elegant control panel with glowing social icons and links for GitHub (github.com/gevak), LinkedIn (linkedin.com/in/gevakip), and Facebook (facebook.com/geva.kipper).

**The Escape / Footer Area**
As the user unlocks all 8 modules, a dramatic UI change occurs: the dark ambient lighting of the page brightens, and an animated "SYSTEM UNLOCKED / ESCAPE ROUTE OPEN" message scrolls across the bottom of the viewport. This reveals the footer, which is styled like a flight-recorder timestamp. The footer features crisp, monospace typography giving clear credit: "Escape Room Architecture & UI Generated by AI Model." Next to this, a dynamic JavaScript-powered digital stopwatch continually counts upward in real-time (Hours : Minutes : Seconds : Milliseconds), hardcoded to calculate the exact time elapsed since the precise timestamp of 2026-07-04 02:12:37, tying the entire futuristic, ever-shifting narrative together.
