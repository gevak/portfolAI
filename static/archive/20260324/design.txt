**Overall Concept & Aesthetic:**
The website is a single-page WebGL/Three.js experience designed as an interactive, first-person 3D "cyber-gallery" maze. To maintain elegance and prevent user frustration, the maze walls are constructed from frosted, semi-transparent glass (glassmorphism), allowing glowing hints of colors from other rooms to bleed through and guide the user. The dark-mode environment is illuminated by minimalist, dynamic neon strip lights embedded in the floor and ceiling. The interface combines sleek, brutalist typography with high-tech HUD (Heads-Up Display) elements. 

**Navigation & HUD (Heads-Up Display):**
Users navigate using standard keyboard controls (WASD/Arrows) and mouse, with a point-and-clic/tap-to-walk fallback for mobile devices. Overlaying the 3D WebGL canvas is a static, minimalist 2D UI matrix.
At the absolute bottom of the screen, a fixed footer displays the text: *"Architecture generated by Artificial Intelligence."* Next to it is a sleek, monospace digital stopwatch ticking upwards in real-time (HH:MM:SS), calculating the exact time elapsed since **2026-03-24 01:27:28**.

**The Spawn Point (The Foyer):**
The user spawns in a circular, high-ceilinged room. In the center floats a slowly rotating, crisp typographic hologram. 
The primary text reads: **Geva Kipper**. 
Beneath it, a glowing sentence states: *"Currently engineering highly scalable, impact-driven software solutions at Google."*
Encircling the floor around this hologram is a flickering subtitle: *"Warning: Topography is re-designed daily by AI. This spatial layout is ever-changing."* 
A singular path of pulsing floor lights guides the user into the labyrinth to discover the project nodes.

**The Maze Exhibits (The 7 Content Nodes):**
Instead of traditional web "boxes," the projects are distinct alcoves or chambers within the maze. When a user steps into a chamber, camera focus locks slightly, and elegant 2D information cards slide into the screen from the sides, featuring interactive buttons for the links.

*   **Exhibit 1: GlglStats (LZ77 Music Project)**
    *Visuals:* This room features a massive, glowing 3D audio waveform that pulses mimicking pop music rhythms. 
    *UI Panel:* Describes the Python project measuring Israeli pop-chart repetitiveness via LZ77 compression. It features two glowing portal-style buttons: one pointing to the live site (https://glglstats.geekip.dev) and an embedded, stylized video player linking to the PyData TLV presentation (https://www.youtube.com/watch?v=jrKAtOsj1Lo).

*   **Exhibit 2: Ahuzat Dibuk (Tel-Aviv Parking)**
    *Visuals:* The floor of this chamber is a literal, glowing 3D heatmap projection of a city grid.
    *UI Panel:* Details the Python scraping and data visualization project for parking occupancy. Beside the description are sleek metallic icons linking to the press coverage on Channel 13 (https://13tv.co.il/item/news/domestic/internal/parking-902956407/) and Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), plus a community icon linking to the Secret Tel-Aviv Facebook post.

*   **Exhibit 3: 2D Part Orienting (Computational Geometry)**
    *Visuals:* A minimalist, brightly lit room featuring a complex 3D wireframe polygon suspended in mid-air, slowly being manipulated by robotic "pushing" lines.
    *UI Panel:* Explains the C++ implementation for constructing push plans. It holds an academic-styled button linking to the TAU open-source repository (https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/).

*   **Exhibit 4: The Turing Jest (LLM Comedy Arena)**
    *Visuals:* Designed like a futuristic gladiatorial arena in miniature. Two glowing server racks, colored red and blue, emit sparks as they face off against each other.
    *UI Panel:* Describes the LMSYS-like arena where LLMs compete in making and explaining puns. A prominent, playful button links to the battlefield (https://jest.geekip.dev).

*   **Exhibit 5: Rhymes of Israel (Chrome Extension)**
    *Visuals:* A chaotic cyclone of floating text representing news headlines, which funnel into an oversized, stylized high-school yearbook that continually flips its pages.
    *UI Panel:* Outlines the LLM-powered comedic headline rewriter. Contains a prominent "Install Extension" button integrating Google's Chrome Web Store design language (https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb).

*   **Exhibit 6: The Meta Portfolio Lab**
    *Visuals:* On a solitary pedestal sits a perfect, miniature 3D model of the exact maze the user is currently navigating. A tiny green dot represents the user staring at themselves.
    *UI Panel:* Acknowledges the meta-nature of the portfolio itself. Features a self-referential loop button that simply routes back to the domain (https://geekip.dev).

*   **Exhibit 7: The Exit Terminal (Socials)**
    *Visuals:* The end of the maze. The walls open up to a starlit digital void. A sleek command console sits at the edge.
    *UI Panel:* Features oversized, animated, glowing logos that react to the user's cursor. They serve as the definitive exit points: GitHub (github.com/gevak), LinkedIn (linkedin.com/in/gevakip), and Facebook (facebook.com/geva.kipper).
