**Overall Aesthetic and Global Styling**
The website utilizes a deep, mystical "midnight" aesthetic using a dark palette—obsidian black and deep indigo (#0a0a1a) backgrounds, accented with iridescent antique gold (#D4AF37) and ethereal violet. A subtle, custom HTML/CSS particle effect runs in the background, simulating slowly falling stardust or softly glowing embers to create atmospheric depth. Typography relies on a sophisticated serif typeface (like Playfair Display or Cinzel) for headings to evoke an ancient grimoire or tarot deck, paired with a highly legible, modern sans-serif (like Inter) for body copy. The layout is a single-viewport "reading table," where the content centralizes around an interactive spread of cards.

**Header and Introduction**
Centered at the very top of the spread is the developer’s identity. The name "Geva Kipper" is rendered in a large, elegant gold serif font that catches a subtle CSS `text-shadow` glow. Directly beneath it is a clean, single-row subtitle: "Software Engineer currently building infrastructure and crafting code at Google." Below this sits a secondary subtitle, italicized and with a slightly muted opacity (grey-violet): "This digital reading is ever-shifting; this website is re-designed each day by AI." The header is separated from the main content by a delicate, responsive golden line that fades out at the edges.

**The Tarot Spread (Main Content)**
The core interface employs a responsive CSS Grid to lay out eight distinct "Cards" in two rows of four (or a single scrollable arc on mobile devices). Upon page load, JavaScript triggers a cascading deal animation, sliding the cards into place face down. The card backs feature an intricate, gold-foil geometric pattern reminiscent of traditional Rider-Waite tarot. The interactivity centers on CSS 3D transforms (`rotateY`) and `perspective`. When the user hovers over a card, it slightly levitates (`translateY`) and emits a golden `box-shadow` aura. Upon clicking, the card fluidly flips 180 degrees to reveal the "Major Arcana" representing a specific project or section.
