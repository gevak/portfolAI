**Overall Aesthetic and Layout Strategy**
The website utilizes a "Minimalist Enigma" aesthetic—a dark mode, single-page application built on a deep obsidian canvas (#0B0C10) with glowing cyan and copper neon accents (#66FCF1 and #C5C6C7). The font pairing uses a sleek, geometric sans-serif (like Montserrat or Inter) for headings, combined with a monospace terminal font (like Fira Code) for subtitles, links, and data. The page layout is not a standard grid; instead, it is styled like a high-tech control panel or a digital schematic diagram. 

**The Escape Room Mechanic**
The central UX thesis is that the user is locked out of Geva’s direct contact information. At the bottom of the screen sits a large, frosted-glass "Vault" (Section 8) wrapped in heavy digital padlock iconography and a progress bar reading "0/7 Security Modules Bypassed." To unlock it, the user must engage with the seven project sections scattered across the page. Each project card contains a distinct interaction (a hover state, a mini geometry puzzle, or clicking a specific link) that triggers a CSS animation. When a project is "solved," a glowing neon circuit line shoots out from the project card, tracing a path across the dark background to connect to the Vault at the bottom, lighting up one of seven indicator nodes. 

**Header Section: The Briefing**
At the top center, a clean, bold title reads **Geva Kipper**. Beneath it, an elegant, stark sentence in a slightly muted silver reads: "Software Engineer currently solving complex infrastructure challenges at Google." Below this, mimicking a system boot sequence in monospace font, is the subtitle: *"Notice: This interface is generated and re-designed daily by Artificial Intelligence. The layout you see is ever-changing."* This text should feature a subtle, intermittent CSS glitch or typewriter effect.

**The Seven Puzzle Modules (Projects 1-7)**
These sections are designed as frosted-glass cards (Neumorphism mixed with Cyberpunk) floating on the dark background. Each card must be interacted with to unlock its corresponding circuit line.

*   **Module 1: GlglStats**
    Designed with a background graphic mimicking a compressed audio waveform. Text explains it is a Python project measuring repetitiveness in Israeli pop-chart songs using LZ77 compression and visualizing the patterns. 
    *Unlock action:* Hovering over the audio waveform aligns the frequencies to reveal the unlock.
    *Links:* A stylized button to the live site (https://glglstats.geekip.dev) and a secondary link to the PyData TLV presentation (https://www.youtube.com/watch?v=jrKAtOsj1Lo).

*   **Module 2: Ahuzat Dibuk**
    Styled with a miniature, faint heat-map grid in the background. The text details the Python scraping and data visualization project tracking Tel-Aviv parking lot occupancy for residents. 
    *Unlock action:* Clicking a specific "hot" sector on the background heat-map.
    *Links:* Broadcast links to Channel 13 (https://13tv.co.il/item/news/domestic/internal/parking-902956407/) and Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), plus the Secret Tel Aviv Facebook publication (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).

*   **Module 3: 2D Part Orienting**
    Features a wireframe polygon graphic. The description explains the C++ open-source computational geometry library implementation for constructing "push plans" to orient polygons. 
    *Unlock action:* The user must drag or rotate the polygon via mouse/touch to fit perfectly into an adjacent silhouette. 
    *Links:* Direct access to the code and info (https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/).

*   **Module 4: The Turing Jest**
    Designed as a split-screen battle arena terminal. The text presents an LMSYS-like arena where LLMs compete at making/explaining puns, rated by users. 
    *Unlock action:* Toggling a "Vote" button between two AI icons.
    *Links:* Arena entry (https://jest.geekip.dev).

*   **Module 5: Rhymes of Israel**
    Visualized with a glowing puzzle piece icon (representing a Chrome extension) overlapping a high-school yearbook aesthetic. The text describes an LLM extension rewriting Israeli news headlines into comedic rhymes. 
    *Unlock action:* Dragging the puzzle piece into an empty extension slot on the card.
    *Links:* Chrome Web Store installation (https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb).

*   **Module 6: Hatul Basak**
    Styled with a dynamic node-graph representing Wikipedia links. Explains the Hebrew port of catfishing.net—an AI-generated daily Trivia game utilizing Wikipedia routing, featuring user-created puzzles. 
    *Unlock action:* Tracing a path between two nodes on the card graphic.
    *Links:* Play game and create puzzles (https://hatulbasak.geekip.dev).

*   **Module 7: Meta Portfolio Project**
    Designed with an infinity mirror/recursive web frame effect. The text points out the meta nature of the project—the very site the user is currently interacting with. 
    *Unlock action:* Clicking a recursive "zoom in" icon until the core node is reached.
    *Links:* Self-reference link to the root (https://geekip.dev).
