**Overall Layout and Aesthetic**
The website is a fixed-viewport, single-page application built with modern CSS and JavaScript, taking up exactly 100vh and 100vw. The visual theme is a sleek, retro-futuristic "Cyber-Vending Machine," utilizing glassmorphism (frosted glass effects) and subtle neon accents against a dark, moody background (e.g., twilight purple and deep charcoal). The user is looking directly at the front of this life-sized digital vending machine. 

**The Header (Vending Machine Marquee)**
At the very top of the machine is a glowing LED-style digital marquee. 
*   **Main Text:** "Geva Kipper — Software Engineer, currently building at Google."
*   **Subtitle:** Below the main text, a smaller amber scrolling ticker reads: "Notice: This environment is re-designed each day by AI and is thus ever-changing."

**The Display Case (Grid Interface)**
Behind a glossy, slightly reflective "glass" pane (achieved with CSS `backdrop-filter`), the projects are arranged on two mental shelves in a 4x2 grid (Slots A1-A4 and B1-B4). Instead of standard buttons, each project is visually represented as a distinct "packaged good" resting on a metal coil:
1.  **GlglStats (A1):** Designed as a holographic cassette tape.
2.  **Ahuzat Dibuk (A2):** Designed as a brightly colored parking ticket or miniature traffic cone.
3.  **2D Part Orienting (A3):** Designed as a glowing, shifting geometric puzzle box.
4.  **The Turing Jest (A4):** Designed as a robotic comedy/tragedy mask.
5.  **Rhymes of Israel (B1):** Designed as a miniature rolled-up newspaper bound in a yearbook-style ribbon.
6.  **Hatul Basak (B2):** Designed as a futuristic tin of premium cat food wrapped in trivia question marks.
7.  **Portfolio Box (B3):** Designed as a tiny, recursive miniature version of the vending machine itself.
8.  **Socials (B4):** Designed as a multi-pack of neon business cards.

**Interaction Mechanics (The Dispense & Reveal)**
When the user clicks on any package (or types its grid coordinate on a small 3D keypad rendered on the right side of the machine), the CSS coil spins. The item undergoes a CSS keyframe animation: dropping off the shelf, bouncing into the "Collection Tray" at the bottom of the screen. Upon hitting the tray, a soundless impact triggers an animation where the item expands outward, filling the center of the screen as an elegant, frosted-glass Modal Card, dimming the rest of the machine.

**Modal Card Contents (The Dispensed Info)**
Each expanded card contains elegant typography, matching icons, and interactive elements:
*   **Box 1 (GlglStats):** Explains it as a Python project measuring repetitiveness of Israeli pop-chart songs based on LZ77 compression, with data visualizations of famous songs. Includes a glowing button for the site (https://glglstats.geekip.dev) and a YouTube embed or sleek link to the PyData TLV presentation (https://www.youtube.com/watch?v=jrKAtOsj1Lo).
*   **Box 2 (Ahuzat Dibuk):** Describes the Python scraping and data visualization project for Tel-Aviv parking lot occupancy heat-maps. Features a grid of news clipping thumbnails linking to Channel 13 (https://13tv.co.il/item/news/domestic/internal/parking-902956407/), Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), and the original Secret Tel Aviv Facebook post (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).
*   **Box 3 (2D Part Orienting):** Highlights the C++ implementation in an open-source computational geometry library to construct a "push plan" orienting polygons. Link rendered as a mechanical blueprint button: https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/.
*   **Box 4 (The Turing Jest):** Details the LMSYS-like arena where LLMs compete in pun-making and explaining, rated by users. Standard action button links to: https://jest.geekip.dev.
*   **Box 5 (Rhymes of Israel):** Showcases the Chrome extension using LLMs to rewrite Israeli news headlines into comedic high-school yearbook rhymes. Link features a Chrome logo accent: https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb.
*   **Box 6 (Hatul Basak):** Explains the Hebrew port of catfishing.net (Wikipedia trivia). Notes the AI-generated daily challenges and user-created themed puzzles. Play/Create link: https://hatulbasak.geekip.dev.
*   **Box 7 (The Meta Portfolio):** A self-referential message pointing out that this item *is* the site holding them. Loop-link provided to: https://geekip.dev.
*   **Box 8 (Connection Multi-Pack):** A sleek grid of three large social buttons linking to github.com/gevak, linkedin.com/in/gevakip, and facebook.com/geva.kipper.

A subtle "X" or clicking outside the card reverses the animation, shrinking the card away and returning focus to the vending machine for the next selection.
