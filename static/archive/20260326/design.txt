The website is conceived as a sleek, single-page, futuristic 3D-effect vending machine set against an ambient, dark-mode backdrop with shifting, generative lighting. The overarching aesthetic is "Cyberpunk Elegance"—utilizing frosted glassmorphism, brushed gunmetal textures, and vibrant neon LED accents. Instead of traditional scrolling, the user interacts with the interface by pressing glowing mechanical buttons to "vend" project cartridges, which drop into a tray and seamlessly expand into readable interface cards.

**The Header Panel (Top of the Machine)**
At the crest of the vending machine is a wide, glowing retro-futuristic signboard. The main logo features the name **Geva Kipper** in an elegant, metallic sans-serif typography. Directly beneath it is a backlit, high-contrast LCD screen displaying the text: *"Constructing the framework of tomorrow as a Software Engineer at Google."* Wrapping around the edge of this header is a scrolling red LED ticker displaying the subtitle: *"SYSTEM NOTICE: This environment is re-designed daily by artificial intelligence. The reality you see is ever-changing."*

**The Vending Interface & Animation Mechanics**
The center viewport is a large, frosted-glass window showcasing seven distinct "products" sitting within silver metal coils on three illuminated shelves. Next to the glass is a metallic keypad. When a user clicks a product or its corresponding keypad number, the UI triggers a smooth JavaScript animation: the coil spins, the item drops into the dispensation tray at the bottom, and instantly scales up, expanding into an elegant, floating glassmorphism modal in the center of the screen. Closing the modal returns the item to the machine.

**The Seven Dispensed "Products" (Content Sections)**
Each dispensed project expands into a rich informational card featuring stylized typography, the description, and interactive buttons:

1. **GlglStats (Shelf 1, Slot A):** Packaged visually as a retro cassette tape or vinyl record sleeve. The expanded card details a Python project measuring Israeli pop-chart song repetitiveness using LZ77 compression, complete with data visualizations of famous songs. The card contains two neon-outlined action buttons: one linking to the live site (*https://glglstats.geekip.dev*) and an embedded, playable YouTube iframe of the PyData TLV conference presentation (*https://www.youtube.com/watch?v=jrKAtOsj1Lo*).
2. **Ahuzat Dibuk (Shelf 1, Slot B):** Packaged to look like a futuristic, glowing parking meter. The expanded view describes the Python scraping and data visualization project that generates heat-maps for Tel-Aviv parking lot occupancies. It features a "Media Coverage" section with sleek hyperlink buttons to Channel 13 (*https://13tv.co.il/item/news/domestic/internal/parking-902956407/*) and Mako (*https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm*), alongside a social link to the Secret Tel-Aviv Facebook publication (*https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/*).
3. **2D Part Orienting (Shelf 2, Slot A):** Represented by an intricate, glowing polygonal puzzle cube. The expanded panel delves into the C++ implementation within an open-source computational geometry library, detailing the algorithm used to construct a push plan for polygon orientation. A prominent action button directs users to the code and documentation (*https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/*).
4. **The Turing Jest (Shelf 2, Slot B):** Styled as a sleek, metallic comedy/tragedy mask. This card explains the LMSYS-style arena where Large Language Models compete at crafting and explaining puns, rated by human users. A brightly pulsing button invites the user to "Enter the Arena" (*https://jest.geekip.dev*).
5. **Rhymes of Israel (Shelf 3, Slot A):** Designed to look like a digital, holographic highschool yearbook. The description highlights the Chrome extension that utilizes an LLM to rewrite Israeli news headlines into comedic, yearbook-style rhymes. An "Install Extension" button links directly to the Chrome Web Store (*https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb*).
6. **Meta Portfolio (Shelf 3, Slot B):** A visually recursive item—a miniature, glowing replica of the exact vending machine the user is currently looking at. The text cheekily acknowledges the meta nature of the project (the site itself). It includes a recursive self-reference button leading to *https://geekip.dev*.
7. **Social Connectors (Bottom Shelf/Tray Core):** Packaged as a sleek multi-pack of holographic business cards. Once expanded, it presents three large, minimalist icon-based buttons that hover with CSS animations, securely linking to GitHub (*github.com/gevak*), LinkedIn (*linkedin.com/in/gevakip*), and Facebook (*facebook.com/geva.kipper*).

**The Maintenance Panel (Footer Space)**
At the very bottom base of the vending machine is a simulated dark-metal maintenance hatch. Engraved into the metal using CSS inset styling is a small plaque that reads: *"Manufactured & Designed by Assistant AI Model."* Next to it is a glowing, digital green stopwatch display, running via a JavaScript interval, tracking the exact hours, minutes, and seconds that have passed since "2026-03-26 01:50:53," grounding the futuristic UI in a highly specific temporal reality.
