The overall aesthetic of the single-page website merges modern dark-mode elegance with a dynamic, typographic UI. The background is a deep, matte charcoal, allowing neon-accented text and smooth, glassmorphism elements to pop. Functioning as a continuous scroll, the layout is anchored by interactive typography and physics-based CSS/JS animations, giving the user the sense that they are operating a sleek, futuristic terminal.

The hero section is dominated by the core interactive theme: an "Anagram Solver." At the center, the developer's name, **Geva Kipper**, is displayed in massive, bold, decoupled letter tiles. Driven by JavaScript, these letters continuously detach, float, scramble, and lock into new configurations to spell out various software engineering skills (e.g., pulling in ghost letters to form words like "API KEEPER" or "PEAK GRIP"), before snapping seamlessly back to his name. Just below this kinetic display sits a clean, sans-serif sentence: "Software Engineer at Google, [insert single sentence detailing current work]." Underneath this, stylized in an automated, monospaced terminal font with a pulsing indicator dot, is the subtitle: "Notice: This website is entirely re-designed each day by AI and is thus ever-changing."

Below the hero area, the page transitions into a sleek, asymmetrical masonry grid containing eight card-like sections. Rather than traditional boxes, these are styled as "Output Solutions" generated by the anagram engine above. Each card features a dark, semi-transparent frosted-glass styling, elegant hover-state glows, and minimal iconography that represents the project's core concept. 

**Card 1: GlglStats** 
Topped with an animated, CSS-rendered audio waveform that subtly compresses itself (referencing LZ77), this card introduces a Python project measuring the repetitiveness of Israeli pop-chart songs and visualizing repetition patterns. It includes two sleek hyperlink buttons: one pointing to the live site (https://glglstats.geekip.dev) and another styled as a video link to the PyData TLV conference presentation (https://www.youtube.com/watch?v=jrKAtOsj1Lo).

**Card 2: Ahuzat Dibuk** 
This card features an abstract, glowing heat-map grid in its corners to represent its data visualization nature. It details the Python scraping project collecting statistics on Tel-Aviv parking lot occupancy for local residents. Below the description is a neat row of media pills linking to news coverage on 13tv (https://13tv.co.il/item/news/domestic/internal/parking-902956407/), Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), and the original viral Facebook publication (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).

**Card 3: 2D Part Orienting** 
Designed with a wireframe, rotating geometric polygon hovering in the background, this section outlines the C++ implementation. It describes the open-source computational geometry library algorithm used for constructing a "push plan" to orient polygons. A sharp, technical-looking link routes users to the additional info and code (https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/).

**Card 4: The Turing Jest** 
Visually divided to represent a duel, this card features two opposing chat-bubbles. It introduces an LMSYS-like arena where LLMs compete at making and explaining puns, rated by users. A prominent neon button links directly to the arena (https://jest.geekip.dev).

**Card 5: Rhymes of Israel** 
Decorated with a newspaper icon that morphs into a comedy mask via CSS transition on hover, this card presents the Chrome extension that utilizes an LLM to rewrite Israeli news headlines into comedic, high-school yearbook-style rhymes. It contains a direct installation link to the Chrome Web Store (https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb).

**Card 6: Hatul Basak** 
This box features a subtle backdrop depicting a minimalist silhouette of a cat intertwined with a question mark. It describes the Hebrew port of catfishing.net—a Wikipedia-based Trivia game utilizing AI for daily challenges, alongside user-generated themed puzzles. A playful call-to-action link invites users to play and create at https://hatulbasak.geekip.dev.

**Card 7: Meta Portfolio Project** 
Leaning into the meta-humor, this card utilizes an "infinity mirror" CSS border effect. It briefly explains the project of building this exact AI-designed portfolio website. It includes a self-referencing anchor link pointing directly back to https://geekip.dev.

**Card 8: Connect** 
The final box acts as a stripped-back, elegant interactive dock. Instead of heavy paragraphs, it uses large, sleek vector icons that animate upon hover, linking directly to his GitHub (github.com/gevak), LinkedIn (linkedin.com/in/gevakip), and Facebook (facebook.com/geva.kipper).

At the absolute bottom of the page, the design strips away the glassmorphism and returns to pure, raw terminal aesthetics. Centered in a glowing, monospace green font is a permanent footer reading: "Design generated and compiled by [Insert AI Model Name]." Directly below this text is a live, ticking digital stopwatch, running via JavaScript, actively counting the exact hours, minutes, and seconds that have transpired since the specific timestamp: 2026-05-24 02:39:17.
