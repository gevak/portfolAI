**Global Aesthetic and Layout Overview**
The website should be designed as a single-page, full-screen interactive experience modeled after a nostalgic 90s OS desktop (reminiscent of Windows 95/98), but built with smooth, modern CSS and JavaScript. The background should be a classic muted teal (`#008080`) or a subtle repeating retro pattern. Use a crisp, modern sans-serif font for readability, paired with pixelated accent fonts for headers to merge modern elegance with retro charm. The layout consists of several pre-opened, draggable "application windows" scattered across the screen in a cascading masonry style. Each window should have a classic 3D-beveled gray border, top blue title bars with decorative minimize/maximize/close buttons, and a drop-shadow that creates depth. JavaScript should be used to make the windows draggable and to bring them to the front when clicked.
