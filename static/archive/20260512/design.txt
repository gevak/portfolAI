The overall layout functions as a full-screen, interactive 2D physics playground where the standard rules of web design are subjected to real-time, user-controlled gravitational forces. The central visual hook is a floating, glass-morphic "CSS Editor" window permanently pinned to the absolute center-right of the screen. This panel displays exposed CSS variables (e.g., `--gravity-y`, `--gravity-x`, `--bounciness`, `--wind-resistance`) equipped with draggable numeric sliders. As the user alters these values, every container on the page reacts dynamically—falling to the bottom, floating to the ceiling, drifting sideways, or ricocheting off one another based on the updated physics parameters. 

The aesthetic blends dark-mode minimalism with vibrant, tactile neo-brutalism. The background is a deeply saturated, grainy obsidian matte. To ensure usability amidst the chaos, every physical element on the page features a "magnetic hover" state; when the cursor enters an element, it temporarily ignores all physics, freezing in place, scaling up by 5%, and projecting a soft neon drop-shadow to allow for easy reading and interaction with links and media.

The header serves as a solid, static "shelf" pinned to the top-left of the screen, acting as a collision boundary for the physics engine. It features elegant, oversized typography using a sharp sans-serif font for "Geva Kipper", followed by a refined, smaller monospaced sentence: "Currently engineering scalable solutions and pushing technical boundaries at Google." Below this, styled as a glowing, iridescent banner, is the subtitle: "This environment is re-designed daily by AI; a transient, ever-changing digital ecosystem."

The core content is divided into eight distinct "physics blocks." These blocks are styled as thick, rounded cards with a translucent frosted-glass aesthetic, allowing the dark background to blur underneath. Each card features unique visual accents matching its project:
Block 1 (GlglStats) incorporates a background motif of compressed audio waveforms. It includes a sleek embedded video player for the PyData TLV YouTube link, a clean button linking to the project site, and concise typography explaining the LZ77 Israeli pop-chart compression.
Block 2 (Ahuzat Dibuk) utilizes a subtle, shifting thermal heat-map gradient background. It houses the Tel-Aviv parking explanation alongside an organized, pill-shaped link cluster pointing to the Channel 13 and Mako news stories, plus a distinct Facebook icon linked to the original Secret Tel Aviv post.

Block 3 (2D Part Orienting) features a wireframe, rotating polygon animation in the card's background. The C++ CGAL push-plan algorithm text sits above a prominent, terminal-styled link pointing to the TAU computational geometry project page.
Block 4 (The Turing Jest) splits visually down the middle in an LMSYS-arena style, contrasting two different colors. It features typography styled like a modern leaderboard and a prominent call-to-action button linking to the jest.geekip.dev arena.
Block 5 (Rhymes of Israel) adopts a playfully chaotic design—a mix of a newspaper clipping overlay with high-school yearbook pen doodles. It describes the LLM headline-rewriting extension and houses a large, official "Available on Chrome Web Store" badge linking to the extension URL.

Block 6 (Hatul Basak) employs a whimsical, trivia-node connecting line art background. It features the catfishing.net Hebrew port explanation, emphasizing the AI-generated daily challenges, and includes a prominent "Play & Create" button linking to the live game page.
Block 7 (The Meta Portfolio) acts as a visual anomaly, designed like an infinite mirror or recursive screen. The text humorously addresses the meta-nature of the portfolio itself, containing a self-referential hyperlinked button to geekip.dev.
Block 8 (Socials) breaks from the rectangular card shape, designed instead as a cluster of three heavy, circular "billiard balls" bound together by springs. Each sphere contains a large, frictionless icon linking respectively to GitHub, LinkedIn, and Facebook.

At the absolute bottom of the viewport, serving as the physical "floor" that the project blocks crash into, is a slim, neon-lined footer. The left side explicitly credits the AI model (acting as the designer) in a subtle, technical font, framing it as the "Architect." The right side of the footer features a high-contrast, glowing stopwatch ticker. Utilizing a futuristic monospaced typeface, it actively tracks the exact hours, minutes, and seconds elapsed since "2026-05-12 02:27:02", creating a sense of relentless momentum contextualized within an AI-governed temporal baseline.
