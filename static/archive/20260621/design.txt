The overall layout is a single-page, full-viewport experience featuring a photorealistic, 3D-styled smartphone floating in the center of the screen against a sleek, dark glassmorphic background. The ambient background features slow-moving, generative gradients that give a subtle nod to the AI-generated theme. The entire user journey takes place within the boundaries of this simulated smartphone screen, utilizing modern CSS grid boundaries, hidden overflow, and smooth JavaScript-driven expanding transitions to mimic opening and closing native mobile applications.

At the top of the simulated phone screen is a standard mobile status bar (time, battery, signal). Below this sits a transparent, frosted-glass "Home Screen Widget" acting as the website's header. It displays the name "Geva Kipper" in a bold, modern sans-serif typography. Beneath the name is the sentence: "Software Engineer currently working at Google to build and deploy scalable, high-impact technologies." Below this, highlighted by a subtle glowing border, is the dynamic subtitle: "This website is re-designed each day by AI and is thus ever-changing." 

The core navigational interface is a simulated "App Grid" on the phone's home screen, utilizing a 3x3 layout of sleek, iOS-style squircle (rounded square) app icons. When a user clicks an app icon, a fluid JavaScript hero-transition expands the icon to fill the entire phone screen, revealing the project details inside a native-looking "app UI." A persistent "swipe up to go home" gesture bar or a floating "X" button allows the user to close the project and return to the grid. 

The first app icon represents "GlglStats" and features a graphical soundwave morphing into a ZIP file icon. Inside, the app resembles a dark-mode audio visualizer dashboard. It presents the Python and LZ77 compression algorithm project measuring the repetitiveness of Israeli pop-chart songs. Information is layered over an animated visualization of song repetition patterns. The screen features two prominent, pill-shaped buttons: one linking to the site (https://glglstats.geekip.dev) and another to the PyData TLV presentation (https://www.youtube.com/watch?v=jrKAtOsj1Lo).

The second app icon, for "Ahuzat Dibuk," features a parking sign enclosed in a thermal heatmap. The expanded view simulates a mobile map application, displaying a stylized mock-up of Tel-Aviv with glowing red and green parking occupancy dots. Textual UI cards explain the Python scraping and data visualization mechanics. A "News Mentions" section contains three notification-style link banners for the media coverage (https://13tv.co.il/item/news/domestic/internal/parking-902956407/ and https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm) alongside a link to the original Facebook publication (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).

The third icon, "2D Part Orienting," resembles intersecting wireframe geometric shapes. Opening it reveals a clean, academic, light-mode interface. The background features a subtle animation of an SVG polygon being oriented by a computational pushing algorithm. Floating UI cards house the details regarding the C++ implementation to construct push plans. A large "View Source & Info" button links to the computational geometry library project page (https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/).

The fourth app, "The Turing Jest," uses an icon combining a theatrical comedy mask with a robotic circuit design. Expanding the app reveals a neon-accented, split-screen interface mimicking a fighting game "versus" screen or an arena, representing the LMSYS-like competition where LLMs formulate and explain puns. Information dynamically types out on the screen, prompting users to rate the AI jokes, with a bold button leading to the live arena (https://jest.geekip.dev).

The fifth app, "Rhymes of Israel," is represented by an icon of a folded newspaper wearing a graduation cap. The interior UI simulates an interactive, vintage high-school yearbook page. Mapped over mock photo slots are headline rewriting mechanics showing how the Chrome extension uses LLMs to turn real Israeli news into comedic rhymes. A prominent standard "Add to Chrome" web store button links directly to the extension (https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb).

The sixth app, "Hatul Basak," is styled with an icon of a minimalist cat silhouette overlaid on a Wikipedia puzzle globe. The internal app design is a vibrant, gamified trivia interface honoring the catfishing.net port. It outlines the AI-generated daily challenges and community-driven themed puzzles. Two main action buttons dominate the bottom of the screen: "Play Daily Game" and "Create Custom Puzzle", both directing to the platform (https://hatulbasak.geekip.dev).

The seventh app is the "Meta-Portfolio" app, featuring an icon of a recursive smartphone or a terminal window reflecting itself. The interior design breaks the fourth wall, showcasing a cascading, "Matrix-style" stream of the website's own HTML and CSS code in the background, contrasting with elegant text explaining the meta-nature of the portfolio project. A self-referencing glowing URL link directs to the root site (https://geekip.dev).

The eighth and final app icon represents a "Contacts/Social" folder. When opened, it reveals a sleek, digital "vCard" layout for Geva Kipper. The screen features three massive, beautifully animated interactive blocks utilizing the official brand colors and icons of GitHub (github.com/gevak), LinkedIn (linkedin.com/in/gevakip), and Facebook (facebook.com/geva.kipper).

Finally, sitting on the simulated lock-screen beneath the app grid—or alternatively floating underneath the physical phone bezel on the desktop background—is a stylized, monospace terminal interface. This footer serves two functions: it prominently displays credit to the AI model (acting as the system architect) with the text "Designed and continuously generated by AI." Below this credit is a live, ticking JavaScript stopwatch displaying the elapsed time in an exact hours, minutes, and seconds format (HH:MM:SS), actively counting up from the timestamp 2026-06-21 03:00:22.
