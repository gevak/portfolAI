The overarching aesthetic of the single-page application is "Digital Botany"—an elegant, deep-dark-mode interface utilizing a rich palette of obsidian black, forest green, and bioluminescent emerald. The initial page load acts as an interactive landing screen devoid of traditional navigation. The viewport displays a minimalist, immersive canvas featuring a dark, stylized meadow made of geometric, subtle, slow-drifting wireframe three-leaf clovers. 

Centered at the top of the viewport in a crisp, modern sans-serif typeface (such as Inter or Roobert) is the name "Geva Kipper", followed by a sleekly styled sentence: "Software Engineer currently building the future at Google." Just beneath this, a shifting, glitch-aesthetic subtitle reads: "This digital space is re-designed each day by AI—forever shifting, never static." The rest of the page remains seemingly empty, prompting the user to interact with the environment. As the user moves the cursor, the three-leaf clovers subtly repel like a magnetic field. Hidden randomly within this field is a single, subtly pulsing four-leaf clover. 

When the user hovers over the four-leaf clover, it emits a bright green neon glow. Clicking it triggers a dramatic, "decryption" ripple effect across the screen. The wireframe clover field dissolves into the background, and the portfolio's core content—seven frosted glassmorphic cards—rises fluently into a masonry CSS grid layout via a cascading entrance animation, effectively revealing Geva’s "hidden coding skills."

**Card 1: GlglStats**
This card features an animated background motif of LZ77 compression patterns resembling audio equalizer bars. The typography clearly outlines the Python project measuring repetitiveness in Israeli pop-chart songs. Included are two stylish, pill-shaped buttons: one directing to the live site at `https://glglstats.geekip.dev`, and another featuring a minimalist play-icon linking to the PyData TLV conference presentation at `https://www.youtube.com/watch?v=jrKAtOsj1Lo`.

**Card 2: Ahuzat Dibuk**
This section integrates a subtle, shifting red-to-green thermal heat-map as its background texture, signifying the Tel-Aviv parking lot occupancy data. It includes a brief description of the Python scraping project, followed by three distinct, icon-paired text links pointing to its media coverage: Channel 13 (`https://13tv.co.il/item/news/domestic/internal/parking-902956407/`), Mako (`https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm`), and the original grassroots Facebook post (`https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/`).

**Card 3: 2D Part Orienting**
Designed with a computational geometry aesthetic, this card features faint, rotating 2D polygons in its corners. It describes the C++ algorithm for constructing "push plans" within an open-source library. A sharp, technical-looking button reads "View Research & Code," linked to `https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/`.

**Card 4: The Turing Jest**
This block uses a subtly playful split-screen background—half dark, half slightly lighter—resembling a competitive arena. It explains the LMSYS-like arena where LLMs compete in pun-making and user ratings. A prominent, glowing emerald link directs users to battle the models at `https://jest.geekip.dev`.

**Card 5: Rhymes of Israel**
Incorporating a faint visual texture of a crumpled, highschool yearbook page blended into the dark glassmorphism, this card details the LLM-powered Chrome extension formatting Israeli news into comedic rhymes. A stylized "Add to Chrome" button directs the user to `https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb`.

**Card 6: The Meta Portfolio**
This card features an endless mirror or recursive visual loop in its background container, winking at the fact that it describes the very site the user is navigating. It explains the portfolio website project itself, anchored by a self-referential link to `https://geekip.dev`.

**Card 7: The Network**
Diverging slightly in shape—perhaps a wide spanning horizontal row at the bottom of the grid—this final block acts as the social hub. It displays large, elegant, monochromatic icons for GitHub, LinkedIn, and Facebook that illuminate in their respective brand colors upon hover. They link directly to `github.com/gevak`, `linkedin.com/in/gevakip`, and `facebook.com/geva.kipper`.

Anchoring the bottom of the single-page layout is a minimalist footer pane. Flushed to the left in a stark monospace font is the credit: "Design & Assembly by AI." Flushed to the right is a live, ticking digital stopwatch (Hours:Minutes:Seconds) calculating the exact time elapsed since the precise timestamp of `2026-03-17 01:28:04`, ticking ceaselessly to emphasize the ephemeral, ever-changing nature of the project.
