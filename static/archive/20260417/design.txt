The website is a single-page, full-viewport experience designed around a sleek, retro-futuristic vending machine. The visual aesthetic utilizes elegant "glassmorphism"—frosted glass panes, subtle neon glowing borders, and deep metallic textures against a minimalist dark-mode background. The entire site is housed within the borders of this machine. At the very top, an animated glowing LED marquee serves as the header. It reads: "Geva Kipper // Software Engineer currently building [insert specific work at Google] at Google." Directly beneath it, a smaller scrolling ticker continuously displays the subtitle: "WARNING: This interface is re-designed each day by AI and is thus ever-changing." 

The central feature is the glass display window, featuring a grid of eight distinct "slots" held by metallic CSS-animated coils. Instead of traditional boxes, each of the eight sections is represented as a stylized, clickable vending machine product (like a soda can, candy bar, or tech gadget) that hints at the project's nature. On the right side of the machine is a glossy keypad. When a user clicks an item or types its grid number on the keypad, a smooth CSS animation shows the item spinning off its coil, dropping behind the glass into the bottom "dispenser tray".

Once an item lands in the tray, it automatically triggers a sleek, translucent modal overlay that expands to reveal the full details, text descriptions, and interactive links of the dispensed project. 

The eight vending slots and their corresponding modal contents are designed as follows:
Slot A1 is an energy drink can labeled "LZ77". Its modal presents 'GlglStats', detailing the Python project measuring Israeli pop-chart song repetitiveness using LZ77 compression and visualizing repetition patterns. It includes two glowing, hover-animated buttons linking to the live site (https://glglstats.geekip.dev) and the PyData TLV conference presentation (https://www.youtube.com/watch?v=jrKAtOsj1Lo).
Slot A2 is designed as a miniature parking meter or a ticket with a heat-map gradient. Its modal presents 'Ahuzat Dibuk', detailing the Python scraping and data visualization for Tel-Aviv parking lot occupancy. It features a media grid linking to Channel 13 (https://13tv.co.il/item/news/domestic/internal/parking-902956407/), Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), and the Secret Tel-Aviv Facebook post (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).
Slot A3 takes the shape of a geometric puzzle or Rubik's cube. Its modal presents '2D Part Orienting', explaining the C++ implementation in an open-source computational geometry library for constructing a polygon push plan. It includes a link to the code and info via TAU (https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/).
Slot A4 is packaged like a comedy/tragedy theater mask candy. It opens the modal for 'The Turing Jest', describing the LMSYS-like arena where LLMs compete in making and explaining puns, rated by users, with a direct link to the arena (https://jest.geekip.dev).

Slot B1 looks like a classic high school yearbook wrapped in a newsprint texture. The modal introduces 'Rhymes of Israel', the Chrome extension that uses an LLM to rewrite Israeli news headlines into comedic rhymes. It features an installation link for the Chrome Web Store (https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb).
Slot B2 is a bottle of stylized catnip labeled in Hebrew. Its modal reveals 'Hatul Basak', the Hebrew port of catfishing.net. The text explains the daily AI-generated trivia challenges and user-created themed puzzles, with a play/create link (https://hatulbasak.geekip.dev).
Slot B3 is a meta-item: a tiny, infinitely recursive, glowing vending machine. Its modal describes the current portfolio website project itself, acting as a self-referential easter egg with a link to the root site (https://geekip.dev).
Slot B4 is a "variety snack pack" featuring the logos of popular networks. Its modal acts as the contact card, distributing links to GitHub (github.com/gevak), LinkedIn (linkedin.com/in/gevakip), and Facebook (facebook.com/geva.kipper).

At the bottom of the screen, styled like an illuminated floor mat or an electronic receipt printing out of the vending machine's base, sits the footer. It features minimal, monospaced typography giving credit to the AI model that dreamed up the design. Beside this credit is a live JavaScript-powered stopwatch, styled like a red digital clock face, actively ticking up to show the exact elapsed Time (Hours:Minutes:Seconds) since the timestamp 2026-04-17 01:58:15.
