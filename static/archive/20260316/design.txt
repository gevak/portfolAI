The website is designed as a sophisticated, dimly lit modern art museum. The background is a deep, textured charcoal, simulating a matte gallery wall. A dynamic JavaScript-driven spotlight effect follows the user’s cursor, softly illuminating the textured wall and casting realistic shadows behind the elements to create depth. Typography relies on a high-contrast pairing: an elegant, sculptural serif (like Playfair Display) for headers to evoke a fine-art aesthetic, and a crisp, modern sans-serif (like Inter) for descriptions to maintain readability.

At the very top of the page, acting as the gallery’s welcoming foyer, is the developer's name, "Geva Kipper," presented in grand, silver-foiled lettering that catches the ambient cursor light. Beneath it rests a single, crisp sentence: "Currently engineering impactful solutions at Google." Below this, an elegant, italicized subtitle rests subtly in the background: *"Curated anew each dawn: This exhibition is dynamically re-designed daily by AI, existing in a state of perpetual metamorphosis."* 

The core of the page features a masonry or horizontal-scroll layout of seven "Masterpieces." Each of the first six projects is encased in a distinct, photorealistic gallery frame—some sleek brushed metal, others matte black wood—each utilizing heavy, textured white passe-partout (matting). Inside each frame, a subtly animated CSS/WebGL canvas represents the project visually. When the user hovers over a frame, a 3D perspective shift pulls the artwork forward, the room’s ambient lighting dims to highlight that specific piece, and a pristine white museum plaque smoothly slides out from the wall beside or below the frame to display the project's details and interactive links.

**Frame 1: GlglStats**
The artwork inside this frame displays a mesmerizing, pulsating wave-form of algorithmic patterns dissolving into musical notes. The accompanying plaque reads: "A Python project measuring the repetitiveness of Israeli pop-chart songs using LZ77 compression, visualizing repetition patterns in famous tracks." Links are styled as elegant museum audio-guide buttons, featuring a direct portal to the project (https://glglstats.geekip.dev) and a projector icon linking to the PyData TLV conference presentation (https://www.youtube.com/watch?v=jrKAtOsj1Lo).

**Frame 2: Ahuzat Dibuk**
This canvas glows with a stylized, neon-toned heatmap overlaying a dark, abstract map of Tel-Aviv streets. The plaque details a Python scraping and data visualization endeavor that collects parking lot occupancy statistics to generate heat-maps for local residents. The press links are displayed as framed newspaper clippings on the plaque, linking to Channel 13 (https://13tv.co.il/item/news/domestic/internal/parking-902956407/), Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), and the original Facebook publication (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).

**Frame 3: 2D Part Orienting**
The frame holds a kinetic, wireframe animation of geometric polygons constantly calculating and orienting themselves into perfect alignment. Its museum plaque describes a C++ implementation within an open-source computational geometry library, detailing an efficient algorithm for constructing a 'push plan' to orient polygons. A minimalist blueprint icon anchors the link to the code and additional info (https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/).

**Frame 4: The Turing Jest**
Inside this frame, two classical theatre masks (comedy and tragedy) are woven entirely from glowing neural network nodes that spark rhythmically. The plaque explains this LMSYS-like arena where Large Language Models fiercely compete to craft and explain puns, with the victor decided by user ratings. A theater ticket icon directs visitors to the arena (https://jest.geekip.dev).

**Frame 5: Rhymes of Israel**
The artwork here presents a grim, traditional newspaper headline that is continuously torn away to reveal colorful, scribbled, high-school yearbook-style poetry underneath. The plaque introduces a Chrome extension utilizing an LLM to rewrite Israeli news headlines into comedic rhymes. An install button, shaped like a gallery acquisition tag, links to the Chrome Web Store (https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb).

**Frame 6: The Meta Portfolio**
This frame produces an optical "infinity mirror" illusion, showing an endlessly receding gallery wall containing the very site the user is currently browsing. The plaque humorously notes the self-referential nature of a portfolio website displaying itself as a project. A minimalist "You Are Here" icon links back to the site’s root (https://geekip.dev).

**Section 7: The Artist's Pedestal**
Instead of a wall frame, the seventh element is a 3D-rendered marble museum pedestal sitting in the foreground. Resting atop it under a glass cloche are three finely crafted, glowing metal icons. These serve as direct, interactive conduits to the artist’s external presence: GitHub (github.com/gevak), LinkedIn (linkedin.com/in/gevakip), and Facebook (facebook.com/geva.kipper). Hovering over the cloche lifts the glass, illuminating the links.

At the very bottom of the page, acting as the gallery's exit sign, is a sleek, glowing digital installation. The text reads: "Exhibition architecture and curation by Assistant AI." Directly beside it, a digital stopwatch utilizes a glowing, red monospaced font to tick endlessly upward, displaying the exact hours, minutes, and seconds that have passed since the gallery's inception on `2026-03-16 01:51:45`.
