# Chatbot Terminal Portfolio Design

## Overall Aesthetic
The entire page mimics a retro terminal interface merged with a modern chat application. The background is a deep navy gradient (#0a0e27 to #1a1f3a) with subtle animated matrix-style falling characters in the background using the Hebrew alphabet mixed with code symbols, barely visible at 5% opacity. The interface uses a monospace font (Fira Code) for code elements and Inter for readable text.

## Chat Interface Core
The main content area is styled as a sleek chat window with rounded corners and a subtle glass-morphism effect (semi-transparent white background with backdrop blur). On the left side is a narrow sidebar showing a simplified "AI Assistant" avatar (a geometric hexagon with pulsing gradients cycling through cyan, purple, and green).

## The Six Project Boxes
Each project appears as a chat message bubble from the AI assistant, alternating between left-aligned (projects 1, 3, 5) and right-aligned (projects 2, 4, 6) to create visual rhythm. Each bubble has:

- A gradient border (unique color scheme per project: GlglStats uses purple-pink, Ahuzat Dibuk uses orange-red, 2D Part Orienting uses blue-teal, Turing Jest uses yellow-green, Rhymes of Israel uses red-violet, Portfolio uses rainbow gradient)
- A semi-transparent dark background with subtle hover animation that brightens and expands slightly
- Project title in bold with an emoji prefix (üéµ for GlglStats, üöó for Ahuzat Dibuk, üìê for 2D Part Orienting, üòÇ for Turing Jest, üì∞ for Rhymes, üåê for Portfolio)
- Description text that appears to type out when scrolled into view
- Links styled as terminal commands in brackets with hover effects: [LAUNCH_SITE], [WATCH_TALK], [READ_NEWS], [VIEW_CODE], [INSTALL]

Each bubble includes a small "Ask me more about this ‚Üí" button at the bottom that, when clicked, expands to show a pre-written AI response with additional technical details and tech stack information.

## Interactive Input Box
At the bottom of the chat interface, before the social links, there's a functional text input field styled as a terminal command line with the prompt "root@geva:~$". Users can type questions, and a simple keyword-based chatbot responds with canned answers about Geva's work, skills, and projects. The responses appear as new chat bubbles with a brief typing indicator animation (three bouncing dots).

- Smooth scroll snapping to each project bubble
- Parallax effect on the background Hebrew matrix characters
- Hover effects that create ripple animations from the cursor position on each bubble
