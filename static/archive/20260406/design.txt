The website features a dark, immersive aesthetic reminiscent of a modern mystic’s divinatory table. The background is a velvety midnight blue (`#0A0B1A`) layered with a subtle, dynamic JavaScript particle effect that resembles slowly swirling cosmic dust or incense smoke. The typography contrasts this darkness using an elegant, golden serif font (such as Playfair Display or Cinzel) for headers, and a clean, highly legible sans-serif (like Inter) for body text. The mouse cursor is replaced with a glowing, minimalist four-pointed star that leaves a brief pixel-trail as the user interacts with the page.

At the top center of the page, gleaming in a heavy gold-foil gradient, is the name "Geva Kipper." Directly beneath it is a centered, elegant subtitle in a soft silver tone: "Weaving algorithms into reality as a Software Engineer at Google." Below this, styled as a mysterious, glowing runic inscription that pulses gently with a CSS keyframe animation, is the second subtitle: "The cosmic alignment of this realm shifts daily; this website is re-designed every 24 hours by AI and is thus ever-changing." 

The core of the page is a responsive CSS Grid laid out like a massive Tarot card spread (a stylized "Celtic Cross" or a sweeping arc). The layout presents seven physical-looking "cards." Each card has a sleek 3D hover effect using CSS transforms; when a user hovers, the card slightly levitates and casts a golden glow. Upon clicking, a smooth 180-degree 3D flip animation reveals the back of the card—the "reading"—which contains the project details on a dark parchment-textured background. 

Card I: The Wheel of Fortune (GlglStats). The front features geometric, mystic mandala art intertwined with musical notes. The back explains this Python project measuring the repetitiveness of Israeli pop-chart songs using LZ77 compression and visualizing the patterns of famous songs. It houses two illuminated, styled buttons: one linking to the live site at https://glglstats.geekip.dev, and another linking to the PyData TLV presentation at https://www.youtube.com/watch?v=jrKAtOsj1Lo. 

Card II: The Chariot (Ahuzat Dibuk). The front displays an esoteric illustration of overlapping, glowing chariots arranged in a grid. The back details the Python scraping and data visualization project that collected statistics on Tel-Aviv parking lot occupancy to generate heat-maps for residents. It features a list of golden hyperlinked runes pointing to news coverage on Channel 13 (https://13tv.co.il/item/news/domestic/internal/parking-902956407/) and Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), alongside an icon linking to the Secret Tel Aviv Facebook post (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).

Card III: The Magician (2D Part Orienting). The card front shows a sorcerer manipulating abstract, glowing polygons. The reverse describes the C++ implementation within an open-source computational geometry library of an efficient algorithm constructing a 'push plan' to orient a given polygon. A mystic portal button links to the source code and info at https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/.

Card IV: The Fool (The Turing Jest). Illustrated with two jesters locked in a cosmic duel, the back of this card reveals the LMSYS-like arena where large language models compete in making and explaining puns, rated by human users. An intricately bordered link directs seekers to https://jest.geekip.dev.

Card V: The Hierophant (Rhymes of Israel). The front presents an ancient scroll inscribed with glowing code. The back tells the story of an LLM-powered Chrome extension formatting Israeli news headlines into comedic, high-school yearbook-style rhymes. A stylized Chrome Web Store badge links directly to https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb.

Card VI: The Mirror (Meta Portfolio). The front art shows an Ouroboros (a snake eating its own tail) wrapping around a glowing web browser window. The reverse points out the meta-nature of the project—the very site the user is exploring. It contains a self-referential infinity-loop icon linking back to https://geekip.dev.

Card VII: The World (Connections). The final card displays a glowing celestial globe bound by three colored threads. The back acts as a digital rolodex of connections, featuring minimalist social icons adorned with mystical borders: a GitHub icon (github.com/gevak), a LinkedIn icon (linkedin.com/in/gevakip), and a Facebook icon (facebook.com/geva.kipper).

At the absolute bottom of the single-page layout is the footer, separated by a delicate, glowing golden horizontal line. It features a centrally aligned, elegant text crediting the AI model (e.g., "Divined and forged into the digital ether by [AI Model Name]"). Embedded directly below this credit is a live, ticking JavaScript stopwatch styled like an ethereal digital hourglass, accurately calculating and displaying the hours, minutes, and seconds that have passed since the temporal anchor point: 2026-04-06 01:54:36.
