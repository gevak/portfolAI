The overall aesthetic is a modern, deep-dark-theme interface (using a background color like #0F172A) with neon, glowing accents and glassmophic textures. The layout structurally mimics a classic Periodic Table of Elements using a CSS Grid. The background features a subtle, slowly drifting particle animation resembling molecular bonds, tying into the scientific, data-driven theme. Typography heavily utilizes a clean, geometric sans-serif (like 'Space Grotesk') paired with a classic monospace font (like 'JetBrains Mono') for technical metrics and element symbols.

The hero section sits at the top center, floating above the "dip" in the periodic table shape. It features the developer's name, **Geva Kipper**, in a bold, glowing, holographic font. Directly beneath is the sentence: "Crafting intelligent data and software solutions at Google." Below this, a smaller, glitch-effect monospace subtitle reads: *"This digital space is re-designed daily by AI; you are exploring today's transient synthesis."* 

The core of the page is the Interactive Periodic Table. To populate the grid, background "elements" represent various coding skills (e.g., [Py] Python, [Cp] C++, [Ll] LLMs). However, the focal points are the Seven Project/Link Boxes, styled as visually distinct, glowing "Noble Gases" and "Heavy Metals". By default, each of these 7 core items sits as a compact square in the grid, displaying a distinct chemical symbol (e.g., [Gl]), an atomic number, and an atomic weight. When hovered, an animated "electron orbit" circles the square. When clicked, the grid smoothly blurs and darkens, and the element expands via a fluid, center-screen modal transition into a detailed "Element Fact Card" containing the project specifics:

**1. The [Gl] Element (GlglStats):** Expanding to reveal a bright yellow and blue theme, this section details a Python project measuring Israeli pop-chart repetitiveness via LZ77 compression and visualizing repetition patterns. It features a glowing primary link to [https://glglstats.geekip.dev](https://glglstats.geekip.dev) and an embedded, stylized thumbnail linking to the PyData TLV presentation: [https://www.youtube.com/watch?v=jrKAtOsj1Lo](https://www.youtube.com/watch?v=jrKAtOsj1Lo).

**2. The [Ad] Element (Ahuzat Dibuk):** Expanding with deep green and heatmap-orange gradients, this card presents the Tel-Aviv car parking lots scraping and data visualization project. The UI mimics a geographic heat map. Links to news coverage are styled as floating "citations": [Channel 13 News](https://13tv.co.il/item/news/domestic/internal/parking-902956407/), [Mako](https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), and the original [Secret Tel Aviv Facebook Publication](https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).

**3. The [Po] Element (2D Part Orienting):** Glowing in a stark, terminal-green, this card explains the C++ open-source computational geometry implementation used for constructing a polygon orienting ‘push plan’. The background features wireframe polygons slowly rotating. It prominently features a link for additional info and code: [https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/](https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/).

**4. The [Tj] Element (The Turing Jest):** With an electric purple, AI-themed aura, this card presents the LMSYS-like arena where LLMs compete at making and explaining puns, rated by users. User-rating stars decorate the card’s borders. A glowing command-line button links to the site: [https://jest.geekip.dev](https://jest.geekip.dev).

**5. The [Ri] Element (Rhymes of Israel):** Expanding with a gradient of high-school yearbook paper textures and vibrant digital chrome, this section describes the LLM Chrome extension that rewrites headlines into comedic rhymes. A floating "Add to Chrome" badge links directly to the extension: [https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb](https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb).

**6. The [Me] Element (Portfolio / Meta):** Rendered as a highly reflective, mirror-like glass card, this self-referential element presents the current portfolio website itself. It contains an infinity-mirror visual effect in the background and houses a direct self-reference link: [https://geekip.dev](https://geekip.dev).

**7. The Lanthanide Series (Social Links):** Separated at the bottom of the table, mimicking the detached rows of a traditional periodic table, acts the seventh section. Expanding this wide row reveals stylized, neon icons branching out like chemical bonds to his profiles: [github.com/gevak](https://github.com/gevak), [linkedin.com/in/gevakip](https://linkedin.com/in/gevakip), and [facebook.com/geva.kipper](https://facebook.com/geva.kipper).

At the absolute base of the single page rests a minimalist footer resembling a retro-digital lab ledger. It contains a self-referential credit reading: "Design synthesized entirely by an AI Assistant." Beside it, built using JavaScript, is a live, monospace digital stopwatch. It dynamically tracks the exact hours, minutes, and seconds that have continuously elapsed since the precise timestamp: **2026-04-09 01:44:20**, pulsing quietly like a heartbeat.
