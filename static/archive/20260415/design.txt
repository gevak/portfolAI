**Overall Concept & Navigation Structure**
The website utilizes a vertical scroll-driven parallax architecture to simulate an underwater descent. The user’s viewport acts as the camera, following a sleek, modern, fixed-position submarine graphic. As the user scrubs down the page, background layers (water currents, floating particles, marine life) move upwards at varying speeds to create depth. The background color utilizes CSS linear gradients, transitioning seamlessly from a bright, sunlit azure at the top to a bioluminescent twilight zone, and finally to a pitch-black abyssal trench at the footer. The UI elements should utilize a "glassmorphism" aesthetic (translucent dark panels with strong background blurs and thin, glowing borders) to represent the submarine's internal Heads-Up Display (HUD). 

**Hero Section (The Surface & Mission Briefing)**
The site opens at the ocean's surface with gentle, animated SVG waves at the top border. Centered on the screen is a clean, glassmorphic terminal window. 
*   **Header:** Displays "Geva Kipper" in a bold, elegant sans-serif font (like Inter or Montserrat). 
*   **Current Role:** Below the name, a modern, minimalist badge reads: "Software Engineer, currently building scalable solutions and driving innovation at Google."
*   **AI Redesign Subtitle:** Designed to look like a blinking sonar alert using a monospaced technical font (e.g., JetBrains Mono): *"SYSTEM NOTICE: Oceanic topography is completely re-designed each day by AI and is therefore ever-changing."*
*   **Call to Action:** A pulsing, glowing downward arrow labeled "Initiate Dive" that prompts the user to start scrolling. A vertical "Depth Meter" is fixed to the right side of the screen, tracking scroll progress in meters.

**The Sunken Treasures (Projects 1-7)**
As the submarine descends, it encounters projects laid out as alternating "sunken anomalies" on the left and right sides of the descent path. When scrolling into view, a CSS-animated "sonar sweep" (a radial green or blue line) passes over each treasure, triggering the glassmorphic HUD card to fade in with the project details. 
*   **Box 1 (GlglStats):** The visual is a sunken, seaweed-covered DJ turntable. The HUD card presents the project title with a visualizer graphic of a soundwave morphing into a binary compression string. The text explains the Python LZ77 pop-chart repetitiveness analysis. It includes two glowing, pill-shaped buttons: "Sonar Link: Project Site" (glglstats.geekip.dev) and "View PyData TLV Log" (YouTube link).
*   **Box 2 (Ahuzat Dibuk):** The visual is a grid of glowing, monolithic stone tablets on the seafloor resembling a parking lot, emitting a thermal heat-map glow (red/orange/green). The text details the Python scraping/data visualization for Tel-Aviv parking. Links are styled as intercepted transmissions: "Channel 13 Broadcast", "Mako Intel", and "Facebook Dispatch".
*   **Box 3 (2D Part Orienting):** Visualized as scattered, glowing geometric polygons floating in the water. The HUD text explains the C++ computational geometry implementation and push-plan algorithm. A single, technical-looking terminal button links to the open-source code and library info.
*   **Box 4 (The Turing Jest):** The visual is an ancient Greek amphitheater half-buried in the sand, with two robotic theater masks facing off. The HUD describes the LMSYS-like arena where LLMs compete in pun-making. A button labeled "Enter the Arena" links to jest.geekip.dev.
*   **Box 5 (Rhymes of Israel):** Visualized as an antique, brass deep-sea diving helmet emitting holographic newspaper clippings that turn into bubbles. The text discusses the LLM Chrome extension rewriting Israeli news into yearbook-style comedic rhymes. A button branded with the Chrome logo links to the Webstore installation.
*   **Box 6 (Hatul Basak):** The visual is an enigmatic, glowing sphinx-cat statue resting on an underwater ledge. The HUD outlines the Hebrew catfishing.net port, the automated AI daily challenges, and user-generated puzzles based on Wikipedia trivia. A vibrant button links to the game site to play and create puzzles.
*   **Box 7 (The Meta Portfolio):** The visual is a highly polished, floating mirror reflecting a wireframe version of the user's submarine back at them. The HUD notes the self-referential nature of presenting the portfolio site itself as a project. A self-referencing loop button directs to geekip.dev.

**The Ocean Floor (Box 8 & Footer)**
*   **Box 8 (Social Links):** The background is now pitch black. The site implements a JavaScript mouse-tracking spotlight effect (a CSS radial gradient mask) that acts as the submarine's headlights, revealing the ocean floor wherever the cursor moves. Hidden in the dark are three glowing bioluminescent deep-sea pearls. When illuminated by the cursor spotlight, they expand into sleek icons for GitHub (github.com/gevak), LinkedIn (linkedin.com/in/gevakip), and Facebook (facebook.com/geva.kipper).
*   **Footer Structure:** At the absolute bottom rests a high-tech, glowing underwater research base. A minimalist readout at the base credits the design: *"Topography generated by AI. Architecture by [Insert LLM Model Name, e.g., GPT-4 / Claude]."* 
*   **The Mission Timer:** Attached to the AI credit is a digital, red-LED styled stopwatch. Using JavaScript, this timer dynamically calculates and continuously ticks up the exact hours, minutes, and seconds that have passed since the temporal anchor point: `2026-04-15 01:54:13`, serving as a mysterious, ever-running mission clock at the bottom of the sea.
