The website is a single-page, infinite-scroll descent into a neo-gothic, cyberpunk "Mad Scientist Laboratory." The global color palette uses an obsidian black background accented by bioluminescent neon greens, toxic purples, and electric blues. The background features a subtle CSS parallax effect of thick, metallic pipes routing data and slow-rising, translucent bubbles. A bespoke cursor resembling a glowing laser pointer interacts with the environment, casting a faint, colored light-radius on the elements it hovers over. Typography pairs a sleek, ultra-modern sans-serif (like Inter or Roboto) for readability with a jagged, terminal-style monospace (like Fira Code) for headers and data points.

At the very top, the "Main Control Console" serves as the header. The developer’s name, **Geva Kipper**, is displayed in a massive, flickering neon tube font that occasionally sparks or glitches. Beneath the name is a crisp, monospace readout: *"Currently synthesizing data and engineering scalable algorithms as a Software Engineer at Google."* Below this, a metallic, hazard-striped banner frames the subtitle: *"WARNING: This environment is re-engineered daily by Artificial Intelligence. Expect volatile anomalies and mutating layouts."* 

Beneath the header lies the main content area, designed as a masonry grid of eight "Containment Vats" replacing traditional portfolio cards. Each vat uses CSS glassmorphism—frosted edges, specular highlights, and high-backdrop-blur—to appear as a cylindrical glass tube protruding from the screen. Inside each vat, a dynamic background of falling, bubbling code (simulated via JavaScript canvas) acts as the "fluid." When the user hovers over a vat, the fluid "boils" with rapid, intense bubbling animations, and the glass rim glows brightly with a project-specific neon color. The project information floats inside these vats like suspended holographic projections.

**Vat 1: Project GlglStats** 
The fluid is toxic green, with a visual motif of compressing sound waves. The text floats in the center, detailing the LZ77 compression analysis of Israeli pop charts. Below the description are two interactive "valve" buttons—one linking to the live experiment (https://glglstats.geekip.dev) and another featuring a YouTube icon for the PyData TLV presentation (https://www.youtube.com/watch?v=jrKAtOsj1Lo). 

**Vat 2: Ahuzat Dibuk** 
The fluid is a deep, glowing thermal orange, with a faint, pulsing heat-map grid animating in the background. The text describes the Tel-Aviv parking data scraping project. Three illuminated "data-port" links at the bottom connect to the media coverage on Channel 13 (https://13tv.co.il/item/news/domestic/internal/parking-902956407/), Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), and the original Secret Tel-Aviv Facebook broadcast (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).

**Vat 3: 2D Part Orienting** 
The fluid is electric blue, featuring constantly rotating, wireframe polygons floating among the code bubbles. The text outlines the C++ push-plan algorithm injected into the open-source computational geometry library. A glowing blueprint-style button links to the source at the TAU CGL site (https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/).

**Vat 4: The Turing Jest** 
The fluid is ultraviolet, containing a visual of two robotic theater masks clashing. It explains the LMSYS-like arena where LLMs battle for comedic supremacy. A "Release Monsters" button links directly to the arena (https://jest.geekip.dev).

**Vat 5: Rhymes of Israel** 
The fluid is a radioactive yellow. The hologram displays a serious newspaper clipping that burns away to reveal a comedic highschool yearbook page. It presents the LLM headline-rhyming Chrome extension, with a sleek "Install Modification" button pointing to the Chrome Web Store (https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb).

**Vat 6: Hatul Basak** 
The fluid is cyan, housing a digital, pixelated cat that swats at the rising CSS bubbles. It describes the Hebrew Wikipedia trivia catfishing port, the AI-generated daily challenges, and user-made puzzles. A retro-arcade-styled button invites users to "Initiate Game Sequence" (https://hatulbasak.geekip.dev).

**Vat 7: Portfolio Site**
This is the "Meta Vat." It contains a shimmering silver fluid with an infinite mirror effect (achieved via nested CSS transforms) reflecting the website itself. It credits the self-referential nature of the project and houses a self-contained feedback loop button linking to https://geekip.dev.

**Vat 8: Social Connectivity Terminal** 
Unlike the others, this vat is partially shattered, with thick, sparking optical cables spilling out of the glass. It serves as the contact module. Deep within a stabilizing red fluid, three prominent, pulsing nodes hold the icons and links for the developer's GitHub (github.com/gevak), LinkedIn (linkedin.com/in/gevakip), and Facebook (facebook.com/geva.kipper).

At the very bottom of the page sits the "AI Core," the anchor of the footer. It is designed as a massive, glowing server rack. A sleek, terminal-style text crawl spells out: *"Architecture and genetic sequencing of this laboratory synthesized by your AI overseer."* Directly beneath this, etched into a digital, crimson seven-segment display, is an active JavaScript stopwatch. It counts upwards in real-time (HH:MM:SS format), labeled as *"Temporal distance since containment breach (2026-05-23 02:28:56)."*
