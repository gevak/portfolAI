The overall aesthetic is a "Neo-Retro Desktop OS"—a blend of modern web capabilities (smooth animations, subtle glassmorphism, crisp vector scaling) with nostalgic 90s graphical user interface elements. The background is a rich, dithered teal with faint, gently rolling CRT scanlines. In the center of this desktop background, treated like a built-in wallpaper watermark, is the main typography: "GEVA KIPPER", written in a large, elegant serif font. Directly underneath is a sharp monospace subtitle: "Software Engineer currently building at Google." Below this sits a secondary, slightly smaller, glowing subtitle: "System architecture and UI generated by AI: This portfolio is re-designed every day, ensuring an ever-changing digital landscape." 

The core interactable elements are seven distinctly styled, draggable application windows scattered across the screen in a cascading layout. Each window features a classic thick frame, a top title bar with horizontal striping, and simulated minimize/maximize/close buttons. Modern JavaScript allows these windows to be smoothly dragged, brought to the front on click, and minimized to an omnipresent bottom taskbar. To maintain elegance, the typography inside the windows uses a high-legibility modern sans-serif, contrasting beautifully with the retro pixel-art icons and borders. 

Window 1, "GlglStats", is designed to resemble a classic data terminal or retro media player. The window is split into two panes. The left pane features a localized, pixelated visualizer representing LZ77 compression patterns, alongside a short paragraph explaining the Python project’s measurement of Israeli pop-song repetitiveness. The right pane houses two chunky, interactive retro buttons: one labeled "Run glglstats.exe" linking to https://glglstats.geekip.dev, and another styled as a VHS tape icon labeled "Play PyData Presentation" linking to the YouTube video (https://www.youtube.com/watch?v=jrKAtOsj1Lo).

Window 2, "Ahuzat Dibuk", takes the visual form of a vintage radar or GIS mapping application. Grounded in a dark-mode, high-contrast UI, it displays a stylized, static heatmap of Tel-Aviv parking occupancies. The descriptive text sits overlaid in a semi-transparent inset box, explaining the Python scraping and visualization logic. Along the bottom of this window runs a classic "News Ticker" banner, containing three distinct hyperlinks disguised as active news alerts: Channel 13 coverage (https://13tv.co.il/item/news/domestic/internal/parking-902956407/), Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), and the original viral Secret Tel-Aviv Facebook post (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).

Window 3, "2D Part Orienting", is styled like an early 90s CAD (Computer-Aided Design) software tool. The application window features a faux toolbar at the top and a prominent central canvas displaying an abstract geometric polygon with simulated "push plan" vectors. The text explaining the C++ computational geometry implementation is formatted as a "System Log" at the side. A prominent "View Source & Docs" button, styled with a distinct bevel, links out to https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/.

Window 4, "The Turing Jest", heavily leans into a nostalgic IRC chatroom or dual-pane BBS interface. The window indicates an LMSYS-like arena where users judge LLM-generated puns. Half the window shows a simulated chat transcript of two AIs delivering jokes, while the other half contains the project description and a glowing "Enter Arena" hyperlink formatted as a command-line prompt, pointing to https://jest.geekip.dev.

Window 5, "Rhymes of Israel", mimics a classic "Sticky Note" desktop widget crossed with a retro newspaper clipping. The background of this specific window is a pale yellow paper texture. The text explains the use of an LLM via a Chrome extension to rewrite Israeli news headlines into comedic, high-school yearbook style rhymes. At the bottom right corner, a prominent, colorful puzzle-piece icon—referencing a browser extension—serves as the link to the Chrome Web Store (https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb).

Window 6, the "Portfolio Project", is framed as a nostalgic web browser window (reminiscent of early Netscape), complete with a URL bar displaying "https://geekip.dev". The interior content is highly self-referential and 'meta', playfully acknowledging that this is the very site the user is interacting with. The text is formatted like a classic "Welcome to the World Wide Web" landing page, with a hyperlinked spinning globe or a simple blue underlined link serving as the self-referential anchor to https://geekip.dev.

Window 7 operates as the "Network Neighborhood" or "System Directory". It is a smaller, more compact window displaying three large, highly-detailed desktop icons arranged in a grid: a floppy disk labeled "GitHub" (github.com/gevak), a rolodex labeled "LinkedIn" (linkedin.com/in/gevakip), and a vintage address book labeled "Facebook" (facebook.com/geva.kipper). Hovering over these icons triggers a modern, smooth CSS lift effect and a glow, marrying the retro look with modern tactile feedback.

Anchoring the bottom of the screen is a classic OS gray Taskbar, spanning the full width of the viewport. On the far left, in place of a traditional 'Start' button, sits an embossed button reading "System Details". Next to it, permanently engraved into the taskbar frame, is the credit line reading: "OS Architecture mapped and generated by AI." On the far right, exactly where a system tray clock belongs, is a digital stopwatch styled in green segment-display LCD font. A live JavaScript function continuously updates this timer—calculating from the target epoch of "2026-04-02 01:47:28"—displaying the exact hours, minutes, and seconds that have ticked by, serving as a dynamic, living heartbeat for the portfolio.
