The overarching design combines the nostalgic interactivity of a top-down, 8-bit RPG world map with ultra-modern, elegant web aesthetics (like glassmorphism and smooth CSS animations). The background is an expansive HTML5 Canvas or CSS Grid representing a stylized, dark-themed pixelated landscape (deep charcoals, muted neon accents). The user controls a customized 8-bit avatar of Geva using their keyboard (WASD/Arrows) or by clicking/tapping to path-find. As the hero moves through the map, distinct geographical zones represent different career achievements. 

Floating persistently at the top of the viewport—styled like a clean, modern video game HUD—is the header. It features the name "Geva Kipper" in a bold, pixel-art inspired but highly legible font, followed by a sleek sans-serif sentence: "Software Engineer at Google, crafting internet-scale solutions." Directly below this is the subtitle, rendered in a shifting, gradient text to imply automation: "This environment is re-designed day-by-day by Artificial Intelligence; you are exploring today's iteration."

The eight items requested serve as interactive landmarks or "points of interest" on the map. When the avatar walks into a landmark's proximity radius, a gorgeous, frosted-glass dialogue box (a modern twist on the classic RPG text box) gracefully slides onto the screen containing the project details, typography, and interactive modern hyperlinks.

**The Map Zones and Interactive Boxes:**
1. **The GlglStats Tower:** A landmark resembling a pixelated radio antenna or equalizer. Entering it triggers a box presenting the Python LZ77 compression pop-chart project. The UI box must feature a primary link to `https://glglstats.geekip.dev` and an embedded thumbnail or clean button linking to the PyData TLV YouTube presentation (`https://www.youtube.com/watch?v=jrKAtOsj1Lo`). 
2. **Ahuzat Dibuk District:** Visualized on the map as a mini concrete parking lot glowing with pulsing neon heat-map colors. The pop-up box will detail the Python scraping and data visualization project for Tel-Aviv parking. It must include an elegant row of media links: Channel 13 (`https://13tv.co.il/item/news/domestic/internal/parking-902956407/`), Mako (`https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm`), and the original Secret Tel-Aviv Facebook publication (`https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/`).
3. **The Geometry Temple:** A beautifully shifting, algorithmic 2D polygon structure. Approaching it drops down the '2D Part Orienting' box, describing the open-source C++ computational geometry push-plan algorithm. A prominent button will link to the code and info: `https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/`.
4. **The Turing Jest Colosseum:** An arena-style building adorned with pixelated comedy and tragedy masks. The triggered box will pitch the LMSYS-like arena for LLM pun battles, accompanied by a glowing link to `https://jest.geekip.dev`.
5. **The Rhymes of Israel Newsstand:** A classic street-corner news kiosk emitting tiny musical-note sprites. The interface box will present the Chrome Extension that rewrites news headlines into highschool yearbook rhymes via LLM, providing a direct call-to-action button to the Chrome Web Store: `https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb`.
6. **The Hatul Basak Clearing:** A whimsical forest patch inhabited by pixelated cats and puzzle pieces. The associated UI box will detail the Hebrew Wikipedia Triivia game (a port of catfishing.net) and its AI-generated daily challenges, linking to `https://hatulbasak.geekip.dev` for users to play and create puzzles.
7. **The Meta Portal:** A swirling, glowing mirror on the map. Stepping into it triggers a self-referential glass box highlighting this exact AI-generated portfolio project, inviting the user to step further into the matrix with a link to `https://geekip.dev`.
8. **The Crossroads Tavern (Socials):** A welcoming wooden signpost or inn at the starting area. The dialogue box here serves as the contact hub, featuring clean, minimalist vector icons linking out to `github.com/gevak`, `linkedin.com/in/gevakip`, and `facebook.com/geva.kipper`.

Anchored to the absolute bottom of the screen is the final UI element: a retro RPG dialogue frame with a border that softly pulses. Inside, a typewriter-effect text credits the exact AI model that generated the page. Next to this credit, rendered in a digital watch-style monospace font, is a relentlessly ticking stopwatch actively displaying the hours, minutes, and seconds elapsed since `2026-06-15 03:31:36`, anchoring the ever-changing theme in an exact point in time.
