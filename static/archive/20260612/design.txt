**Overall Concept & Visual Theme**
The user interface is built around a photorealistic, 3D CSS-rendered vintage industrial filing cabinet (think cold-rolled steel, brass handles, and weathered paper labels) sitting in the center of a dimly lit, minimalist room. The cabinet serves as the primary navigation. When a user interacts with a drawer, it smoothly slides open toward the camera. Instead of holding manila folders, the open drawer reveals a glowing, high-contrast digital screen nestled within the physical metal casing. This screen displays a modern, dark-mode code editor (like VS Code or a Unix terminal), presenting the project details through syntax-highlighted code, JSON, and terminal logs.

**Header & Subtitle (The Cabinet Top)**
Resting on top of the cabinet is a heavy, engraved brass nameplate. It elegantly displays the developer’s name: **Geva Kipper**, followed by a sleek sub-heading: "Software Engineer at Google, utilizing code to build massive-scale solutions." 
Taped diagonally across the corner of the brass plate is an animated, glowing cyberpunk sticky-note that serves as the subtitle. It reads: "SYSTEM ALERT: This portfolio's architecture and design are re-configured and re-designed every single day by Artificial Intelligence. You are viewing today's ephemeral iteration."

**The Drawers (Project Sections)**
The main body of the single page is a vertical scroll experience that brings each of the 8 drawers into focus. As the user scrolls, the active drawer physically protrudes (using smooth CSS `transform: translateZ`). Clicking a drawer fully opens it to reveal the digital/code interface inside. 

*   **Drawer 1: GlglStats**
    *   *Exterior Label:* Typewriter font on smudged paper reading `LZ77_Audio.py`.
    *   *Interior:* Python syntax highlighting. The description of measuring repetitiveness of Israeli pop-chart songs via LZ77 compression is formatted as a Python triple-quote docstring. A glowing inline terminal window contains the embedded PyData TLV YouTube presentation (`https://www.youtube.com/watch?v=jrKAtOsj1Lo`), and a clickable `return` statement links to `https://glglstats.geekip.dev`.
*   **Drawer 2: Ahuzat Dibuk**
    *   *Exterior Label:* `TLV_Parking_Scraper.sh`.
    *   *Interior:* Designed to look like a raw terminal running a data scraping pipeline. The background features a faint, ASCII-art heatmap. The descriptions of the Tel-Aviv parking lot occupancy statistics are formatted as continuous log outputs. Terminal hyperlinks (`[Ctrl+Click to open]`) route out to the Channel 13 and Mako news stories, as well as the original Secret Tel Aviv Facebook post.
*   **Drawer 3: 2D Part Orienting**
    *   *Exterior Label:* `push_plan.cpp`.
    *   *Interior:* C++ IDE environment styling. The open-source computational geometry library and push-plan algorithm descriptions are formatted as class definitions and inline comments. A bright, clickable `#include <project_page>` points directly to `https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/`.
*   **Drawer 4: The Turing Jest**
    *   *Exterior Label:* `jest_arena.json`.
    *   *Interior:* Formatted strictly as a colorful, nested JSON object. Keys like `"description"` hold the explanation of the LMSYS-like arena for LLM pun battles, and `"live_url"` holds the direct glowing link to `https://jest.geekip.dev`.
*   **Drawer 5: Rhymes of Israel**
    *   *Exterior Label:* `manifest.json`.
    *   *Interior:* A hybrid between a Chrome Extension manifest file and Javascript code. The LLM highschool yearbook comedic rhyming functionality is written out as a Javascript function block. A stylized "Install Extension" button mimics a Chrome Web Store UI element floating above the code, linked to the provided URL.
*   **Drawer 6: Hatul Basak**
    *   *Exterior Label:* `trivia_graph.yaml`.
    *   *Interior:* YAML syntax structure. The description of the Wikipedia-based trivia game, Hebrew catfishing.net port, and AI daily challenges sits as structured data. The call to action to play and create puzzles is a prominent, retro-styled glowing terminal text link pointing to `https://hatulbasak.geekip.dev`.
*   **Drawer 7: Portfolio Website**
    *   *Exterior Label:* `index.html`.
    *   *Interior:* Highly meta HTML syntax aesthetics. The description of this very AI-generated portfolio site is wrapped in `<meta name="description">` and `<div>` tags. A `<a href="https://geekip.dev">` tag pulses with a neon CSS animation to invite infinite recursion clicks.
*   **Drawer 8: Social Links**
    *   *Exterior Label:* `Network_Config`.
    *   *Interior:* A command-line network diagnostic screen. Running "ping" scripts visualize the connection to external networks. Three large, ASCII-style command buttons trigger the links out to `github.com/gevak`, `linkedin.com/in/gevakip`, and `facebook.com/geva.kipper`.

**Footer (The Floor of the Room)**
At the base of the cabinet, lying on the dusty concrete floor texture, is an old-school, glowing red Nixie-tube digital display wired into the cabinet. It serves two purposes:
First, a stamped metal plaque beside it credits the site's creation: "Site engineered and generated by Claude AI" (or whichever model generates the final code). 
Second, a JavaScript-powered stopwatch continuously counts upward in a monospace red-glow font, tracking exact Hours, Minutes, and Seconds elapsed since the epoch timestamp: `2026-06-12 02:52:54`.
