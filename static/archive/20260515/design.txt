**Overall Theme and Visual Language**
The website utilizes a "Neo-Retro" 1990s graphical user interface, specifically mimicking Windows 95 or Windows 3.1. The background is a solid classic teal (`#008080`), acting as the "Desktop". All UI elements—windows, buttons, and panels—use signature 90s beveled edges (crisp CSS outset and inset borders in light gray, white, and dark gray). Typography contrasts classic pixelated fonts (like MS Sans Serif or a modern equivalent like 'VT323' and 'Press Start 2P') for window titles and buttons, with sleek, modern sans-serifs for the actual readable content, ensuring an elegant and usable experience. 

**The Desktop Interface & Hero Section**
The user lands on the desktop where a prominent, un-closable window is centered, titled "System_Properties.exe". Inside, the layout mimics a classic computer spec sheet. 
*   **Developer Info:** It features a highly dithered, 16-bit style avatar of Geva Kipper. Next to it reads the primary text: "User: Geva Kipper," followed by, "Current Operation: Engineering software solutions at Google."
*   **Dynamic Subtitle:** Beneath this is a section labeled "Display Drivers:" which incorporates the subtitle: "Warning: This graphical interface is dynamically re-designed each day by AI; environment is ever-changing." This text pulses slightly or uses a retro marquee effect.

**The Application Windows (The 8 Sections)**
The projects are presented as individual open "Program Windows" scattered in a masonry-style, cascading layout across the teal desktop. Each window has a classic blue title bar with standard minimize/maximize/close dummy buttons.

1.  **GlglStats (Performance Monitor Window):** Styled like a classic Task Manager or Resource Monitor. The inside displays static retro-styled jagged chart lines. Text explains the LZ77 compression and Israeli pop-chart repetitiveness analysis. An embedded YouTube UI containing the PyData TLV presentation (https://www.youtube.com/watch?v=jrKAtOsj1Lo) sits beneath it, disguised as an old Windows Media Player instance. A prominent hyperlink leads to https://glglstats.geekip.dev.
2.  **Ahuzat Dibuk (Disk Defragmenter Window):** The UI is designed to look like a disk defragmenter, but the "blocks" form a Tel-Aviv parking heatmap. The text details the Python scraping and data visualization project. The news links (News 13: https://13tv.co.il/item/news/domestic/internal/parking-902956407/ and Mako: https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm) are styled as "System Alerts", while the original Facebook post (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/) is styled as a readable "ReadMe.txt" file link.
3.  **2D Part Orienting (MS Paint Window):** Designed like heavily simplified 90s CAD or MS Paint. The inner canvas shows a wireframe polygon with simulated "push plan" vectors around it. The text describes the C++ implementation. A prominent "View Source.bat" button links to https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/.
4.  **The Turing Jest (MS-DOS Prompt Window):** A stark contrast to the gray GUIs, this section has a black background with green or light gray terminal text. Two pixelated ASCII-art comedy/tragedy masks flank the text, describing the LMSYS-like arena for LLM pun battles. An interactive flashing cursor invites the user to click the link: https://jest.geekip.dev.
5.  **Rhymes of Israel (Notepad Window):** Designed as an open Notepad application. The text, in a Courier-style font, describes the Chrome Extension that rewrites news headlines into comedic rhymes. A floating, classic 3.5" floppy disk icon acts as the "Install Extension" button, pointing to the Chrome Web Store: https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb.
6.  **Hatul Basak (Minesweeper/Trivia Window):** The window takes visual cues from classic Windows games, featuring a pixel-art cat avatar. The text details the Hebrew Wikipedia-based Trivia port of catfishing.net and its daily AI challenges. A prominent "Play_New_Game.exe" button links to https://hatulbasak.geekip.dev.
7.  **Portfolio Website Project (Early Web Browser Window):** Styled as a vintage browser (Netscape Navigator or early Internet Explorer). The window displays an "infinite mirror" or a recursive, wireframe graphic of the teal desktop itself. The text acknowledges the meta-nature of the project, with an address bar linking securely to https://geekip.dev.
8.  **Social Protocols (Network Neighborhood):** Instead of a text-heavy window, this is an open folder window set to "Large Icons" view. It displays three prominent, 90s-styled pixel-art icons mapping to off-site directories: a global network icon for LinkedIn (linkedin.com/in/gevakip), a connected servers icon for GitHub (github.com/gevak), and a classic rolodex or address book icon for Facebook (facebook.com/geva.kipper).

**The Taskbar & System Tray (Footer Section)**
Fixed to the bottom of the viewport is the classic gray Windows Taskbar. 
*   **Left Side:** A classic "Start" button featuring a custom pixel-art abstract logo. Beside it, inset boxes show minified tabs of the opened desktop programs.
*   **Middle Section:** An inset panel featuring a scrolling marquee or static status text: `AI.sys core loaded: Design generated by [Insert AI Model Name]`.
*   **Right Side (System Tray):** Set against a darker gray inset box, mirroring the classic OS system clock, is the stopwatch. It displays in a highly legible digital font format `HH:MM:SS`, continuously ticking up the exact hours, minutes, and seconds elapsed since the anchor timestamp of `2026-05-15 02:36:18`.
