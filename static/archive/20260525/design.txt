**Global Aesthetic and Layout**
The website is styled as a functional, elegant vintage graphical operating system (reminiscent of Classic Mac OS or early NeXTSTEP), blending retro 8-bit visual language with smooth, modern web capabilities. The color palette revolves around a crisp monochromatic base (stark whites, deep blacks, and utilitarian grays) accented with vibrant, modern neon hues (cyan, amber, magenta) for interactive elements. The background is a subtle, dithered pixel-art pattern. The layout operates as a "Desktop" where the eight required sections are styled as pre-opened, partially overlapping OS windows that the user can drag, bring to focus, and interact with. On mobile, these windows elegantly snap into a vertically scrolling masonry grid. Typography utilizes a crisp, modern pixel font for headers and UI elements, paired with a highly legible monospace font for body text.

**Top System Menu Bar**
A sticky, persistent status bar at the very top of the screen.
*   **Left Side (System Info):** Features an Apple-style drop-down menu icon, followed by the text: **Geva Kipper**. Beside it, in a slightly muted color, a single elegant sentence: "Currently engineering modern solutions at Google."
*   **Right Side (System Status):** A subtitle styled as a system notification reading: "Status: Architecture is re-designed each day by AI and is thus ever-changing." 

**The Desktop Windows (The 8 Sections)**
Each section is housed in a distinct OS-style window featuring a classic title bar, etched window borders, a drop shadow, and faux minimize/maximize/close buttons. 

*   **Window 1: GlglStats (The Media Player)**
    *   *Design:* Styled like a vintage media player (e.g., early Winamp) or a spectrum analyzer. The background of the window content is dark CRT green.
    *   *Content:* Displays the python project description alongside a visual generated "sound wave" composed of ASCII blocks representing the LZ77 repetition patterns of Israeli pop charts. 
    *   *Links:* A glowing "Launch Software" button links to https://glglstats.geekip.dev. A "Play Video" button (styled as an old VHS play symbol) links to the PyData TLV presentation on YouTube.

*   **Window 2: Ahuzat Dibuk (The Radar Monitor)**
    *   *Design:* Designed as a "System Diagnostics" or "Radar" terminal. Features a pixelated, heatmap-style graphic of a city grid.
    *   *Content:* Text explaining the Python scraping and visualization project for Tel-Aviv parking occupancy. 
    *   *Links:* Links to the news coverage (13tv, Mako) are styled as raw "Data Logs" or "Press Dispatches", while the Secret Tel-Aviv Facebook link is designed as a "Community Forum Post" hyperlink icon.

*   **Window 3: 2D Part Orienting (The CAD Wireframe)**
    *   *Design:* Styled as a retro blueprint or CAD software window. The content background is a subtle grid pattern with a geometric polygon rendered in bright cyan vector lines.
    *   *Content:* Describes the C++ open-source computational geometry implementation for polygon push-planning.
    *   *Links:* An "Inspect Source/Documentation" button (styled like a floppy disk save icon) links to the Tel Aviv University project page.

*   **Window 4: The Turing Jest (The Chat Terminal)**
    *   *Design:* A split-screen MS-DOS style command prompt window.
    *   *Content:* Describes the LMSYS-like arena for LLM pun battles. The text types out with a blinking cursor animation.
    *   *Links:* An "Enter the Arena" command prompt link directing to https://jest.geekip.dev.

*   **Window 5: Rhymes of Israel (The Digital Newspaper)**
    *   *Design:* Styled like a 90s HyperCard stack or an early digital newspaper clipping, utilizing a black-on-white high-contrast serif font overlaying a textured digital paper background.
    *   *Content:* Describes the Chrome extension that rewrites news headlines into comedic highschool yearbook rhymes using an LLM.
    *   *Links:* A prominent "Install Extension" button styled as an old Google Chrome installer package directing to the Chrome Web Store.

*   **Window 6: Hatul Basak (The 8-Bit Game Menu)**
    *   *Design:* A retro arcade or early PC game launch window, featuring 8-bit custom pixel art of a cat. 
    *   *Content:* Explains the Hebrew port of catfishing.net, the AI-generated daily challenges, and user-generated themed puzzles. 
    *   *Links:* A pulsing "Insert Coin / Play Now" button links to https://hatulbasak.geekip.dev.

*   **Window 7: Portfolio (The Virtual Machine)**
    *   *Design:* A "System Properties" window utilizing the Droste effect. It contains a miniature, stylized vector illustration of the desktop layout the user is currently looking at.
    *   *Content:* A meta-reference text noting that the current site is itself a portfolio project.
    *   *Links:* A "Refresh Virtual Environment" button linking back to https://geekip.dev.

*   **Window 8: Social Connections (The Address Book)**
    *   *Design:* A "Network Connections" or "Address Book" dialog box.
    *   *Content:* Links to the developer's external profiles modeled as rolodex cards or network nodes.
    *   *Links:* GitHub, LinkedIn, and Facebook profiles, each equipped with minimalist, pixelated versions of their respective corporate logos.
