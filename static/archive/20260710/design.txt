**Overall Aesthetic & Layout**
The website is designed as a sleek, isometric 3D virtual vending machine standing against a minimalist, dark-gradient background (deep indigo fading to black). The design utilizes a modern "glassmorphism" style—frosted acrylic panels, glowing neon-grid shelves, and metallic chassis accents. The entire page is a single, centered, viewport-filling interactive interface. Typography is a mix of a clean geometric sans-serif for readable blocks and a glowing dot-matrix monospace font for digital displays.

**The Top Panel (Header & Marquee)**
The top of the vending machine features a scrolling, glowing LED marquee display. 
*   **Main Display:** Reads "Geva Kipper — Software Engineer currently building things at Google."
*   **Sub-Display:** Just beneath it, a smaller digital panel with amber text displays the subtitle: "NOTICE: This interface is re-designed each day by an Artificial Intelligence. Layout and aesthetics are ever-changing." 

**The Product Grid (The Interactive Dispenser)**
The central area of the vending machine is a glowing, glass-enclosed display featuring 8 unique "products" hanging on silver CSS-animated spirals. The layout is a 2x4 grid. When a user clicks a product's alphanumeric code (e.g., A1, B4) on the right-side numeric keypad, the spiral turns, the item drops into the dispenser tray below, and clicking the tray opens a sleek, frosted-glass modal overlay. This modal presents the project details formatted like "Nutrition Facts" or "Product Specs," alongside stylish, tactile buttons for the links.

**The Bottom Panel (Coin Return & Maintenance Area)**
At the very bottom of the machine, framed by a brushed steel texture matching a coin-return slot and an air vent, sits the footer data.
*   **Credit Plaque:** Engraved text styled to look stamped into the metal: "Machine Architecture & Assembly by AI Model."
*   **Maintenance Timer:** Beside the plaque is a glowing red seven-segment digital clock. Using JavaScript, this acts as a live stopwatch ticking up in real-time `[HOURS : MINUTES : SECONDS]`, dynamically calculating the exact time elapsed since the baseline timestamp: `2026-07-10 02:14:58`.
