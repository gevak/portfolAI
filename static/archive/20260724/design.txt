The website is a single-page immersive digital universe. The visual foundation is a deep-space, interactive HTML5 Canvas background in ultra-dark indigo and void black, populated by parallaxing stars and subtle, slowly shifting nebulas in faint hues of purple and cyan. The typography is modern and futuristic, utilizing "Space Grotesk" for headings and clean "Inter" for body text. 

At the top-left of the viewport, a fixed, sleek glassmorphism panel—translucent with a 1px frosted-glass border—introduces the developer. The main header reads "Geva Kipper" in a glowing starlight-white gradient. Below it, a single sentence states: "Software Engineer at Google, building scalable infrastructure and solving complex algorithmic challenges." A smaller, stylized subtitle in faint neon blue reads: "Phase-shifted reality: This digital universe is re-imagined and re-designed daily by AI, making every visit an ephemeral and ever-changing experience."

The core interaction model relies on a draggable, pannable, and zoomable 2D space environment. Scattered across the viewport are eight distinct star clusters. When the user hovers over a cluster, glowing SVG lines animate (drawing themselves point-to-point) to connect the stars, forming a constellation that represents the project's theme (e.g., a music note, a car, a cat silhouette). Clicking a glowing "primary star" inside the constellation triggers a smooth camera zoom to that location, dims the background universe, and fades in an "Astronomical Data Plate"—a sleek, floating card containing the project details. 

The eight project panels are designed as follows:
1. **The GlglStats Constellation (Shaped like a sound wave):** The floating card presents this Python project, explaining its use of LZ77 compression to measure and visualize repetitiveness in Israeli pop-chart songs. It features two glowing action buttons: a primary link launching into the live heatmaps (https://glglstats.geekip.dev) and a secondary stylized Play button that opens the PyData TLV conference presentation (https://www.youtube.com/watch?v=jrKAtOsj1Lo).
2. **The Ahuzat Dibuk Constellation (Shaped like a parking grid):** This card focuses on data scraping and visualization. It describes the Python tool used for generating Tel-Aviv parking lot occupancy heat-maps for residents. Below the text, three distinct news-feed style buttons route users to the media coverage on Channel 13 (https://13tv.co.il/item/news/domestic/internal/parking-902956407/), Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), and the original viral Secret Tel Aviv publication (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).
3. **The 2D Part Orienting Constellation (Shaped like an irregular polygon):** Designed with sharp, angular UI elements to reflect computational geometry, this box details the C++ algorithm implementation for constructing polygon "push plans." A hyperlinked terminal-style arrow points to the open-source library and code details hosted at TAU (https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/).
4. **The Turing Jest Constellation (Shaped like a jester's hat):** A playfully styled card introducing the LMSYS-like arena for LLM pun battles. The UI features a vs-screen motif and a prominent neon button inviting users to rate the AI jokes at the live site (https://jest.geekip.dev).
5. **The Rhymes of Israel Constellation (Shaped like a quill):** This panel describes the Chrome extension that uses an LLM to rewrite Israeli news headlines into comedic, highschool-yearbook style rhymes. A vibrant, animated "Add to Chrome" style button links directly to the Chrome Web Store (https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb).
6. **The Hatul Basak Constellation (Shaped like a cat):** This card details the Hebrew port of catfishing.net. The text highlights the Wikipedia-based trivia, the AI-generated daily challenges, and user-created puzzles. A large interactive button links to the game to play or create puzzles (https://hatulbasak.geekip.dev).
7. **The Meta-Portfolio Constellation (Shaped like an infinite spiral):** A self-referential, elegantly designed card acknowledging the recursion of presenting the site currently being viewed. Its border features a continuously looping CSS gradient, with a central anchor linking back to the domain itself (https://geekip.dev).
8. **The Comm-Link Constellation (Shaped like a satellite):** This final box serves as the contact and social hub. It mimics a retro-futuristic communication terminal, featuring three glowing orbit rings, each containing an icon and link to the developer's external profiles: GitHub (github.com/gevak), LinkedIn (linkedin.com/in/gevakip), and Facebook (facebook.com/geva.kipper).

At the absolute bottom center of the viewport, anchored above the cosmic background, sits a minimalist footer in a monospace technical font. It reads: "Celestial architecture generated by Claude 3.5 Sonnet. Ephemeral Instance Uptime:" followed by a live, ticking JavaScript stopwatch displaying the exact hours, minutes, and seconds elapsed since the anchor timestamp: 2026-07-24 02:02:19.
