The website is a single-page application heavily inspired by premium movie review aggregation sites (like Letterboxd or Rotten Tomatoes) mixed with cinema-theater aesthetics. The overall color palette is "Midnight Screen" (deep charcoal and pitch black) accented with glowing "Cinematic Gold" and vivid "Tomatometer Red." The UI relies on smooth CSS transitions, glowing neon hover states, and dynamic JavaScript layouts to feel like a high-end, editorial film catalog.

**The Marquee (Header & Hero Section)**
The top of the page features a glowing, vintage theater marquee. The main title reads **"GEVA KIPPER: THE AUTEUR OF CODE"** in a bold, condensed sans-serif font. Beneath it, styled like a movie tagline, is the sentence: *"Currently scripting the future as a Software Engineer at Google."*
Just below the marquee, an illuminated typography ticker scrolls continuously with the subtitle: *"Director's Note: This website architecture is computationally re-designed each day by AI. You are experiencing an ever-changing, ephemeral premiere."* 

**The Filmography & Reviews (The 8 Grid Sections)**
The main body is a masonry grid of "Movie Posters" representing your projects. Each card includes a "Critic's Score" badge that specifically grades your coding skills for that project (e.g., "98% Clean Code", "5/5 Algorithmic Complexity"). Hovering over a poster dims the image and brings up the "Review" (the project description) and action buttons (links).

**The End Credits (Footer)**
The bottom of the page transitions into stark black, resembling the very end of a movie credits crawl. Centered on the screen is the text: *"Set Design, Cinematography, and Web Generation by AI Model."*
Beneath this sits exactly one dynamic UI element: a glowing red, monospaced digital timecode (styled like an SMPTE cinematic filming clapperboard timer) updating every second via JavaScript. It displays the exact hours, minutes, and seconds passed since **2026-07-03 02:14:40**, serving as a mesmerizing, ticking temporal anchor for the ever-changing daily layout.
