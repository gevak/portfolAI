**Global Aesthetic & Architecture**
The website should be designed as an elegant, high-resolution "Retro-Neumorphic" operating system. Imagine the marriage of early Macintosh or Windows 95 UI with modern minimalist design principles. The background should be a crisp, off-white (or light beige) canvas featuring a very faint, 1px checkerboard or grid pattern to simulate retro screen texture. Instead of blurry, modern drop shadows, use stark, solid black 4px offset shadows for every UI element. Use a modern, readable pixel-art font for headers and window titles, paired with a crisp monospace font (like IBM Plex Mono) for all body text. The site is a single-page application where all interactions happen within this viewport.

**The Header (Top Menu Bar)**
Fix a classic OS menu bar across the top of the browser window. The bar should be inverted (solid black background with white text). 
On the far left, display the developer's name: **Geva Kipper** in a bold, stylized pixel font. 
In the center, include a single sentence: "Software Engineer building the future of [insert specific Google work here] at Google."
Directly beneath this top bar, incorporate a secondary "ticker" or system-alert banner with the subtitle: "System Note: This website interface is hallucinated and re-designed each day by AI, making it an ever-changing digital environment." 

**The Desktop Canvas & Draggable Folders**
The main viewing area serves as an interactive desktop canvas. Scattered naturally across this desktop are eight pixelated icons, representing files and folders. Hovering over an icon gives it a classic dotted-line focus border. Double-clicking an icon seamlessly opens a "Window". 
These windows must be fully draggable by their title bars around the viewport. Multiple windows can be open at once, seamlessly stacking on top of one another when clicked (updating the z-index). Each window features a unified top bar with a title and a clickable "X" to close it, sharp black borders, and a light gray background.
