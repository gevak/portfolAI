The site is designed as a deep-charcoal, dark-mode Integrated Development Environment (IDE), heavily inspired by VS Code, utilizing sleek, monospaced typography (such as Fira Code with ligatures) and vibrant syntax highlighting. The interface fills the entire browser window without native scrollbars, operating entirely as a single dynamic web application. 

### Global Layout & Header
The interface is divided into three main zones: a narrow left-hand "File Explorer" sidebar, a central "Editor Pane" featuring a top row of horizontal file tabs, and a slender "Status Bar" fixed to the bottom of the screen. 

When the page loads, an overlay "Welcome Window" (styled like an IDE launch screen) introduces **Geva Kipper**. Below an avatar or stylized logo, a glowing, syntax-highlighted comment reads: `// Software Engineer currently building and optimizing at Google.` Directly beneath it, an animated, blinking terminal prompt displays the subtitle in a warning-yellow font: `> SYSTEM NOTE: This exact website layout is conceptualized and re-designed each day by AI and is thus ever-changing.` Closing this welcome window reveals the main workspace, where the 8 portfolio items exist as selectable file tabs across the top of the editor.

### The Editor Tabs (The 8 Portfolio Sections)
Selecting a tab instantly swaps the central pane's content. Each section is designed as a "Split View": the left side displays aesthetic, pseudo-functional code relevant to the project, while the right side acts as an embedded "Live Preview" box containing the rich media, interactive UI, and hyperlinks.

**Tab 1: `glglstats.py`**
*   **Aesthetic:** The left side features Python code showcasing an LZ77 compression algorithm. 
*   **Preview Box:** A slick, neon-accented data visualization interface showing pop-chart repetition patterns. It features two prominent, stylized buttons: `[Run: Launch GlglStats](https://glglstats.geekip.dev)` and an embedded IDE media player window containing the PyData TLV project presentation (`https://www.youtube.com/watch?v=jrKAtOsj1Lo`).

**Tab 2: `ahuzat_dibuk.ipynb`**
*   **Aesthetic:** Styled identically to a Jupyter Notebook with executable code cells for data scraping.
*   **Preview Box:** An interactive, glowing heatmap of Tel-Aviv parking lot occupancy. Below the map, an "Execution Output" log contains stylized hyperlink strings pointing to the press coverage on Channel 13 (`https://13tv.co.il/item/news/domestic/internal/parking-902956407/`), Mako (`https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm`), and the original viral Secret Tel Aviv Facebook post (`https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/`).

**Tab 3: `part_orienting.cpp`**
*   **Aesthetic:** Dense, complex C++ architecture utilizing a computational geometry library, presented in standard blue-and-green syntax highlighting.
*   **Preview Box:** A dark, grid-lined canvas displaying an animated geometric polygon being manipulated by an algorithmic "push plan." A large, floating documentation comment links directly to the detailed algorithm and code: `https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/`.

**Tab 4: `turing_jest.js`**
*   **Aesthetic:** Node.js backend logic designed to handle rapid, concurrent API requests.
*   **Preview Box:** A split-screen chat UI mirroring the LMSYS arena, where two anonymous automated LLMs exchange terrible puns. A vibrant, pulsing "Enter Arena" button points to `https://jest.geekip.dev`.

**Tab 5: `manifest.json` (Rhymes of Israel)**
*   **Aesthetic:** A beautifully formatted JSON configuration file standard for Chrome extensions, highlighting permissions and background scripts.
*   **Preview Box:** A faux high-school yearbook page where the photos are replaced with daily Israeli news headlines, translated into comedic rhymes by a typewriter animation. A blue, Chrome-webstore-styled `[Add to Browser]` button links out to `https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb`.

**Tab 6: `hatul_basak.ts`**
*   **Aesthetic:** TypeScript code handling Wikipedia API calls and trivia game loop logic.
*   **Preview Box:** A Hebrew-language UI mimicking a Wikipedia page that is slowly glitching or transforming into a trivia puzzle. The "Daily Challenge" generator sits above an interactive button linking to `https://hatulbasak.geekip.dev` for users to play and create custom puzzles.

**Tab 7: `meta_portfolio.css`**
*   **Aesthetic:** An intricate CSS file with variables controlling the exact colors, glows, and animations of the surrounding website UI. 
*   **Preview Box:** An infinite mirror-loop animation of the user's cursor clicking a link, accompanied by a comment reading `/* Recursion engaged */` and a self-referential link pointing back to `https://geekip.dev`.

**Tab 8: `.env` (Socials & Contact)**
*   **Aesthetic:** Designed to look like a hidden environment variables file, using a stark, monochromatic high-contrast theme. 
*   **Preview Box:** Three glowing API-key reveals that serve as interactive links to external profiles: 
    *   `GITHUB_TOKEN = github.com/gevak`
    *   `LINKEDIN_URI = linkedin.com/in/gevakip`
    *   `FACEBOOK_ID = facebook.com/geva.kipper`

### The Status Bar (Footer)
Fixed to the bottom of the viewport is an IDE-style status bar in a vibrant accent color (e.g., deep purple or electric blue). 
*   **Left Corner:** An icon resembling git branch architecture alongside the text: `</> Automatically Generated by: [Insert AI Model Name]`
*   **Right Corner:** A live JavaScript stopwatch ticking upward in real-time, displaying: `Uptime (since 2026-07-08 02:02:12): [HH:MM:SS]`. The time increments seamlessly, matching the exact seconds passed since the specified date.
