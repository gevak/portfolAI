**Overall Layout & Container Concept**
The single-page structure is centered around a photorealistic, bezel-less interactive smartphone mockup floating in the middle of the browser viewport. The background is a dark-mode, slowly morphing abstract glassmorphism gradient. On mobile devices, the virtual phone cleanly snaps to fill the viewport, making the user's actual phone become the portfolio. Standard scrolling is disabled; all interactions happen through taps and swipes within the phone interface using fluid, physics-based UI scaling animations (similar to iOS app launches). 

**The Header / iOS-Style Smart Widget**
At the top of the virtual phone screen, replacing a traditional website header, is a frosted-glass smart widget. 
*   **Title Element:** Prominently displays "Geva Kipper".
*   **The Bios:** Right below the name, a clean sans-serif sentence reads: "Software Engineer at Google, building scalable and impactful technology solutions."
*   **AI Tagline:** Below the bio, styled as a subtle "system notification" with a pulsing dot, is the subtitle: "System dynamically re-designed each day by AI; this interface is ever-changing." 

**The App Grid (Projects 1-8)**
The lower half of the phone screen features an app grid. Each project is represented as a sleek, neumorphic App Icon. When an icon is tapped, the interface triggers a smooth "expand" animation, growing the app to take up the full screen of the virtual phone. At the bottom of every opened app is a standard "Home Bar"—swiping or clicking it shrinks the app back into its icon on the grid.

**App 1: GlglStats**
*   **App Icon:** A neon-colored music note intersected by a compression zipper.
*   **Inside the App:** The interface mimics a sleek music player or audio-engineering tool. It features an animated waveform graphic to represent LZ77 compression and song repetition patterns. Below this are two interactive elements: A glowing "View Analytics Site" button mapped to *https://glglstats.geekip.dev*, and an embedded, playable video thumbnail of the PyData TLV presentation linking to *https://www.youtube.com/watch?v=jrKAtOsj1Lo*.

**App 2: Ahuzat Dibuk**
*   **App Icon:** A recognizable parking 'P' symbol overlaid on a glowing thermal heat-map.
*   **Inside the App:** Styled like a dark-mode GPS or navigation map. A stylized, static heat-map of Tel-Aviv rests in the background. The content is presented as three "Push Notifications" the user can click on to read the press coverage. These link respectively to the Channel 13 segment (*https://13tv.co.il/item/news/domestic/internal/parking-902956407/*) and Mako review (*https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm*). A lower alert banner links to the original Secret Tel Aviv Facebook release (*https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/*).

**App 3: 2D Part Orienting**
*   **App Icon:** A minimalist geometric polygon surrounded by directional push arrows.
*   **Inside the App:** The design mimics blueprint or CAD software with a grid background. A dynamic graphic illustrates an algorithm building a "push plan" for a polygon. It features a technical, terminal-style "Access Library & Code" button that directs to the computational geometry library at *https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/*.

**App 4: The Turing Jest**
*   **App Icon:** Theater comedy/tragedy masks styled like robotic AI faces.
*   **Inside the App:** Translates the LMSYS arena vibe into an arcade fighting game or chat-battle interface. Two placeholder LLM avatars face off with speech bubbles containing puns. A primary "Enter the Arena to Rate" button acts as the main CTA, sending users to *https://jest.geekip.dev*.

**App 5: Rhymes of Israel**
*   **App Icon:** A vintage newspaper folded alongside a highschool graduation cap. 
*   **Inside the App:** Combines a news-feed layout with yearbook-style scribbles and comedic, rhyming headline examples floating via CSS animations. Features a prominent button styled exactly like the native Chrome Web Store "Add to Chrome" button, linking directly to install the extension: *https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb*.

**App 6: Hatul Basak**
*   **App Icon:** A Wikipedia "W" globe puzzle piece shaped subtly like a cat's silhouette or a catfish.
*   **Inside the App:** Thematically styled as a minimalist, dark-mode trivia game interface. It holds two main cards: "Play Daily AI Challenge" and "Create & Play Custom User Puzzles", both acting as interactive portals to *https://hatulbasak.geekip.dev*.

**App 7: Meta Portfolio**
*   **App Icon:** An "infinity mirror" icon or a recursive picture-in-picture graphic.
*   **Inside the App:** A visually satisfying "breaking the fourth wall" screen showing cascading code lines of the current app’s own UI. Includes a fun, self-referential button stating "Restart the Matrix" that links directly to *https://geekip.dev*.

**App 8: The Socials Hub (Settings)**
*   **App Icon:** A gear icon or a native "Contacts" book graphic.
*   **Inside the App:** Mimics a phone’s settings/contact screen with three toggles or contact cards pointing to the developer's networks. Tapping them directs outwards to GitHub (*github.com/gevak*), LinkedIn (*linkedin.com/in/gevakip*), and Facebook (*facebook.com/geva.kipper*).

**The Desk Surface (Footer Layer)**
Outside the smartphone container, acting as the desk surface the phone is resting on (or pinned to the absolute bottom of the screen viewport), is an ambient data footer. 
*   On the left, a sleek minimalist string of text: "System generated and visualized by AI."
*   On the right, a live, ticking monospace digital stopwatch. It calculates and prominently displays the exact Time Elapsed (showing Hours, Minutes, Seconds) dynamically ticking upward since the target timestamp: **2026-07-02 02:38:32**. It should look like a futuristic "Uptime" tracker.
