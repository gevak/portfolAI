The overall aesthetic of the single-page application is a "Neo-IDE"—a sleek, modern, browser-based coding environment combining a dark-mode frosted glass UI (glassmorphism) with vibrant syntax-highlighting accent colors (cyan, neon green, and magenta). The typography mixes a clean monospace font (like JetBrains Mono) for code elements and a modern sans-serif (like Inter) for readable interface text. The layout is initially split into three distinct IDE panes: a minimal file explorer on the left, a large code editor in the center, and an inactive terminal at the bottom. 

Upon first loading, the website is in its "uncompiled" state. The central code editor displays a mock script where the standard initial variables define the developer. At the top, stylized as code commentary or a docstring, is the title: `const developer = "Geva Kipper";` followed by the string, `"Currently building scalable solutions and engineering at Google."` Right below it sits the subtitle, styled as a glowing warning comment: `// WARNING: This website is re-designed each day by AI and is thus ever-changing.` The only interactive element in this initial state is a prominent, pulsing neon-green "Compile & Run" button situated in the top right corner of the editor pane. 

When the user clicks "Compile & Run," a highly interactive JavaScript animation triggers. The bottom terminal pane slides up, rapidly printing mock build logs (e.g., "Resolving dependencies...", "Building projects...", "Linking GlglStats..."). Once the build successfully completes with an exit code 0, a smooth CSS transition transforms the raw text editor into a beautiful, masonry-style CSS Grid interface representing the "Compiled GUI." The IDE panes fade into the background, revealing the actual portfolio content laid out across eight distinct interactive modules (cards).

The grid consists of eight stylized project boxes, each designed to look like a polished software widget or interactive execution window:
Box 1 (GlglStats) features a subtle LZ77 compression and waveform background graphic. It presents the project description with two neon action buttons: one linking to the live site (https://glglstats.geekip.dev) and a play-icon button linking to the PyData TLV presentation (https://www.youtube.com/watch?v=jrKAtOsj1Lo).
Box 2 (Ahuzat Dibuk) utilizes a dark, red-to-yellow heatmap gradient as its card background. Alongside the text detailing the Tel-Aviv parking scraper, it features a mini-dock of clipping icons linking to the news coverage (Channel 13: https://13tv.co.il/item/news/domestic/internal/parking-902956407/, Mako: https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm) and the Secret Tel-Aviv Facebook post (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).
Box 3 (2D Part Orienting) takes on a C++ header file aesthetic, displaying wireframe polygons being "pushed" into alignment on hover. It contains a single clean button linking to the computational geometry project and code (https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/).
Box 4 (The Turing Jest) is styled like a split-screen chat arena to reflect its LMSYS combat nature, complete with overlapping speech bubbles. An energetic button directs users to the judging arena (https://jest.geekip.dev).

The remaining four boxes complete the bottom of the grid layout:
Box 5 (Rhymes of Israel) resembles a Google Chrome pop-up window, featuring a yearbook-style doodle background. It includes an official-looking Chrome Web Store install button (https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb). 
Box 6 (Hatul Basak) merges Wikipedia's classic serif aesthetic with a playful trivia UI, displaying a "Daily Challenge" teaser. It contains a prominent call-to-action button to play and create puzzles (https://hatulbasak.geekip.dev). 
Box 7 (The Meta Portfolio) is designed as a visual infinite mirror or an "inception" window, playfully acknowledging the exact site the user is on, complete with a self-referential link (https://geekip.dev). 
Box 8 (Social Connectivity) acts as the configuration/environment variables card. It strips away the graphics for an ultra-minimal list of clickable command-line prompts linking to GitHub (github.com/gevak), LinkedIn (linkedin.com/in/gevakip), and Facebook (facebook.com/geva.kipper).

The bottom of the page features a sticky "Status Bar" mimicking the bottom border of VS Code. On the left side of this dark ribbon, in a tiny monospace font, is the acknowledgment: `> System compiled and generated by AI`. On the right side, aligning with the edge of the screen, is an active JavaScript-powered stopwatch continuously ticking up in real-time (HH:MM:SS format), tracking the exact elapsed time since `2026-04-29 02:17:05`.
