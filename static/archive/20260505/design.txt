**Overall Concept & Visual Identity**
The website adopts a sleek, modern "glassmorphism meets generative art" aesthetic, structured around a dramatic two-state experience: a locked landing state and an activated portfolio state. The UI relies on deep space-gray backgrounds (#121212) contrasted with vibrant, fluid, neon gradients (cyans, magentas, and amber) that imply the "constantly shifting AI-generated" theme. The layout utilizes a responsive "bento box" asymmetric grid, keeping all eight required elements visible on a single continuous page without deep scrolling. 

**State 1: The Virtual High-Five Lock Screen**
When the user visits the site, the screen is entirely dark and locked. In the absolute center floats a glowing, neon-outlined handprint. Above it, elegant typography displays the primary identity: "Geva Kipper." Below the name, a clean sans-serif line reads: "Currently writing code and building solutions as a Software Engineer at Google." Beneath that, glowing with a subtle breathing animation, is the subtitle: "This website is re-designed each day by AI and is thus ever-changing." 

To unlock the site, the user must perform a "virtual high-five." Using JavaScript cursor tracking, if the user rapidly swipes their mouse cursor across the handprint (or taps quickly on mobile), the screen registers a "slap." A kinetic, rippling shockwave effect (using a CSS/JS distortion filter or radial gradient burst) explodes from the handprint. The dark overlay shatters into geometric glass shards that fade away, seamlessly revealing the illuminated portfolio grid beneath. 

**State 2: The Unlocked Bento-Box Portfolio Grid**
The revealed grid consists of eight frosted-glass cards with smooth rounded corners, floating over a slowly shifting ambient gradient background. Hovering over any card increases its opacity and reveals a unique, glowing border reflecting its specific project. 

*Card 1 (GlglStats):* Designed with a moving audio-wave visualizer pattern in the background. The text details the Python LZ77 compression pop-chart project. It features two prominent, pill-shaped buttons: "View Live Site" (linking to glglstats.geekip.dev) and an inline glowing "Watch PyData Presentation" button (linking to the YouTube video) accompanied by a small play icon.

*Card 2 (Ahuzat Dibuk):* The background features a subtle, red-to-green thermal heatmap grid. The text explains the Python scraping and parking occupancy visualization project. A cluster of tags acts as links for the press coverage, labeled "Channel 13 News", "Mako / Nexter", and "Secret Tel-Aviv (Facebook)", each opening their respective URLs. 

*Card 3 (2D Part Orienting):* The visual motif is a wireframe polygon continuously re-orienting itself. The typography outlines the C++ computational geometry and push-plan algorithm. A single sleek button reads "View Code & Documentation" pointing to the TAU CGL project page.

*Card 4 (The Turing Jest):* Themed with opposing robotic masks (comedy/tragedy style). It describes the LMSYS-like arena for LLM pun competitions. A bold link button calls users to "Enter the Arena" at jest.geekip.dev.

*Card 5 (Rhymes of Israel):* Designed to look like a digital newspaper slipping out of a high school yearbook. The text introduces the LLM Chrome extension for comedic news headlines. A distinct "Add to Chrome" button points to the Google Chrome Web Store link.

*Card 6 (Hatul Basak):* Features a playful, pixelated cat silhouette masking Wikipedia puzzle pieces. It explains the Hebrew catfishing.net port and its AI-generated daily challenges. A primary button prompts users to "Play & Create Puzzles" linking to hatulbasak.geekip.dev.

*Card 7 (Meta Portfolio):* Has an "infinity mirror" visual effect, rendering a tiny wireframe of the exact website the user is currently looking at. The text cheekily notes the self-referential nature of the project. A recursive loop button links back to geekip.dev.

*Card 8 (Social Links):* This is a sleek, horizontal card anchored at the bottom right of the grid. It features three large, glowing, minimalist icons. They expand on hover to display text: GitHub (gevak), LinkedIn (gevakip), and Facebook (geva.kipper).

**Footer & Meta Elements**
The bottom of the single page anchors a minimalist floating dock. On the left side of the dock, monospaced terminal text pulses gently, giving credit: "Design generated continuously by AI." On the right side, a live JavaScript-driven digital stopwatch ticks upward continuously, displaying hours, minutes, and seconds elapsed since exactly "2026-05-05 02:09:51", adding a mysterious, chronological weight to the AI-redesign theme.
