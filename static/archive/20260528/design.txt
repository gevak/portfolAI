The overall aesthetic of the single-page website is "Modern Cyber-Alchemy," utilizing a deep, dark-mode color palette (matte obsidian and midnight blues) accented by glowing, neon-pastel highlights and frosted glass (glassmorphism) elements. The layout is split into two visual halves: a fixed, interactive "Synthesis Lab" at the top, and a dynamic "Project Gallery" grid that populates at the bottom as the user interacts with the site. The primary typography pairs a sleek, geometric sans-serif (like Space Grotesk) for headings with a highly legible sans-serif (like Inter) for body text to maintain an elegant, futuristic feel.

In the top-left corner, floating discretely, is the header. It displays the name "Geva Kipper" in a bold, glowing gradient font, followed by a single crisp sentence: "Software Engineer currently building at Google." Below this, a slightly muted, italicized subtitle dynamically types out: "Note: This website is re-designed every day by AI, making it an ever-changing, ephemeral experience." 

The centerpiece of the top half is the "Virtual Pot"—envisioned not as a medieval cauldron, but as a sleek, glowing holographic crucible or a frosted-glass induction basin. Hovering in an arch above this basin are eight distinct "Ingredients." Each ingredient is a 3D-styled, interactive glass orb containing a floating, glowing icon representing a specific project. When a user hovers over an orb, a minimal tooltip appears hinting at its contents. The core interaction requires the user to click, drag, and drop these orbs into the glowing basin. Upon dropping, the site uses physics-based JavaScript to snap the orb into the liquid, triggering a satisfying CSS splash animation, releasing colored particles, and smoothly scrolling the page down slightly as a new Project Box materializes in the grid below. 

The eight project boxes are designed as sleek, translucent glass cards that appear with a subtle "crystallizing" fade-in effect once their corresponding ingredient is cooked. They form an elegant masonry grid. 

Box 1 (Triggered by a "Vinyl Record" orb): Presents 'GlglStats'. The card features neon pink and cyan accents. The text explains it as a Python project measuring the repetitiveness of Israeli pop-chart songs using LZ77 compression and visualizing the patterns. Below the description are two elegant buttons: one linking to the live site at https://glglstats.geekip.dev, and the other opening a styled, embedded thumbnail linking to the PyData TLV conference presentation at https://www.youtube.com/watch?v=jrKAtOsj1Lo.

Box 2 (Triggered by a "Map Pin" orb): Presents 'Ahuzat Dibuk'. The card glows with heat-map colors (amber and red). It details the Python scraping and data visualization project for Tel-Aviv parking lot occupancy. At the bottom, a row of sleek media badges links to news coverage on Channel 13 (https://13tv.co.il/item/news/domestic/internal/parking-902956407/), Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), and the original viral Facebook publication (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).

Box 3 (Triggered by a "Geometric Polygon" orb): Presents '2D Part Orienting'. Styled with computational, wireframe aesthetics and steel-blue accents. The text highlights it as a C++ implementation in an open-source computational geometry library, detailing the efficient algorithm for constructing push plans to orient polygons. A clean, hyperlinked button directs users to the code and additional info at https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/.

Box 4 (Triggered by a "Jester Hat / Brain" orb): Presents 'The Turing Jest'. Accentuated in electric purple and gold, this card introduces the LMSYS-like arena where LLMs compete at making and explaining puns, rated by users. A glowing CTA button prominently links to the battle arena at https://jest.geekip.dev.

Box 5 (Triggered by a "Feather Pen" orb): Presents 'Rhymes of Israel'. Using chrome-silver and blue highlights, the card explains this Chrome extension which utilizes an LLM to rewrite Israeli news headlines into comedic, high-school-yearbook-style rhymes. It features a recognizable "Available in Chrome Web Store" badge linking to https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb.

Box 6 (Triggered by a "Cat silhouette" orb): Presents 'Hatul Basak'. Theming here is playful orange and crisp white. It describes the project as a Hebrew port of catfishing.net, a Wikipedia-based Trivia game featuring AI-generated daily challenges and user-created themed puzzles. The card includes a bold button to play and create puzzles at https://hatulbasak.geekip.dev.

Box 7 (Triggered by a "Mirror" orb): Presents the Portfolio project itself. This card is visually meta, featuring an iridescent, slightly recursive border animation. The text jokingly points out the meta nature of the project (building the site they are currently viewing), with a self-referential link pointing back to https://geekip.dev.

Box 8 (Triggered by a "Network Node" orb): Acts as the digital business card. It is an ultra-minimalist, sleek box containing stylized hover-effects and high-contrast icons linking securely to GitHub (github.com/gevak), LinkedIn (linkedin.com/in/gevakip), and Facebook (facebook.com/geva.kipper).

At the very bottom of the page, anchored directly below the expanding project grid, is a sleek footer that spans the width of the page. The footer contains monospace, terminal-style text that provides a stark but elegant contrast to the rest of the site. It reads: "Design synthesized by AI." Next to this text is a live, ticking JavaScript stopwatch with monospaced digits rolling over smoothly, meticulously calculating and displaying the exact time (formatted dynamically as Hours:Minutes:Seconds) that has elapsed since the temporal anchor point of 2026-05-28 02:33:48.
