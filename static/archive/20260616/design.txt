**Overall Concept and Layout Structure**  
The website utilizes a modern, dark-mode, split-screen desktop layout (stacking responsively on mobile) designed to look like a high-end, futuristic "AI Terminal & Evidence Board." The left pane (taking up about 35% of the viewport) houses the dynamic AI Chatbot Interview Interface. The right pane (65% of the viewport) serves as a glassmorphic "Dossier Canvas," displaying an asymmetrical masonry grid of eight interactive project cards that the AI pulls up as visual aides during the interview. The aesthetic is deep obsidian black (`#0f0f13`) with subtle, glowing gradient accents in cyan, neon purple, and soft amber, blending sleek modernism with cyberpunk undertones. Typography relies on a crisp sans-serif (like Inter or system-ui) for reading comfort, contrasted with a clean monospaced font (like JetBrains Mono) for AI responses and metadata.

**Header & Identity Area (Integrated into the Chat Pane)**  
The top of the left chat pane features a sticky header resembling an AI system boot sequence. It prominently displays the Developer's Name: **Geva Kipper**, followed by a sleek, glowing status badge reading: "Currently engineering impactful, large-scale solutions at Google." Immediately below this, a smaller, italicized, low-opacity monospace font displays the subtitle: *"System Note: This dynamic interface is uniquely conceptually re-designed and re-generated each day by AI, ensuring an ever-evolving digital presence."*

**The Chatbot Interview Interface (Left Pane)**  
This pane functions as the primary user navigation. It opens with an animated typing effect from the AI: *"Subject: Geva Kipper loaded. I am ready for your interview. What would you like to know about his work?"* Users can type into a clean, floating input field at the bottom, or click auto-generated suggestion chips (e.g., "Data Vis," "LLMs," "Algorithms," "Comedy"). When a category is queried, the chat generates a brief conversational response, and smoothly auto-scrolls the right pane to highlight or expand the relevant project boxes, pushing the non-relevant ones into a dimmed background state. 

**The Dossier Canvas: Project Boxes (Right Pane)**  
The right side features eight floating, glassmorphic cards resting on the dark background. Each card on hover lightly tilts (3D transform) and intensifies its glowing border. 

*Box 1 (GlglStats):* Designed with a visual motif of an audio waveform morphing into a binary data stream. It details the Python LZ77 compression project analyzing Israeli pop-chart repetitiveness. Contains two distinct chiclet buttons: an external link icon pointing to `https://glglstats.geekip.dev`, and a neon-red YouTube play icon linking to the PyData TLV presentation (`https://www.youtube.com/watch?v=jrKAtOsj1Lo`).

*Box 2 (Ahuzat Dibuk):* Features a background texture resembling a glowing, abstracted city map or heat-map grid. It describes the Python parking lot scraping and data visualization project for Tel-Aviv. Below the text, three staggered, pill-shaped tags link to the media coverage: "Channel 13 News", "Mako Nexter", and "Secret Tel-Aviv (Facebook)".

*Box 3 (2D Part Orienting):* Uses a minimalist aesthetic with animated wireframe polygons rotating in the top corner. It explains the C++ implementation for the computational geometry 'push plan' algorithm. A clean, terminal-style text link directs the user to the TAU project page at `https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/`.

*Box 4 (The Turing Jest):* Themed around modern AI, featuring a stylized icon of two robotic theater masks. It introduces the LMSYS-like arena where LLMs compete at making/explaining puns based on user ratings. A glowing hyperlink button links directly to `https://jest.geekip.dev`.

*Box 5 (Rhymes of Israel):* Incorporates a playful, retro-meets-modern design, resembling a torn digital newspaper clipping transitioning into a high-school yearbook photo frame. It outlines the Chrome extension that uses LLMs to rewrite news headlines into comedic rhymes. A stylized "Add to Chrome" button directs to the Chrome Web Store extension page.

*Box 6 (Hatul Basak):* Contains a subtle, glowing silhouette of a cat overlapping a pixelated trivia question mark. It describes the Hebrew port of catfishing.net, using AI to generate daily Wikipedia-based trivia challenges and user-themed puzzles. A bold "Play / Create" link directs to `https://hatulbasak.geekip.dev`.

*Box 7 (Self-Referential Portfolio):* Designed like an infinity mirror or recursive wireframe screen (the "Inception" box). It wittily references itself as the meta-project the user is currently viewing. A self-referencing loop icon links back to `https://geekip.dev`.

*Box 8 (Social Matrix):* A smaller, sleek "connect" module stripped down to pure iconography. It aligns three highly polished, monochromatic logos that glow on hover: GitHub (`github.com/gevak`), LinkedIn (`linkedin.com/in/gevakip`), and Facebook (`facebook.com/geva.kipper`). 

**Footer & Telemetry Area**  
Spanning the bottom of the entire viewport is a thin, ticker-tape style footer, designed to look like raw system telemetry. On the left side, formatted as a system log, it prominently credits the generative origin: *"Interface Designed by Advanced Large Language Model."* On the right side, floating in a vivid, high-contrast monospace font (e.g., neon amber), a live Javascript-powered stopwatch continuously ticks up. It is styled as `SYS_UPTIME: [HH:MM:SS]`, calculating locally in real-time the exact duration passed since the precise anchor date: `2026-06-16 03:26:23`.
