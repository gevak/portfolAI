**The 414 Puzzle Initial State (The Gate)**
The user lands on a stark, minimalist black screen dominated by a glowing, oversized error message in monospaced red typography: `HTTP 414: URI TOO LONG`. Below this, an absurdly massive, screen-filling block of glowing URL parameters drips down the page like the Matrix code, filled with junk parameters (e.g., `&utm_nonsense=`, `&bloat_factor=99`). This is an interactive puzzle. The user must use their cursor to physically "slice" or backspace the bloated parameters away, hunting for an obviously out-of-place string labeled `&unlock_portfolio=true`. Clicking or isolating this parameter collapses the entire giant text block into a single, bright neon point in the center of the screen, which then ripples outward (like a pebble in water) to reveal the main portfolio underneath.

**Overall Aesthetic & Header**
The revealed single-page portfolio uses a "Cyber-Elegant Bento Box" design. The background is a very deep, dark navy/charcoal with subtle, slowly shifting fluid gradients (purple and blue) in the deep background to represent the "ever-changing AI" nature of the site. The layout relies on glassmorphism—frosted, semi-transparent UI cards with thin, glowing borders. 
At the top, fixed to the screen, is the header. The developer's name, **Geva Kipper**, is displayed in a bold, modern sans-serif font. Directly below it, a sleek typing-animation reveals: *"Software Engineer currently building [insert specific role/team] at Google."* Beneath that, in an italicized, slightly dim monospaced font, is the subtitle: *"Notice: This website's design is regenerated daily by AI. Current state: Ephemeral."*

The portfolio items are arranged in an asymmetrical, interlocking 8-piece bento grid that fills the viewport dynamically. On hover, each box slightly elevates and its thin glass border flares with a neon accent color.

*   **Box 1 (GlglStats):** Designed with an abstract, shifting audio waveform background graphic. It features the project title and a brief explanation of the Python/LZ77 pop-chart compression algorithm. The box is split: one side holds a sleek, embedded thumbnail of the PyData TLV YouTube video (`https://www.youtube.com/watch?v=jrKAtOsj1Lo`), and the other features a primary call-to-action button linking to `https://glglstats.geekip.dev`.

*   **Box 3 (2D Part Orienting):** Features a sleek, wireframe animation of a rotating polygon being "pushed" by invisible lines. The text outlines the C++ computational geometry push-plan algorithm. A minimalist "View Source & Info" button links to the TAU CGL project page (`https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/`).
*   **Box 4 (The Turing Jest):** Styled like a high-tech colosseum or an arcade fighting screen, featuring split red/blue neon accents. The text introduces the LMSYS-like arena for LLM pun battles. An interactive-looking "Enter Arena" button links to `https://jest.geekip.dev`.
