**Global Aesthetic & Layout Strategy**
The design utilizes a "Digital Impact Constellation" aesthetic, built as a single-page parallax scrolling experience. The background is a deep void (dark mode, `#0B0C10`) overlaid with a slowly shifting fluid WebGL mesh gradient that subtly changes colors, reflecting the "ever-changing, AI-generated" nature of the site. A faint, wireframe 3D globe rotates in the fixed background. As the user scrolls through the timeline, the globe spins to center on the geographical or conceptual location of each project, elevating these highly specific creations into the realm of "global social impact." All cards use glassmorphism (translucent dark backgrounds, heavy background blur, and glowing 1px borders) to appear as floating data nodes.

**Hero & Header Section**
The top of the page features a striking, clean presentation using modern typography (e.g., Space Grotesk). Center stage is the developer's name, "Geva Kipper," in a large, bold, stark-white font. Immediately beneath is a sleek, semi-transparent badge containing the single sentence explaining his current work at Google. Below this, the subtitle acts as a focal point: "This website is re-designed each day by AI and is thus ever-changing." This subtitle should have a subtle CSS "glitch" or typewriter cursor animation to emphasize its dynamic, machine-driven nature. From the bottom of this text, a luminous fiber-optic timeline begins, snaking down the center of the page.

**The Timeline Core Mechanic**
A vertically snaking SVG path forms the central timeline. As the user scrolls, a bright neon "data pulse" travels down this line, turning the inactive grey path into a glowing gradient (cyan to magenta). The 8 sections requested are treated as distinct nodes attached to this timeline, alternating left and right in position. When the data pulse reaches a node, the corresponding project card elegantly scales up and fades in using scroll-triggered animations.

**1. GlglStats Node**
This card visualizes "repetitiveness" as its design motif. The background of the card features subtle, animated audio waveforms derived from LZ77 compression patterns. The text introduces the Python pop-chart analysis project. Below the description are two prominent icon buttons: one stylized as a glowing link for the project site (https://glglstats.geekip.dev), and another stylized as a YouTube play button for the PyData TLV conference video (https://www.youtube.com/watch?v=jrKAtOsj1Lo). 

**2. Ahuzat Dibuk Node**
To represent the parking project, this card features a faint, dynamic heat-map pattern that shifts from green to red based on mouse hover. The text explains the Python scraping and data visualization for Tel-Aviv parking lots. Below, a horizontal flex tray of minimalist pill-shaped badges links to the news stories (Channel 13: https://13tv.co.il/item/news/domestic/internal/parking-902956407/, Mako: https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm) and the Facebook Secret Tel Aviv publication (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).

**3. 2D Part Orienting Node**
This card embraces computational geometry. In the top corner of the card, an algorithmic animation continuously calculates and pushes a flat 2D polygon into a corner. The typography here is slightly more technical and monospaced, detailing the C++ open-source algorithm implementation. A sleek, geometric arrow button points to the CGAL project page and code (https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/).

**4. The Turing Jest Node**
Designed as an LLM arena, this card is split vertically into two "chat bubble" halves (representing competing AI answers). The text describes the pun-based LMSYS-like arena. In the center, bridging the two halves, is a prominent, glowing link to judge the puns (https://jest.geekip.dev).

**5. Rhymes of Israel Node**
This card juxtaposes modern Chrome extension UI with a nostalgic, high-school yearbook torn-paper overlay. It presents the LLM comedic headline rewriter. A distinct, animated "Add to Chrome" styled gradient button links out to the Chrome Web Store installation (https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb).

**6. Hatul Basak Node**
To fit the trivia/catfishing theme, this card employs a cryptic, redacted-text aesthetic where parts of the description briefly blur and un-blur like a Wikipedia article loading. It describes the daily AI-generated Hebrew Wikipedia trivia game and user-created themed puzzles. An interactive puzzle-piece icon serves as the main call-to-action button linking to the game (https://hatulbasak.geekip.dev).

**7. Meta Portfolio Node**
This node visually folds in on itself—a recursive "Droste effect." Inside the glass card is an infinitely shrinking wireframe mockup of the very timeline the user is scrolling. The text wittily highlights the meta nature of the portfolio project. The self-reference link is styled as an infinite loop icon pointing to https://geekip.dev.

**8. Social Hub Node**
The timeline culminates in a glowing, multi-pronged central core at the bottom of the page rather than a side-card. This section features three large, metallic 3D icons that tilt dynamically as the user's cursor moves over them. These represent and link to GitHub (github.com/gevak), LinkedIn (linkedin.com/in/gevakip), and Facebook (facebook.com/geva.kipper).

**Footer Section**
The footer drops the glassmorphism for a raw, terminal-like display (green monospace text on black). It plainly prints two lines. The first line credits "Designed by an AI Assistant" as the architect of today's iteration. The second line features the live JavaScript stopwatch, reading: "System Uptime since 2026-07-18 01:51:56: [HH:MM:SS]", ticking up in real-time to bridge the gap between the static launch date and the user's current session.
