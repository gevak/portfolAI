**Overall Aesthetic and Architecture**
The website is an immersive, single-page experience styled as a sleek, modern glass-front vending machine floating in a minimalist, dimly lit, deep-charcoal 3D space. The aesthetic blends elegant glassmorphism with subtle neon accents. The layout is centered, with the machine taking up the majority of the viewport. Instead of standard scrollable content, user interaction relies on hovering over and "purchasing" snacks, which triggers CSS/JS animations mimicking vending machine coils dispensing items. Upon dropping into the bottom tray, the snack expands into an elegant modal overlay displaying the project details.

**Header & Marquee Display**
At the top of the vending machine sits a glowing, edge-to-edge LED marquee panel. The primary text shines in a crisp, modern sans-serif font: "Geva Kipper." Directly underneath, a sleek, static sub-heading reads: "Currently crafting elegant engineering solutions at Google." Below this, an animated LED ticker scrolls continuously in vibrant amber: "WARNING: Website UI and inventory re-designed each day by AI. Always ever-changing." 

**The Vending Machine Inventory (The Grid)**
The central glass window houses a 4x2 grid of metallic coils, displaying eight distinct "snacks." Each snack uses high-resolution CSS 3D transforms to look like actual premium packaging (bags, boxes, cans), featuring custom label art that visually hints at the project. Hovering over a snack zooms it slightly, casts an inner spotlight on it, and displays the project title via a floating tooltip. Clicking a snack triggers a smooth drop animation into the "PUSH" door at the bottom, automatically unfurling a frosted-glass modal with the content.

**Snack Modals & Content Mapping**
1. **GlglStats (Row 1, Slot 1):** Packaged as a vibrant, crinkled bag of "Compression Chips." The modal details a Python project measuring the repetitiveness of Israeli pop-chart songs using LZ77 compression, complete with text on visualizing repetition patterns. It features two bold, pill-shaped buttons: "View Live Site" (https://glglstats.geekip.dev) and "Watch PyData TLV Presentation" (https://www.youtube.com/watch?v=jrKAtOsj1Lo).
2. **Ahuzat Dibuk (Row 1, Slot 2):** Styled as a heavy, metallic energy drink can named "Heatmap Energy." The modal explains the Python scraping and data visualization project for TLV car parking occupancy. It includes a grid of three media links styled as news clippings leading to channel 13 (https://13tv.co.il/item/news/domestic/internal/parking-902956407/), Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), and the original Facebook publication (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).
3. **2D Part Orienting (Row 1, Slot 3):** Packaged as an elegant, origami-style polygon box. The modal text describes the C++ implementation in an open-source computational geometry library to construct a 'push plan' for orienting polygons. It includes a subtle, terminal-styled link for additional info and code: (https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/).
4. **The Turing Jest (Row 1, Slot 4):** Designed as a premium fortune-cookie wrapper. The modal outlines the LMSYS-like arena where LLMs compete in making and explaining puns, rated by users. A laughing-face icon button directs users to the arena: (https://jest.geekip.dev).
5. **Rhymes of Israel (Row 2, Slot 1):** Packaged as a retro, high-school yearbook-themed juicebox. The modal details the Chrome extension using LLMs to rewrite Israeli news headlines into comedic rhymes. Includes a primary call-to-action button to install: (https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb).
6. **Hatul Basak (Row 2, Slot 2):** Styled as a trendy, minimalist tin of cat treats. The modal describes a Hebrew port of catfishing.net, generating AI daily Wikipedia trivia challenges and user-themed puzzles. An interactive button directs to play and create puzzles: (https://hatulbasak.geekip.dev).
7. **Meta Portfolio (Row 2, Slot 3):** Packaged in a mirrored, transparent blister-pack reflecting the website itself. The modal humorously acknowledges the meta-nature of the site you are currently navigating. It houses a self-referential infinity-loop icon linking back to: (https://geekip.dev).
8. **Social Network Sampler (Row 2, Slot 4):** Packaged as a "combo-pack" of three brightly colored gummy candies. The modal serves as a digital business card, featuring sleek, hover-animated icons linked directly to GitHub (github.com/gevak), LinkedIn (linkedin.com/in/gevakip), and Facebook (facebook.com/geva.kipper).

**The Numpad & Footer Interface**
On the right-hand panel of the vending machine sits an illuminated digital keypad and a small LCD screen. The LCD screen constantly updates a live, monospace stopwatch indicating the exact format of hours, minutes, and seconds elapsed since "2026-07-15 01:48:00". Below the keypad, engraved into the brushed metal baseplate of the machine, is elegant, low-contrast typography reading: "Concept, architecture, and design autonomously generated by [Insert AI Model Name]."
