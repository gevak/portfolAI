**Overall Aesthetic and Architecture**
The website adopts a "Cybernetic Biology" aesthetic, blending the organic structure of a DNA double-helix with the sleek, high-tech visuals of modern software engineering. Set against a deep, void-like digital background (dark navy fading to pitch black), the central focal point is a continuously twisting, 3D-rendered DNA strand that acts as the vertical spine of the single-page application. The two main backbones of the helix are constructed from glowing, flowing streams of code, while the connecting "base pairs" (rungs) serve as the suspension points for the project cards. 

**Scrolling and Interaction Mechanics**
As the user scrolls down the page, the DNA strand dynamically rotates on its Y-axis in a parallax effect. The eight main content sections function as horizontal base pairs connecting the two backbones. When a section scrolls into the viewport's center, the helix rotation brings that specific "genetic node" to the forefront, expanding it into a frosted-glass (glassmorphism) project card. Elements in the background remain slightly blurred with a lowered opacity to create depth (z-axis), ensuring the user’s focus remains on the active project.

**The Hero Section (The Upper Telomere)**
The top of the page introduces the start of the genetic sequence. A prominent, elegant floating header displays the name **Geva Kipper** in a bold, bio-luminescent sans-serif typography. **The Sequence: Eight Genetic Nodes (Project Cards)**
Each of the following eight boxes materializes as an active nucleotide pair, featuring glowing geometric borders and smooth hover states that reveal UI elements.

**Node 1: GlglStats**
Designed with an audio-wave motif woven into the DNA rung. The card details the Python project measuring the repetitiveness of Israeli pop-chart songs using LZ77 compression. The UI features a mini, interactive visualizer showing compression patterns. It includes an embedded, sleek video player linking to the PyData TLV presentation (https://www.youtube.com/watch?v=jrKAtOsj1Lo) and a prominent neon button directing to the live site (https://glglstats.geekip.dev).

**Node 2: Ahuzat Dibuk**
This card features a background texture of a heat-map slowly shifting colors to represent Tel-Aviv parking occupancy. It presents the Python scraping and data visualization project. The external links are styled as floating digital news clippings. Clicking the clippings directs the user to Channel 13 (https://13tv.co.il/item/news/domestic/internal/parking-902956407/) and Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), while a Facebook icon links to the original Secret Tel Aviv post (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).

**Node 3: 2D Part Orienting**
The visual theme here shifts to sharp, computational geometry. Tumbling 2D polygons float inside the card, eventually aligning to represent the C++ 'push plan' algorithm. The text details the project's integration into an open-source library, accompanied by a glowing terminal-style button linking to the additional info and code repository (https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/).

**Node 4: The Turing Jest**
Styled like a digital sparring arena, representing an LMSYS-like competition. Two abstract AI avatars face off on opposite sides of the card. The description outlines the LLM pun-making and explanation battleground, with user ratings at its core. A vibrating "Enter Arena" button serves as the link to https://jest.geekip.dev.

**Node 5: Rhymes of Israel**
This node playfully juxtaposes high-tech LLM capabilities with a nostalgic, faded high-school yearbook design overlaid on standard news typography. It describes the Chrome extension that rewrites Israeli news headlines into comedic rhymes. A stylized Chrome web store badge provides the install link (https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb).

**Node 6: Hatul Basak**
A trivia/puzzle aesthetic featuring branching paths reminiscent of Wikipedia link-hopping, alongside subtle feline iconography. The card outlines the Hebrew port of catfishing.net, emphasizing the AI everyday-generated challenges and user-created themed puzzles. Two interactive portals (buttons) allow the user to "Play the Game" or "Create a Puzzle", both leading to https://hatulbasak.geekip.dev.

**Node 7: The Meta Portfolio**
A visually striking recursive node where a miniature version of the entire website's DNA strand spins inside the card itself. It acknowledges the meta-nature of the portfolio project (the very site being built) and provides a self-referential hyper-glow link pointing back to https://geekip.dev.

**Node 8: The Social Synthesis**
The final rung before the DNA sequence terminates. Instead of a traditional project card, this node acts as a molecular bond connecting stylized 3D icons. The text invites connection, with glowing molecular nodes branching out to GitHub (github.com/gevak), LinkedIn (linkedin.com/in/gevakip), and Facebook (facebook.com/geva.kipper).
