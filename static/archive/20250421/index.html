GlglStats

GlglStats is a comprehensive platform for analyzing Israeli radio airplay data. The site provides powerful visualization and search capabilities for exploring what songs and artists are played on different radio stations.

The platform collects data through automated processes and presents it in an intuitive interface that allows users to discover trends, compare stations, and explore the Israeli music landscape.

I presented this project at an industry conference, demonstrating how data analysis can provide insights into media patterns and audience preferences.

Ahuzat Dibuk

Ahuzat Dibuk was an investigative data journalism project that analyzed Tel Aviv's parking ticket data to identify patterns and potential issues in municipal enforcement.

Using Python and data visualization techniques, I uncovered significant anomalies in how parking tickets were distributed, including statistical outliers that suggested potential targeting of specific neighborhoods.

The findings were published in several news outlets including Channel 13 News and Mako, generating public discussion about parking enforcement practices across the city.

The project's impact went beyond its initial findings, leading to conversations about data transparency and algorithmic fairness in urban services.

2D Part Orienting

This computational geometry research project focused on developing algorithms for orienting two-dimensional parts using minimal controls.

The problem involves determining how to orient an arbitrary polygonal part from any initial orientation to a desired final orientation using only simple mechanical operations. This has important applications in manufacturing automation and robotics.

My contribution included developing simulation software to model how different polygon shapes would respond to various orienting strategies, and identifying optimal sequences of operations.

The project combined elements of computational geometry, physics simulation, and optimization to create practical solutions for real-world manufacturing challenges.
