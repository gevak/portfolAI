The website features a "Glass-OS" aesthetic, simulating a modern, slightly futuristic desktop operating system functioning within a single browser window. The background is a deeply saturated, dark canvas with slow-moving, blurred gradient orbs (in tones of cyan, magenta, and deep purple) that give the screen a living, breathing energy. The entire layout relies on a glassmorphism design language—semi-transparent panels, subtle frosted background blurs, and delicate 1px white-translucent borders—allowing the shifting background to peek through the interface elements. Typography uses a sleek, monospace font for technical details and a clean sans-serif for reading text, mimicking a high-end developer environment.

At the center of the desktop sits the "System Profile" window, permanently pinned. The window header features macOS-style traffic light buttons. Inside, large glowing text displays the developer's name: **Geva Kipper**. Directly below is a clean, minimal sentence: "Software Engineer currently solving complex problems at Google." Underneath this, a system-notification style badge displays the subtitle: "System Note: This website's UI is re-designed each day by AI and is thus ever-changing," accompanied by a subtle pulsing dot to indicate "live" AI status.

The desktop is populated by six interactive "Application Windows" clustered around the profile, arranged in a masonry grid or draggable interface. Each window features a title bar with the project name and an content area distinct to its theme:
**Window 1: GlglStats.** Designed to look like a data analytics dashboard. It features a stylized, looping visual of an LZ77 compression graph. The text briefly explains the Python script measuring the repetitiveness of Israeli pop-chart songs. Two sleek buttons are situated at the bottom: an external link to `https://glglstats.geekip.dev`, and a media-player button linking to the PyData TLV conference presentation at `https://www.youtube.com/watch?v=jrKAtOsj1Lo` (which can open an in-window modal or an external tab).
**Window 2: Ahuzat Dibuk.** Styled like a digital map or radar screen. The text describes the Python scraping and data visualization project for Tel-Aviv parking lot occupancy heat-maps. It includes a "Media Coverage" folder icon that expands to show three linked files: News Report 1 (`https://13tv.co.il/item/news/domestic/internal/parking-902956407/`), News Report 2 (`https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm`), and the Original Facebook Publication (`https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/`). 
**Window 3: 2D Part Orienting.** This window mimics a CAD software environment or a C++ IDE. A background watermark displays wireframe polygons. The text outlines the C++ implementation of the pushing plan algorithm within an open-source computational geometry library. A "View Documentation & Code" button links to `https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/`.

The remaining three application windows continue the OS metaphor:
**Window 4: The Turing Jest.** Designed like a retro arcade or an arena leaderboard. The description introduces the LMSYS-like arena where LLMs compete in making and explaining puns, judged by users. A glowing "Enter Arena" button links to `https://jest.geekip.dev`.
**Window 5: Rhymes of Israel.** This window resembles a browser extension storefront. The text details the LLM-powered Chrome extension that transforms Israeli news headlines into comedic, highschool yearbook-style rhymes. An "Install Extension" button links directly to `https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb`.
**Window 6: system_portfolio.exe.** This is a meta-window styled as an infinite-mirror terminal or a self-referential code snippet reflecting the creation of the site itself. The text acknowledges the meta nature of the portfolio project, featuring a self-referential execution link pointing to `https://geekip.dev`.

At the bottom edge of the screen, acting as the operating system's Taskbar or Dock (Section 7), sits a frosted glass shelf. In the center are large, interactive app icons linking to the developer's socials: GitHub (`github.com/gevak`), LinkedIn (`linkedin.com/in/gevakip`), and Facebook (`facebook.com/geva.kipper`). Hovering over these icons causes them to scale up with a fluid spring animation. 

On the far right of this taskbar—mimicking a system clock and info tray—lies the final requirement. It contains a small AI microchip icon alongside the text: "UI Generated by AI Designer." Next to it is a live, monospaced digital stopwatch actively ticking up (in Hours:Minutes:Seconds format), calculating and displaying the exact time elapsed since the system epoch: `2026-03-31 01:50:32`.
