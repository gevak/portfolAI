**Overall Architecture & Aesthetic**
The website is designed as a full-screen, highly interactive spatial UI that mimics a modern, futuristic glass-front vending machine. Set against a dark, cinematic background (deep charcoal and ambient purple/cyan glows), the machine's body is crafted using sleek glassmorphism—frosted glass panels, skeletal metallic framing, and soft neon backlighting. The design is elegant, avoiding clunky retro tropes in favor of a clean, "cyber-boutique" aesthetic. Interaction is driven by modern CSS animations and JavaScript: when a user clicks an item, a 3D metallic coil spins, the item "drops" into a bottom dispensing tray, and then expands upward into a beautifully styled floating modal containing the project’s text and links. 

**The Header / Marquee Display**
At the very top of the vending machine sits a glowing OLED-style marquee banner. In crisp, illuminated typography, it prominently displays "Geva Kipper" alongside the sentence: "Software Engineer currently weaving scalable code and innovative systems at Google." Directly beneath this, scrolling at a smooth, deliberate pace in a vibrant secondary accent color (like amber or neon blue), is the subtitle: "Notice: This digital interface is dynamically re-designed by AI every 24 hours and is thus ever-changing."

**The Inventory Grid (Slots 1 to 6)**
The central glass display houses the projects, arranged on horizontal shelves with rendered metallic coils. Each software project is represented as a uniquely designed "packaged good" featuring vibrant, high-contrast label art.
*   **Slot 1 (GlglStats):** Designed as a retro-futuristic translucent cassette tape. When dispensed, the modal details the Python/LZ77 algorithm measuring Israeli pop-chart repetitiveness, featuring a dynamic audio-wave graphic. It contains two cleanly styled button links: one to the live site (https://glglstats.geekip.dev) and an embedded thumbnail link to the PyData TLV presentation on YouTube (https://www.youtube.com/watch?v=jrKAtOsj1Lo).
*   **Slot 2 (Ahuzat Dibuk):** Represented as a sleek, glowing matchbox car packaging. Its modal describes the Python scraping and data visualization project for Tel-Aviv parking heat-maps. It includes three distinct "press-clipping" links: Ch. 13 News (https://13tv.co.il/item/news/domestic/internal/parking-902956407/), Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), and the original viral Secret Tel-Aviv Facebook post (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).
*   **Slot 3 (2D Part Orienting):** Visualized as a geometric, metallic puzzle cube in a blister pack. The expanded modal highlights the C++ computational geometry push-plan algorithm, presenting a link to the open-source code and additional info at TAU (https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/).
*   **Slot 4 (The Turing Jest):** Wrapped like a premium deck of playing cards featuring a split comedy/tragedy mask. The text explains the LMSYS-like arena for LLM pun battles, alongside a glowing CTA link prompting the user to "Enter the Arena" (https://jest.geekip.dev).
*   **Slot 5 (Rhymes of Israel):** Packaged as an energy drink can with newspaper clippings wrapped around it. The expanded view describes the LLM Chrome extension that rewrites news headlines into comedic, high-school yearbook rhymes, providing a direct link to the Chrome Web Store (https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb).
*   **Slot 6 (Meta-Portfolio):** A deliberate visual easter-egg, represented as a miniature version of the vending machine itself. The modal humorously breaks the fourth wall to describe the very site the user is exploring, offering a self-referential loop link back home (https://geekip.dev).

**The Control Panel & Social Box (Slot 7)**
Instead of being just another item on the shelf, the 7th "box" acts as the vending machine's glowing, brushed-metal payment and control module located on the right side of the glass display. It features an interactive "Contactless Reader" graphic. Clicking this module opens the 7th section: a sleek digital holographic card displaying Geva's social ecosystem, with bold, iconography-driven links to GitHub (github.com/gevak), LinkedIn (linkedin.com/in/gevakip), and Facebook (facebook.com/geva.kipper).

**The Dispense Tray & Footer**
The bottom third of the screen features "The Tray"—a heavily shadowed, glass-door enclosure where clicked items temporarily appear before expanding into their modals. At the very base of the machine, stamped into what looks like an engraved metal chassis plate, is the website’s footer. Elegantly etched into the metal, it credits the AI model (e.g., "Architecture & Design generated by Assistant / ChatGPT") alongside a glowing red, mono-spaced digital stopwatch. The stopwatch actively ticks through hours, minutes, and seconds, visually tracking the exact time elapsed since the precise timestamp: 2026-03-07 01:21:48, grounding the website's daily-generative theme in real time.
