The website is a single-page scrolling canvas styled as an elegant, minimalist digital sketchbook lying flat. The background is a soft, slightly textured off-white (CSS grain overlay), mimicking premium art paper. There are no rigid grid lines or standard HTML containers; instead, layout sections are defined by fluid, hand-drawn SVG borders that appear to sketch themselves in real-time as they enter the viewport. 

The color palette is strictly monochrome graphite and fineliner ink (deep charcoals and blacks) paired with selective "watercolor" splashes that bloom to life in vibrant colors when a user hovers over interactive elements. Modern JavaScript and CSS are used to power SVG `stroke-dasharray` animations, creating the illusion of a rapid, invisible hand sketching the site frame by frame.

At the top of the canvas, the name **Geva Kipper** is written in large, animated typography resembling bold, wet brush strokes. Directly beneath it, nestled in a clean, elegant sans-serif font for contrast, is the sentence: *Building the future of search and software engineering at Google.* Anchoring the header is the subtitle, presented as an erratic, pencil-sketched note with a subtle CSS shake animation: *This website is re-imagined and re-designed each day by an AI—an ever-changing digital canvas.*

The main portfolio is presented as six asynchronous, scattered "sketch blocks" that tilt slightly at varied angles (via CSS transforms), resembling pieces of paper or polaroids taped into the sketchbook. As the user scrolls, each block's contents unfold via animated doodles:

**Box 1: GlglStats** 
The border is drawn as a looped audio waveform. Inside, a pencil animation sketches out overlapping bar charts and LZ77 compression nodes. A vibrant blue splash of watercolor fills the charts on hover. The description text is handwritten. Below it, a hand-drawn "play" button acts as the link to the [PyData TLV conference talk](https://www.youtube.com/watch?v=jrKAtOsj1Lo), framed inside a sketched television screen. A separate scribbled sticky note links to the live site at [https://glglstats.geekip.dev](https://glglstats.geekip.dev).

**Box 2: Ahuzat Dibuk** 
This box resembles a parking ticket tacked to the page. The doodle animation traces out a stylized street map of Tel-Aviv, which suddenly populates with deep red and orange gradient "heat-map" scribbles. The text details the Python scraping project for parking occupancies. Links to the news stories ([Channel 13](https://13tv.co.il/item/news/domestic/internal/parking-902956407/), [Mako](https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm)) are styled as torn newspaper clippings, while the [original Secret Tel Aviv Facebook post](https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/) looks like a blue-ink thumbprint.

**Box 3: 2D Part Orienting** 
An homage to computational geometry, the animations here are perfectly ruled, straight pencil lines and compass curves intersecting to form a polygon. A mechanical, robotic arm is sketched pushing the polygon, demonstrating the C++ push-plan algorithm. The links to the [project info and code](https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/) are enclosed in an intricate, geometrical isometric cube that rotates slightly playfully on cursor hover.

**Box 4: The Turing Jest** 
The doodle sketches two robot faces mimicking the classic theater comedy and tragedy masks. Between them, crossed jester wands are drawn. Speech bubbles materialize displaying mock puns, with animated five-star user ratings penciled below them. A neon-pink watercolor splash accents the LMSYS-style arena description. A doodled glowing button links out to the live arena at [https://jest.geekip.dev](https://jest.geekip.dev).

**Box 5: Rhymes of Israel** 
This block is laid out like a nostalgic high school yearbook page. The animation sketches out miniature, comedic caricatures of news anchors alongside rhyming couplets that type out in a typewriter font. The text explains the LLM-powered Chrome extension formatting Israeli news headlines. A hyperlinked illustration of a classic school composition notebook acts as the download button to the [Chrome Web Store](https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb).

**Box 6: The Meta Portfolio** 
A surreal, recursive Escher-style doodle of a hand drawing a hand, which is drawing a website. The text humorously points out the meta nature of displaying the portfolio site within the portfolio site. An infinity symbol is sketched rapidly in a loop, serving as the self-referential link back to [https://geekip.dev](https://geekip.dev).

**Section 7: Social Connections**
Instead of a standard footer menu, this section sits at the bottom right corner of the sketchbook, designed as a physical bookmark hanging off the page. It features beautifully messy ink-blot icons for GitHub ([github.com/gevak](https://github.com/gevak)), LinkedIn ([linkedin.com/in/gevakip](https://www.linkedin.com/in/gevakip)), and Facebook ([facebook.com/geva.kipper](https://facebook.com/geva.kipper)). Hovering over these icons makes the ink "smudge" dynamically via a CSS blur and translate effect.

**The Footer** 
At the absolute bottom center, styled like a librarian’s archival date stamp pressed into the bottom margin of the paper, is a neat, fixed-width typewriter font. It reads: *Design conceptualized and sketched by AI.* Right beside it, powered by a JavaScript interval, is a dynamic digital stopwatch ticking in real-time, displaying `HH:MM:SS` elapsed exactly since the genesis timestamp: **2026-03-27 01:49:14**.
