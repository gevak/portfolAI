The overall visual language of the site is an elegant, interactive "Digital Garden" combining organic, nature-inspired shapes with sleek glassmorphism. The color palette revolves around soft spring pastels—sage greens, cherry blossom pinks, lavender, and warm sunlight yellows—set against an off-white, textured background resembling high-quality watercolor paper. Typography pairs a modern, flowing serif font for headings (evoking vines and classical elegance) with a crisp sans-serif for body text. 

At the very top of the single-page layout, the Hero Section acts as the "Seed." The developer's name, "Geva Kipper," is displayed in large, elegant typography that appears to gently sway as if caught in a spring breeze. Beneath it, a clean sans-serif sentence reads: "Software Engineer at Google, cultivating scalable infrastructure and nurturing data-driven solutions." Below this, an italicized, semi-transparent subtitle anchors the concept: "This digital ecosystem is re-designed daily by AI. Like a real garden, it is ever-changing, unpredictable, and alive." A subtle pulsing arrow shaped like a falling leaf encourages the user to scroll. 

As the user begins to scroll, the core interactive mechanic of the site takes over: a central, twisting SVG "vine" draws itself dynamically down the center of the screen, tied directly to the user’s scroll position. The six project "boxes" are not traditional rectangles, but rather beautifully rendered, glassmorphic "terrariums" or stylized leaves branching off alternately to the left and right of the central vine. Using an Intersection Observer, each project container remains tightly closed like a pale green bud until it enters the viewport, at which point it uses a smooth CSS transition to "bloom," expanding outward and unfurling its contents, accompanied by small particle animations resembling floating pollen.

The first bloom on the left is **GlglStats**. The container unfurls into a wide, leaf-like shape. The text describes it as a Python project measuring the repetitiveness of Israeli pop-chart songs using LZ77 compression, complete with data visualizations of famous songs. The visual motif inside this container merges audio waveforms with floral patterns. It features two elegant, dew-drop shaped buttons: one linking to the live site at `https://glglstats.geekip.dev`, and a secondary link opening the PyData TLV presentation video (`https://www.youtube.com/watch?v=jrKAtOsj1Lo`).

Further down on the right, the second bloom opens for **Ahuzat Dibuk**. Its internal design features a subtle heatmap background transitioning from cool soil tones to warm yellows, representing the project's Tel-Aviv parking lot occupancy data. The text explains the Python scraping and visualization tool aimed at residents. The links are styled as sunlit badges: two leading to news coverage (`https://13tv.co.il/item/news/domestic/internal/parking-902956407/` and `https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm`), and a third linking to the original Secret Tel-Aviv Facebook post (`https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/`).

The third bloom, located on the left, is **2D Part Orienting**. To reflect the computational geometry aspect, this bloom unfurls into sharp, angular, succulent-like geometric petals. The text details the C++ implementation of an efficient algorithm constructing a 'push plan' to orient a polygon. A sleek, metallic-styled button links to the open-source library and code at `https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/`.

On the right, the fourth bloom reveals **The Turing Jest**. This container is shaped like a twin-headed orchid to represent the face-off between LLMs making and explaining puns in an LMSYS-like arena. The text prompts users to rate the models, with a standalone, prominent button linking directly to the arena at `https://jest.geekip.dev`.

The fifth bloom on the left is **Rhymes of Israel**. Its design incorporates elements of torn newspaper clippings dissolving into flower petals. The text presents the Chrome extension that utilizes an LLM to rewrite Israeli news headlines into comedic, highschool yearbook-style rhymes. An interactive installation button, styled like a blooming sunflower, links to the Chrome Web Store (`https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb`).

The sixth and final project bloom is a meta-reference: **The Portfolio Website**. Designed as a fractal, recursive visual, this element looks like a tiny glowing seedling containing a miniature version of the site itself. The description playfully points out the meta nature of the project. A self-referential link mimicking a glowing root points to `https://geekip.dev`.

At the bottom of the page, the central vine reaches the "Ground" (the footer section), bursting into a complex, beautifully tangled root system. This acts as the seventh section. Deep within the soil-colored, dark-brown footer background, three glowing nodes in the roots represent the social links: GitHub (`github.com/gevak`), LinkedIn (`linkedin.com/in/gevakip`), and Facebook (`facebook.com/geva.kipper`). 

Nestled at the very center of these roots is a glowing, mechanical "time-seed." Elegant, minimal text credits the design: "Cultivated and generated by AI." Just beneath this credit is a live, monospaced JavaScript stopwatch actively ticking upward representing the time elapsed (in hours, minutes, and seconds) since `2026-03-18 01:44:43`, serving as a digital heartbeat for the ever-evolving garden. Hovering over any element in this footer triggers a subtle upward drift of glowing firefly particles, completing the serene, living aesthetic of the page.
