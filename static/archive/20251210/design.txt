## Command Input Interface

Below the header sits a functional command input field styled as a terminal prompt: `geva@portfolio:~$ _` with a blinking block cursor. When visitors type, suggested commands appear as ghost text. Valid commands include: `ls projects`, `cat about`, `help`, `clear`, and clicking any project box executes `cat [project-name]`.

Each box appears as a "terminal window" with a draggable title bar (purely decorative), minimize/maximize/close buttons that trigger playful animations when hovered, and a semi-transparent dark glass background with green border glow.

A bottom bar styled as terminal status line shows: `Generated by Claude (Anthropic) | Uptime: XX:XX:XX` where the uptime counter ticks live since December 10, 2025 01:09:59. The elapsed time displays in HH:MM:SS format, updating every second, with subtle number-flip animations.

## Interactive Elements & Polish

- Typing any text triggers soft mechanical keyboard sounds
- Random "system messages" occasionally appear in corners like `[Memory OK]` or `[Rendering complete]`
- Easter egg: typing `sudo rm -rf /` triggers a fake "deletion" animation before everything reconstructs
- Subtle vignette darkening at screen edges enhances CRT feel
- All transitions use ease-out timing for snappy, responsive feel
