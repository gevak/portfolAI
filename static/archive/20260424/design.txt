The overall layout is an immersive, single-page, scroll-driven experience utilizing a "Z-axis infinite zoom" mechanic. The visual aesthetic is "Tech-Matryoshka." Instead of traditional painted wood, the Matryoshka dolls are constructed using modern glassmorphism—frosted glass curves with subtle, glowing neon outlines set against a dark, elegant space-gray background. This allows the user to faintly see the glowing shapes of the smaller dolls nested inside one another. 

At the very top of the page, floating outside the outermost doll, is the hero typography in clean, monospaced tech fonts. It prominently displays the name "Geva Kipper" followed by a sleek, single-line descriptor of his current software engineering work at Google. Just beneath this, styled as a glowing, glitch-effect terminal prompt, is the subtitle explaining that the website is redesigned each day by AI and is thus ever-changing. 

As the user begins to scroll down, the primary interaction triggers. The outermost glass Matryoshka doll splits horizontally perfectly in the middle. The top half floats upward and fades out, while the bottom half slides downward, creating a frame for the revealed content. Simultaneously, the camera "zooms in" on the nested doll inside, making it the new centerpiece. The space around this newly centered doll populates with the project information for the current layer. This interaction repeats for every single scroll step, peeling back layers of the doll to reveal the developer's work.

Layer one reveals **GlglStats**. The text elegantly describes the Python project measuring repetitive patterns in Israeli pop charts using LZ77 compression. Flanking the smaller doll in the center are interactive UI cards: one contains a direct button styled like a glowing compression algorithm graph linking to https://glglstats.geekip.dev. The other is an embedded, floating video player shaped with rounded corners, showing the PyData TLV presentation (https://www.youtube.com/watch?v=jrKAtOsj1Lo).

Scrolling deeper peels the doll to layer two: **Ahuzat Dibuk**. The UI shifts to a warm, heat-map gradient theme. Text explains the Python scraping and data visualization for Tel-Aviv parking lot occupancy. Clustered around the next tiny doll are modern, news-ticker style badge links pointing to the Channel 13 coverage, the Mako article, and the original viral Facebook post. 

Continuing the zoom, layer three unveils **2D Part Orienting**. The design reflects computational geometry, with background particles forming polygons. The text highlights the C++ open-source algorithm for constructing a polygon push plan, alongside a wireframe-styled button linking to the official CGL TAU project page. 

Layer four splits open to reveal **The Turing Jest**. The UI takes on a playful, retro-arcade aesthetic for this LMSYS-like arena. Text details the LLM pun-competition and rating system, accompanied by a glowing button shaped like a comedy mask pointing to https://jest.geekip.dev.

Diving into layer five presents **Rhymes of Israel**. The aesthetic turns into a nostalgic, high-school yearbook motif cut with modern Chrome UI elements. It explains the LLM-powered Chrome extension that rewrites news headlines into comedic rhymes, featuring a prominent "Add to Chrome" call-to-action button linking to the Chrome Web Store.

Layer six exposes **Hatul Basak**. The layout uses a Wikipedia-inspired typography mixed with trivia-card motifs. It describes the AI-generated and user-created Hebrew port of catfishing.net, offering an interactive puzzle-piece button that links to https://hatulbasak.geekip.dev to play and create daily trivia challenges.

Layer seven cracks open to show the ultimate meta-joke: **The Portfolio Website Project**. The text acknowledges the recursive nature of building the exact site the user is currently looking at. It features an "Inception-style" infinity mirror graphic and a self-referential link pointing back to https://geekip.dev.

Finally, the scroll reaches layer eight, the "Seed" or the core of the Matryoshka. This final doll does not split; instead, it is a solid, glowing metallic orb. As it spins gently on the screen, three sleek, minimalist icons orbit around it: GitHub (github.com/gevak), LinkedIn (linkedin.com/in/gevakip), and Facebook (facebook.com/geva.kipper), allowing the user to click and connect.

Anchored to the absolute bottom of the viewport viewport, layered over the dark background in a muted, elegant monospace font, is the footer. It features a line of text giving credit to "the AI model who created and designed this site," followed by a live, ticking digital stopwatch dynamically calculating and displaying the exact elapsed hours, minutes, and seconds since the timestamp `2026-04-24 02:04:15`.
