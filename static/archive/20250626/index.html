GlglStats

Python Data Visualization LZ77 Signal Processing

GlglStats is a project that measures the "repetitiveness" of songs from the Israeli pop charts (Galgalatz). Using the LZ77 compression algorithm as a proxy for complexity, it analyzes lyrical patterns to quantify how formulaic songs are and visualizes these trends over time.

Ahuzat Dibuk

Python Web Scraping Data Visualization Geo-Data

This project scrapes real-time occupancy data from Tel Aviv's municipal parking lots ("Ahuzat HaHof"). It then aggregates and visualizes this information, creating insightful heatmaps that help residents understand parking availability patterns throughout the city.

2D Part Orienting

C++ Computational Geometry Algorithms Open Source

A C++ implementation of an efficient algorithm to determine the orientation of a planar convex polygon. The algorithm, based on constructing a "push plan," can find the orientation with a minimal number of steps. This work was contributed to the CGAL open-source computational geometry library as part of a university project.
