**Overall Visual Concept & Layout Strategy**
The website features a sleek, single-page dark-mode aesthetic utilizing "glassmorphism" (translucent, frosted-glass panels with subtle drop shadows) to create depth against a deep obsidian background. The central architectural metaphor is a sprawling, interactive Periodic Table. The background grid consists of dimmed, minimalist atomic cells representing various programming languages and software frameworks (e.g., Py for Python, Cp for C++, Rct for React, Tf for TensorFlow). These language cells are color-coded into "element families" (AI, Backend, Frontend, Data Visualization). Dominating this grid, resting organically on top as oversized, illuminated "Isotope" blocks, are eight main project and information sections. As the user scrolls or moves their cursor, an elegant hover effect lights up the background framework elements that power each specific project. 

**Header & Typography**
At the top-left of the viewport, styled like a periodic table's legend, sits the developer's identity in bold, modern sans-serif typography: "Geva Kipper". Beneath the name is the bio, presented in a crisp, secondary font: "Software Engineer at Google, architecting highly scalable, data-driven backends and innovative digital experiences." Below this, utilizing a subtle CSS typewriter or glitch animation to draw the eye, is the dynamic subtitle: "This website is re-designed each day by AI and is thus ever-changing." 

**The Eight "Element Box" Sections**
The eight required sections are designed as expanded, highly detailed periodic table squares, each assigned an "Atomic Number" (1 through 8) and a 1-to-2 letter "Element Symbol". They are laid out in an asymmetrical, masonry-style grid over the foundational language table:

*   **Box 1 (Element 'Gl'): GlglStats**
    A sleek panel glowing with a subtle Python-blue gradient border. The title reads "GlglStats," accompanied by a description of its Python/LZ77 compression architecture used to measure and visualize repetitiveness in Israeli pop-chart songs. The links are styled as interactive valence electrons orbiting the text: one pulsing icon pointing to the live site (`https://glglstats.geekip.dev`) and an embedded thumbnail linking to the PyData TLV presentation (`https://www.youtube.com/watch?v=jrKAtOsj1Lo`). 
*   **Box 2 (Element 'Ad'): Ahuzat Dibuk**
    A data-themed cell featuring an abstract, low-opacity heatmap in the background. It explains the Python scraping and visualization framework used to monitor Tel-Aviv parking lot occupancy. Links to media coverage are presented as a mini "citations" list at the bottom of the card, with distinct pill-shaped buttons routing the user to News 13 (`https://13tv.co.il/item/news/domestic/internal/parking-902956407/`), Mako (`https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm`), and the original viral Secret Tel-Aviv Facebook post (`https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/`).
*   **Box 3 (Element 'Po'): 2D Part Orienting**
    Styled with sharp, geometric background accents representing polygons, this C++ domain cell details the open-source computational geometry implementation of a 'push plan' algorithm. A clean, terminal-styled button at the bottom right links to the code and documentation (`https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/`).
*   **Box 4 (Element 'Tj'): The Turing Jest**
    This block features a split-color tone to represent AI competition. The text introduces the LMSYS-like arena where LLMs battle through puns, subject to user rating. A prominent "Enter the Arena" glowing button directs users to `https://jest.geekip.dev`. Hovering over this card subtly highlights all LLM and Javascript framework cells in the background table.
*   **Box 5 (Element 'Ri'): Rhymes of Israel**
    A playful but clean card with yearbook-style subtle background doodles. It describes the LLM-powered Chrome extension formatting Israeli news headlines into comedic rhymes. A recognizable Chrome Web Store installation button anchors the bottom of the card (`https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb`).
*   **Box 6 (Element 'Hb'): Hatul Basak**
    A Wikipedia-trivia themed element box. It describes the Hebrew port of catfishing.net, emphasizing its AI-generated daily challenges and user puzzle creation system. A central interactive CTA button styled as a puzzle piece links directly to `https://hatulbasak.geekip.dev`.
*   **Box 7 (Element 'Pw'): Portfolio Meta-Element**
    Designed as a "recursive" visual loop, this card acts as an homage to the site itself. It explains that the user is currently looking at this very meta-portfolio project, complete with an infinitely retreating double-mirror graphic. The self-reference link points to `https://geekip.dev`.
*   **Box 8 (Element 'So'): Social Isotopes**
    Rather than a traditional text description, the final box is a cluster of three smaller sub-elements—**Gh** (GitHub: `github.com/gevak`), **Li** (LinkedIn: `linkedin.com/in/gevakip`), and **Fb** (Facebook: `facebook.com/geva.kipper`). Hovering over each triggers a unique neon glow (white for GitHub, blue for LinkedIn and Facebook) that ripples across the entire periodic table layout.

**Footer & Temporal Logic**
The bottom edge of the viewport houses a fixed, full-width, ultra-minimalist footer bar. On the left side, formatted in a pale, monospaced font, is the designer credit: "Site concept and architecture generated dynamically by an AI Model." Anchored to the right is the temporal anomaly component: a live JavaScript stopwatch utilizing a digital LED-style typeface, relentlessly ticking upwards to show the exact HOURS : MINUTES : SECONDS passed since the timestamp `2026-06-13 02:45:09`. Standardizing the countdown to handle absolute delta time ensures the interface retains its futuristic, slightly surreal sci-fi aesthetic.
