**General Aesthetic & Visual Identity**
The website is styled as a "Neo-Retro Desktop OS"—a sophisticated blend of nostalgic 90s graphical user interfaces and modern, fluid web design. The desktop background is a deep, elegant sage green featuring a subtle, animated CRT scanline overlay and a soft shadow vignette at the screen edges. Instead of harsh, blocky graphics, the UI employs "glassmorphism"; windows have classic 3D beveled borders but are made of frosted, semi-transparent glass with smooth CSS drop shadows. Typography marries a crisp, pixel-art font for headers (e.g., VT323) with a clean, highly legible modern monospace font for body text (e.g., JetBrains Mono).

**The Top Menu Bar**
Anchored to the top of the viewport is a classic Mac-style white menu bar. On the left side, it proudly displays the system user: "Geva Kipper // Software Engineer at Google, currently building scalable infrastructure and pushing the boundaries of web technologies." On the right side, rendered in a slightly glowing, pulsing yellow text, is the subtitle: "System Note: Daily AI Re-design Active. Layout and aesthetics are ever-changing." 

**The Desktop Interactions**
Scattered across the desktop are eight highly detailed, custom-designed pixel-art folder icons. Using JavaScript, these folders are fully draggable with realistic physics, casting soft shadows on the desktop. When a user double-clicks a folder, it expands via a smooth CSS scale-and-fade animation into a draggable, frosted-glass Application Window. Multiple windows can be open at once, complete with z-index stacking, focusing when clicked, and classic "minimize, maximize, close" title-bar buttons.

**Window Contents & Project Descriptions**
Each of the eight sections opens into a window uniquely themed to fit its content:

1. **GlglStats Viewer App:** This window mimics a retro media player merged with an analytics dashboard. A visualizer panel displays CSS-animated waveform repetition patterns of famous songs. A dedicated media frame embeds the PyData TLV presentation (https://www.youtube.com/watch?v=jrKAtOsj1Lo). The main interface contains a stylized button to launch the live LZ77 Python project (https://glglstats.geekip.dev).

2. **Ahuzat Dibuk Heat-Map Radar:** Designed to look like an old-school radar or municipal mapping software, this window features a glowing green grid background. It explains the Python scraping/occupancy project. The UI features three "Data Tape" icons linking to the news coverage (https://13tv.co.il/item/news/domestic/internal/parking-902956407/ and https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm) and a classic "BBS Notice board" icon linking to the Secret Tel Aviv Facebook post (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).

3. **2D Part Orienting CAD Editor:** Styled like a vintage Computer-Aided Design (CAD) interface. The window shows a wireframe polygon continuously orienting itself via CSS animations on a digital blueprint background. Next to the graphical display, a "Terminal" pane details the C++ computational geometry push-plan algorithm, with a prominent glowing hyperlink to the code and research (https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/).

4. **The Turing Jest Arena:** This window visually references classic arcade fighting games crossed with a command-line interface. The text explains the LMSYS-like arena for LLM pun battles. It features a bright red, flashing "ENTER ARENA" arcade-style button that leads to the user-rating site (https://jest.geekip.dev).

5. **Rhymes of Israel Scrapbook:** This window breaks the strict OS aesthetic, styled like a digital highschool yearbook full of scraped newspaper clippings. The clippings are dynamically rewritten by the LLM Chrome extension. A large, prominent "Install Extension" button styled as a floppy disk links directly to the Chrome Web Store (https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb).

6. **Hatul Basak Trivia Terminal:** Designed to look like a hypercard stack or vintage educational game. The window explains the Wikipedia trivia mechanics, AI daily challenges, and user-generated themed puzzles. A colorful "Play Daily Challenge" button directs the user to the interactive game portal (https://hatulbasak.geekip.dev).

7. **Meta Portfolio IDE:** Opening this folder reveals a window resembling a code editor (like VS Code but retro). Inside the text editor, it displays the HTML/CSS code that makes up the desktop site itself, referencing the meta-nature of the project. A "Compile/View" button links back to the root site (https://geekip.dev).

8. **Network Connections / Rolodex:** Styled as the OS's "Control Panel" for networking. It features interactive, embossed buttons shaped like ID cards for social connections. Clicking them links to GitHub (github.com/gevak), LinkedIn (linkedin.com/in/gevakip), and Facebook (facebook.com/geva.kipper).

**The Taskbar & Footer Clock**
At the bottom of the screen sits a thick, translucent taskbar. On the far left, an etched metallic badge reads: "System Architecture & Design generated by AI Model." On the far right is the "System Uptime" widget: a live JavaScript stopwatch modeled after a retro LCD digital display, continuously ticking up and displaying the exact hours, minutes, and seconds that have elapsed since the epoch timestamp: 2026-05-02 02:06:22.
