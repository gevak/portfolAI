The website follows a 3D isometric "escape room" layout built over a single page canvas, leveraging Three.js (or advanced CSS 3D transforms) to create a sleek, futuristic vault. The color palette is neo-noir elegant: polished dark obsidian, brushed steel, and sparse cinematic lighting with vibrant neon accents to guide the user's eye. The user is positioned in the center of the room, able to pan around. The room is filled with eight interactive objects (pedestals, consoles, and artifacts) serving as the "locks" to the escape room. Clicking an object triggers a smooth camera zoom, unlocking an intricate mechanical animation that reveals a sleek, frosted-glass "UI Card" containing the project details and links.

At the very top center of the screen, floating above the room as a permanent, glowing holographic projection, is the header: "Geva Kipper", followed by the sentence, "Currently building the future of web infrastructure as a Software Engineer at Google." Below this, flickering with a subtle digital glitch effect, is the subtitle: "Warning: Room layout is re-designed daily by AI and is thus ever-changing." 

Section 1 is disguised as a vintage, golden 3D tape deck retrofitted with modern LED audio dials. When clicked, it projects a data-compressed 3D waveform. The glass UI card describes 'GlglStats', a Python project measuring the repetitiveness of Israeli pop-chart songs using LZ77 compression and visualizing repetition patterns. It features two glowing metallic buttons linking to the live site at https://glglstats.geekip.dev and a PyData TLV conference video presentation at https://www.youtube.com/watch?v=jrKAtOsj1Lo.

Section 2 takes the form of a miniature, glowing 3D topographical map of Tel-Aviv under a glass cloche. Interacting with it illuminates the streets with a red-to-green heat-map. The UI card introduces 'Ahuzat Dibuk', a Python scraping and data visualization project for car parking lot occupancy. It includes three floating holographic tags directing to news coverage: Channel 13 (https://13tv.co.il/item/news/domestic/internal/parking-902956407/), Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), and the original Secret Tel Aviv Facebook publication (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).

Section 3 is a levitating, kinetic polygon suspended inside two rotating metal rings. When activated, a small mechanical arm inside the ring demonstrates an algorithm by pushing the polygon until it is perfectly oriented. The UI card presents '2D Part Orienting', highlighting the C++ implementation in an open-source computational geometry library used for constructing push plans. A terminal-style link directs users to the code and info at https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/.

Section 4 is represented by a dual-faced digital theater mask, half smiling and half frowning. Clicking it causes the masks to exchange speech bubbles of code before rendering the UI card for 'The Turing Jest'—an LMSYS-like arena where LLMs compete at making and explaining puns, rated by users. An "Enter the Arena" button links directly to https://jest.geekip.dev.

Section 5 sits on a pedestal as a classic high school yearbook that magically morphs into a glowing digital news tablet. Unlocking it showers the screen with comedic rhymes. The text explains 'Rhymes of Israel', a Chrome extension utilizing an LLM to rewrite Israeli news headlines into yearbook-style comedic rhymes. A stylized Chrome installation badge links to https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb.

Section 6 appears as an interactive obsidian puzzle box carved with the silhouette of a cat. When clicked, Wikipedia globe puzzle pieces scatter and magnetically snap together. The UI card introduces 'Hatul Basak', a Hebrew port of catfishing.net and a Wikipedia-based Trivia game featuring AI-generated daily challenges and user-created themed puzzles. A glowing game controller icon provides the link to play and create at https://hatulbasak.geekip.dev.

Section 7 is an intricate, free-standing digital mirror reflecting a miniature, recursive version of the entire 3D escape room. When selected, the mirror ripples, presenting the ultimate meta-project: this exact portfolio website. The text acknowledges the recursive nature of building the site currently being viewed, accompanied by an infinity-symbol link pointing to https://geekip.dev.

Section 8 poses as the biometric security terminal located directly next to the escape room's massive exit door. Activating the terminal scans a glowing handprint and pops up a futuristic ID badge containing the developer's social matrix. It features three sharply designed vector icons looping to his GitHub (github.com/gevak), LinkedIn (linkedin.com/in/gevakip), and Facebook (facebook.com/geva.kipper).

At the bottom of the viewport, serving as the footer deeply integrated into the heavy steel texture of the room's "Exit Door", is an engraved plaque reading: "Escape Room Environment Architected and Hallucinated by AI." Above this plaque, cast in aggressive, glowing red segmented LED typography, is an active stopwatch counting upwards in real-time. It displays the exact hours, minutes, and seconds that have elapsed since 2026-04-20 02:03:25, constantly ticking to remind the user how long they have ostensibly been locked inside the simulation.
