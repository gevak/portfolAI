**Global Aesthetic and Layout Overview**
The overarching aesthetic is a premium, "Tinder-for-Tech" glassmorphism UI styled in dark mode. The background is a fluid, slow-moving liquid gradient in deep hues of midnight blue, amethyst, and neon cyan. This fluid background serves as a visual metaphor for the site’s AI-generated, ever-changing nature. The entire interface is contained within a single viewport (no vertical scrolling). The layout is divided into three absolute-positioned layers: a top header for the developer’s identity, a centered interactive card stack for the projects, and a fixed footer for the AI credit and temporal data.

Immediately below the name, include the sentence: *"Software Engineer currently building and innovating at Google."* Beneath this, implement a monospaced, typewriter-effect subtitle in an accent color (like electric pink or green) that reads: *"Warning: This website is re-designed each day by AI and is thus in a state of ever-changing mutation."* 

**The Interactive Card Stack (The Match Mechanic)**
The core of the page is a centered deck of eight stacked cards, utilizing modern JavaScript touch/drag events for a swipe-based interface. As a recruiter drags a card horizontally, it rotates slightly. Swiping left overlays a bold, red "PASS" stamp on the card before it flies off-screen. Swiping right overlays a vibrant, green "MATCH" stamp; when a card is matched, its accompanying links are visually extracted and docked into a persistent, glowing "Matched Projects" tray fixed to the right side of the screen. Each card resembles a dating profile, featuring a unique visual header, a "bio" section describing the project, and stylized tap-buttons for the relevant links.
