**Global Aesthetic and Background Physics**
The website utilizes a full-viewport, deep-ocean gradient background (transitioning from midnight navy at the bottom to a bioluminescent cyan near the top) overlaid with a WebGL/Canvas-powered interactive particle system. This system simulates fluid dynamics and ocean spray. Thousands of translucent white, aqua, and seafoam-green particles drift gently like floating plankton or mist until the user's cursor moves. The cursor acts as a physical force—a gust of wind or a crashing wave—throwing particles outward with realistic velocity, friction, and gravity. As particles scatter, they slowly fall and dissolve, creating a mesmerizing, trailing wake of ocean spray behind the mouse. 

**Layout and Typography**
To contrast with the chaotic, organic motion of the background, the UI is built using elegant, ultra-minimalist glassmorphism. Text sits elegantly on frosted glass panels (`backdrop-filter: blur`) with incredibly thin, semi-transparent white borders. Typography relies on a pairing of an elegant serif for headings (e.g., Playfair Display or Lora) and a crisp, geometric sans-serif for body text and data (e.g., Inter or Roboto Mono). The overall layout is a dynamic masonry grid where the cards gently bob up and down along the Y-axis at random, slow intervals, mimicking buoys resting on ocean swells.

**Hero Section**
Anchored at the top center, a primary glass panel introduces the developer. In large, elegant serif font, it reads **Geva Kipper**. Directly underneath, a sleek sans-serif sentence states: "Software Engineer, currently building at Google." Below this, a subtle, horizontally scrolling ticker or softly glowing sub-headline reads: *"Redesigned each day by AI. An ever-changing digital landscape."* When the cursor enters this hero block, the background particles temporarily flock toward the edges of the box, framing the text in glowing sea mist.

**The Project Grid (8 Modules)**
Despite the conventional constraint of six boxes, the asymmetrical design gracefully accommodates eight floating modules. Each card features an elegant hover state: on hover, the glass becomes slightly more opaque, and an internal "splash" animation ripples across the card's surface. 

*   **Module 1: GlglStats** – Designed with subtle waveform graphics, this box introduces a Python project measuring the repetitiveness of Israeli pop-chart songs via LZ77 compression, complete with data visualizations. It features two elegant, pill-shaped buttons: one linking to the live site (https://glglstats.geekip.dev) and another styled with a subtle YouTube play icon linking to the PyData TLV conference presentation (https://www.youtube.com/watch?v=jrKAtOsj1Lo).
*   **Module 2: Ahuzat Dibuk** – Themed with a faint, warm-toned heatmap gradient bleeding through the glass. It explains the Python scraping and data visualization tool for Tel-Aviv parking lot occupancy. It includes a row of sleek, minimalist icons linking to its media coverage: Channel 13 (https://13tv.co.il/item/news/domestic/internal/parking-902956407/), Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), and the original viral Secret Tel Aviv Facebook post (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).
*   **Module 3: 2D Part Orienting** – Features a blueprint-style background of geometric polygons. The text details a C++ implementation within an open-source computational geometry library to construct a 'push plan' for orienting polygons. A stark, simple link directs to the academic project page and code (https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/).
*   **Module 4: The Turing Jest** – Designed with faint outlines of theatrical comedy masks. This box presents the LMSYS-like arena where LLMs compete at generating and explaining puns, judged by user ratings. A prominent hyperlink invites users into the arena (https://jest.geekip.dev).
*   **Module 5: Rhymes of Israel** – Styled lightly like a high school yearbook page with newspaper accents. It promotes a Chrome extension utilizing an LLM to rewrite Israeli news headlines into comedic rhymes. A stylized Chrome Web Store button links directly to installation (https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb).
*   **Module 6: Hatul Basak** – Incorporates a faint, cryptic question-mark motif. It introduces the Hebrew port of catfishing.net—a Wikipedia-based Trivia game featuring daily AI-generated challenges and user-created themed puzzles. A "Play & Create" link directs users to the game (https://hatulbasak.geekip.dev).
*   **Module 7: The Meta Portfolio** – A visually self-referential box containing a wireframed miniature of the webpage itself. It cheekily points out the meta nature of showcasing the very site you are currently visiting, with a self-reference link attached (https://geekip.dev).
*   **Module 8: The Connector** – A streamlined, purely navigational box dedicated to social and professional connections. It houses minimalistic vector logos for GitHub (github.com/gevak), LinkedIn (linkedin.com/in/gevakip), and Facebook (facebook.com/geva.kipper), which glow with a bioluminescent cyan when hovered.

**Footer and Timepiece**
The bottom edge of the screen features a full-width, frosted docking bar. On the left side, elegant text credits the AI: *"Conceived and generated by AI."* On the right side sits a live, ticking chronometer utilizing a crisp, monospaced terminal font. The text above the timer reads "Time elapsed since visual inception (2026-07-17 02:02:21):" followed by a rapidly updating display of Hours:Minutes:Seconds. Whenever the timer ticks a new minute, a gentle wave of particles pulses outward from the footer, integrating the time flow with the ocean spray physics.
