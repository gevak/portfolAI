**Global Aesthetic and Layout Strategy**
The site employs a dark-mode, cyber-cinematic aesthetic mixed with modern glassmorphism. The background is a deep obsidian (#0A0A0A) featuring a very subtle, slowly shifting gradient mesh in the background. The typography uses a high-contrast combination: an elegant, razor-sharp serif for headings and a clean, monospace sans-serif for project descriptions and technical data. The overall layout is a single, continuous scrolling page driven by a modern CSS Grid, utilizing GSAP (or similar JS libraries) for high-octane physics and animations. 

**The Hero Section: "Rage-Typing Simulator"**
Upon loading, the user is greeted by a nearly blank screen with a massive, glowing, neon-accented cursor and a faint, blinking prompt: "MASH KEYS TO GENERATE." The page intercepts all keyboard inputs. Every time the user aggressively mashes any key on their keyboard, rapid-fire JS events trigger simulated screen-shake, subtle spark particles, and mechanical typewriter audio cues. 
Instead of rendering what the user types, the keystrokes sequentially force Geva’s bio onto the screen in massive, beautifully kerned typography. 
The text that aggressively reveals itself, letter by letter per keystroke, is: 
**"Geva Kipper."** 
Below it, the Google single-sentence appears: **"Currently engineering scalable solutions and pushing the boundaries of software at Google."**
Beneath that, an elegant, smaller subtitle fades in autonomously: *“This website is re-designed each day by AI and is thus ever-changing.”*
Once this entire intro block is "typed out," the screen flashes with a satisfying "ACCESS GRANTED" micro-interaction, smoothly unlocking the scroll and fading in the project grid below.

**The Project Grid (The 8 Boxes)**
The rest of the single-page application is an asymmetrical masonry CSS grid. Each of the 8 sections is a "card" designed with a frosted-glass effect (backdrop-filter blur), subtle inner white borders, and an interactive 3D tilt effect mapped to mouse movement.

**Box 1: GlglStats**
Design this card with a dynamic, looping background visualization of audio waveforms or compression blocks. Keep the text overlay clean. Highlight the core features—Python, LZ77, and Israeli pop-chart repetitiveness. At the bottom, place two sleek, pill-shaped buttons: one linking to the live site (https://glglstats.geekip.dev) and the other featuring a recognizable YouTube play icon for the PyData TLV conference presentation (https://www.youtube.com/watch?v=jrKAtOsj1Lo).

**Box 2: Ahuzat Dibuk**
Give this card a subtle, animated heat-map glowing effect in its background, transitioning between oranges, reds, and purples to represent parking occupancy. Describe the Python scraping and data visualization. For the links, include a row of monochromatic media logos or stylized external link icons for Channel 13 (https://13tv.co.il/item/news/domestic/internal/parking-902956407/), Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), and a Facebook icon linking to the Secret Tel-Aviv post (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).

**Box 7: The Meta Website**

The text will cheekily acknowledge exactly what the user is looking at. Include a sleek self-reference link pointing back to the domain (https://geekip.dev).

**Box 8: Connect / Socials**
This final box breaks the heavy text mold and acts as a minimalist control panel. It features only three massive, beautifully animated SVG icons for GitHub (github.com/gevak), LinkedIn (linkedin.com/in/gevakip), and Facebook (facebook.com/geva.kipper). On hover, the icons should emit a neon glow corresponding to their brand colors.

**The Footer & Telemetry**
Pin a slender, full-width glassmorphic bar to the very bottom of the viewport. On the left side, aligned with a pulsing green "status" dot, write in a terminal font: "Design Conceptualized by AI Model." On the right side, incorporate a live, ticking JavaScript stopwatch. It should physically count the [Hours] : [Minutes] : [Seconds] displaying exactly how much time has passed since `2026-07-16 01:58:28`, keeping the numbers rolling smoothly to give the page a constant feeling of forward momentum.
