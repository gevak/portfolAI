**Global Visual Aesthetic & Architecture**
The website utilizes a modern, dark-mode glassmorphism aesthetic over a deep obsidian/midnight-blue background. The typography heavily relies on a crisp sans-serif font (like Inter or Roboto) for UI elements, contrasted closely with a monospace font (like Fira Code or JetBrains Mono) representing the developer's mathematical/code-driven nature. Accent colors are electric cyan and muted neon gold. The layout is a single-page CSS Grid utilizing seamless JavaScript transitions.

**State 1: The "13" Puzzle Unlocking Sequence**
When the page loads, the user does not see the portfolio initially. Instead, they are presented with an elegant, interactive mathematical puzzle taking up the full viewport. 
*   **Visuals:** A slowly shifting, interactive particle network of glowing nodes floats on the screen. Each node displays a random integer between 1 and 9. 
*   **Interaction:** The user acts as a pathfinder. Clicking and dragging between nodes draws an electric cyan line. A floating monospace cursor dynamically displays the sum of the connected nodes. 
*   **The Unlock:** When the user connects a sequence of nodes that perfectly adds up to the number **13**, the line flashes gold. The numbers dissolve, the nodes rapidly converge into the center of the screen forming a giant glowing "13", which then bursts into a shockwave animation, seamlessly fading out to reveal the unlocked portfolio matrix.
