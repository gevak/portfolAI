**Global Art Direction & Architecture**
The website utilizes a "Biometric Tech-Vault" aesthetic—a fusion of a high-end, dark-mode IDE (like VS Code) and elegant glassmorphism. The background is a very deep, rich charcoal (`#0B0E14`) with a subtle, slowly drifting mesh gradient of muted cyan and electric purple. The layout is a single-page, fluid grid. Typography relies on a crisp sans-serif (like *Inter*) for readable body text, and a geometric monospace (like *JetBrains Mono*) for all headers, metadata, and interactive code elements. 

**Header Section**
Centered at the top is the name **Geva Kipper** in large, luminous, debossed white text. Beneath it, styled as a glowing terminal prompt, is the bio: `> Software Engineer currently building, scaling, and optimizing systems at Google.` Below this, rendered in a smaller, shifting iridescent gradient (to imply mutation), is the subtitle: *"This website's layout and aesthetics are re-designed each day by AI, ensuring an ever-changing digital footprint."* 

**The Core Interaction: The Puzzle Grid**
The main body consists of an asymmetric masonry grid containing 8 "Crypto-Blocks" (cards). Initially, all blocks are locked. A locked block is styled as a frosted-glass panel containing only a flashing cursor and a minimalist padlock icon. 
When a user clicks a block, it expands slightly, blurring the rest of the page. The block prompts the user with a micro-coding puzzle (e.g., a one-line Python regex missing its pattern, or a JavaScript array operation missing a method). The user must type the solution directly into the card. Upon correct entry, the card flashes with a matrix-green border, a quick "decrypting" text scramble animation plays, and the project data permanently fades into view.

**Unlocked Content Blocks (The 8 Sections)**
Once decrypted, each card distinctively presents its content:

1. **GlglStats:** Features a background motif of a compressed audio waveform. The title is bold and stylized. The text explains the Python/LZ77 repetitiveness analysis of Israeli pop charts. Include two sleek, pill-shaped buttons: one labeled "[ Live Project ]" linking to `https://glglstats.geekip.dev`, and another with a play icon labeled "[ PyData TLV Talk ]" linking to the YouTube presentation. 
2. **Ahuzat Dibuk:** The card incorporates a subtle heatmap gradient (dark reds to bright yellows) in its top border. The text details the Python scraping project for Tel-Aviv parking occupancy. Links are displayed as minimalist press badges reading "Channel 13 News," "Mako Section," and "Secret Tel-Aviv Facebook," pointing to their respective URLs.
3. **2D Part Orienting:** Designed with a computational geometry flair, featuring a slow-rotating wireframe polygon in the background. The text describes the C++ push-plan algorithm. A monospaced link titled "Source & Documentation" points to the CGL TAU URL.
4. **The Turing Jest:** The layout inside this card mimics two opposing chat bubbles. The text introduces the LMSYS-like arena for LLM pun battles. An interactive-looking button reading "Enter the Arena" links to `https://jest.geekip.dev`.
5. **Rhymes of Israel:** Features a slightly more playful, high-school yearbook typography for its title, contrasting with the tech vibe. It explains the Chrome extension's LLM headline-rhyming capability, accompanied by a prominent "Add to Chrome" button linking to the Chrome Web Store.
6. **Hatul Basak:** Integrates a fragmented Wikipedia puzzle-globe motif mixed with a cat silhouette. The copy explains the Hebrew catfishing.net port, daily AI challenges, and user-themed trivia. A bold text-link reading "Play & Create Puzzles" directs to `https://hatulbasak.geekip.dev`.
7. **Meta Portfolio Site:** Designed uniquely as an "infinite mirror" or "window within a window" to represent the meta nature of the project. A self-reference hyperlink formatted as `< GeekIP.dev />` points back to `https://geekip.dev`.
8. **Social Nexus:** The final block, once unlocked, acts as a minimalist terminal directory. It displays high-contrast monochrome icons for GitHub, LinkedIn, and Facebook. Hovering over each icon injects its respective brand color (Hub-gray, LinkedIn-blue, Facebook-blue) and makes the URLs (`github.com/gevak`, `linkedin.com/in/gevakip`, `facebook.com/geva.kipper`) type themselves out beside the icon.
