The website will be designed as a first-person, pseudo-3D isometric or stylized flat-design convenience store aisle. The color palette features a mix of dark, cinematic midnight tones illuminated by warm, neon lighting. The layout utilizes CSS Grid and Flexbox to create tiered metallic or cream-colored shelving that scales responsively across devices, with subtle parallax scrolling to provide depth.

At the very top, mimicking a hanging illuminated store marquee, the header reads "Geva Kipper" in a bold, retro-modern typography. Attached beneath it is a glowing digital LED ticker that reads: "Currently engineering digital delicacies at Google." Hanging from a digital chain next to it is a crumpled, stylized paper sign containing the subtitle, written in a neon marker font: "Fresh Inventory! Layout re-designed each day by AI and is thus ever-changing."

The main body of the site consists of eight distinct "products" sitting on the shelves. Each project is represented as a clickable, high-fidelity 3D-CSS object (like a box, can, or toy). When the user hovers over an item, it glows and slightly floats. Clicking an item triggers a slick CSS transition, flipping the object around to reveal its "Nutritional Info" (the project details and links) in a sleek modal overlay designed to look like the back of the packaging.

1. The first item on the top shelf is a vibrant, transparent cassette tape labeled "GlglStats". The packaging details a Python project measuring the repetitiveness of Israeli pop-chart songs using LZ77 compression and visualizing the patterns. The back label features two glowing play buttons: one linking to the live site (https://glglstats.geekip.dev) and the other shaped like an MTV-era sticker, linking to the PyData TLV presentation (https://www.youtube.com/watch?v=jrKAtOsj1Lo).

2. Sitting neatly in a classic blister pack is a die-cast toy car labeled "Ahuzat Dibuk". The blister pack card displays a bright, thermal parking heat-map as its background. The text describes the Python scraping and data visualization project for Tel-Aviv parking lot occupancy. The "As Seen on TV" corner on the back features vintage TV and newspaper icons linking to Channel 13 (https://13tv.co.il/item/news/domestic/internal/parking-902956407/), Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), and the original Facebook publication (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).

3. The third item is a geometric, Rubik's Cube-style puzzle box labeled "2D Part Orienting". Its sleek, technical design reflects its C++ open-source computational geometry roots. The text on the reverse explains the efficient algorithm for constructing a push plan to orient polygons, complete with a barcode that acts as a hyperlink clicking through to the code and info (https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/).

4. On the middle shelf rests a retro mechanical gag toy or an arcade ticket dispenser branded "The Turing Jest". The vibrant, carnival-inspired design explains the LMSYS-like arena where LLMs compete at making and explaining puns for user ratings. A glowing arcade button on the back serves as the link to (https://jest.geekip.dev).

5. Next to it is a glossy, brightly colored highschool yearbook wrapped in plastic, titled "Rhymes of Israel". The "jacket flap" text details the Chrome extension that utilizes an LLM to rewrite Israeli news headlines into comedic rhymes. Instead of a price tag, it features an interactive "Tear-off Coupon" that says "Install Extension", linking to the Chrome Web Store (https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb).

6. The sixth item is a vintage trivia board game box labeled "Hatul Basak". Its cover art is an intricate mosaic of Wikipedia pages. The description explains it is a Hebrew port of catfishing.net, featuring AI-generated daily challenges and user-created themed puzzles. The game's punch-out tokens serve as interactive buttons linking directly to the platform to play and create puzzles (https://hatulbasak.geekip.dev).

7. Positioned as an eye-catching endcap display is a holographic trading card in a protective sleeve representing the Portfolio Website itself. The card mirrors the actual convenience store visuals, creating a recursive, meta visual loop. A sleek, foiled sticker on the card highlights the self-referential nature of the project and provides the link to (https://geekip.dev).

8. On the side of the shelving unit hangs a rotating wire postcard rack, holding three distinct, stylized plastic loyalty cards. These act as the contact section, seamlessly integrating the developer's socials. The three interactive cards feature the recognizable branding of their respective platforms, linking out to github.com/gevak, linkedin.com/in/gevakip, and facebook.com/geva.kipper.

The bottom of the page transitions into a glowing, retro cash register checkout counter. An animated digital receipt prints out the footer information. The top line of the receipt gives credit, reading: "Store constructed by AI". Below that is a green, 7-segment LCD style display embedded in the register acting as a live stopwatch. It dynamically counts the hours, minutes, and seconds that have elapsed since "2026-07-11 02:00:17", blending the interactive tech theme with the retro convenience store aesthetic.
