The initial state of the application is a full-screen, deep obsidian viewport that acts as the entry puzzle. Centered on the screen is a minimalist "digital cryptex" made of three concentric, frosted-glass rings. Each ring contains scattered, glowing cyan code syntax (brackets, semicolons, and operators). To unlock the site, the user must click and drag to rotate the rings, aligning three specific characters across the vertical axis to form a complete HTML tag: ` < ` on the outer ring, ` / ` on the middle ring, and ` > ` on the inner ring. Upon exact alignment, the locked state triggers an unlocking animation: the rings glow intensely, emit a soft chime, and rapidly expand outward until they disappear off-screen, dissolving the dark overlay to reveal the main portfolio beneath.

The main portfolio utilizes a dark-mode glassmorphism aesthetic with a slow-moving, abstract fluid-mesh background in subtle tones of deep indigo and violet. The header is locked to the top-left, featuring the name "Geva Kipper" in a bold, modern sans-serif typeface, followed by the sentence: "Taking part in shaping the future of the web as a Software Engineer at Google." Right beneath it, in a slightly dimmer, monospace font, sits the subtitle: "Notice: This website is completely re-designed each day by an AI and is thus ever-changing." 

The core of the site is an asymmetrical masonry grid composed of seven floating, translucent frosted-glass cards. Each card features subtle 3D tilt effects on mouse hover and a distinct, CSS-animated thematic background that visually represents the project without overwhelming the text.

Card 1 (GlglStats) features a live, CSS-animated audio equalizer graphic at its base. The text describes a Python project utilizing LZ77 compression to measure and visualize repetitiveness in Israeli pop-chart songs. Elegant neon buttons provide links to the live site at https://glglstats.geekip.dev and to the PyData TLV conference presentation at https://www.youtube.com/watch?v=jrKAtOsj1Lo.

Card 2 (Ahuzat Dibuk) utilizes a faint red-to-green heat-map radial gradient as its background. The text outlines the Python scraping and data visualization project that tracks Tel-Aviv parking lot occupancy. It includes embedded badge links to media coverage on Channel 13 (https://13tv.co.il/item/news/domestic/internal/parking-902956407/), Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), and the original viral Facebook post (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).

Card 3 (2D Part Orienting) features a slow-rotating, wireframe polygon in the top right corner. The section describes the C++ implementation of a "push plan" algorithm for polygon orientation within an open-source computational geometry library. A deeply integrated text link directs users to the code and additional info at https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/.

Card 4 (The Turing Jest) splits its background diagonally with a subtle light/dark contrast to emulate a battle arena. It details an LMSYS-like platform where LLMs compete at making and explaining puns, rated by users. A prominent CTA button points to https://jest.geekip.dev.

Card 5 (Rhymes of Israel) utilizes a textured background reminiscent of a high school yearbook page with overlaid animated typewriter text. It introduces the Chrome extension that leverages an LLM to rewrite Israeli news headlines into comedic rhymes. A stylized Chrome icon links to the installation page at https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb.

Card 6 (The Portfolio Meta-Project) contains a CSS "infinite mirror" depth effect behind the text. It playfully details the creation of the very website the user is currently viewing, acknowledging the meta nature of the project. It includes a self-referencing link back to https://geekip.dev.

Card 7 (Connect) is a smaller, wider card anchored at the bottom of the grid, entirely dedicated to social and professional connections. It houses three large, glowing, minimalist wireframe icons that link directly to github.com/gevak, linkedin.com/in/gevakip, and facebook.com/geva.kipper. 

The footer rests at the absolute bottom of the single-page layout, styled like a developer's terminal bar. It features crisp, green monospace text reading: "Design generated and implemented by Gemini AI." Next to this credit is a live JavaScript stopwatch that dynamically ticks up, displaying the exact hours, minutes, and seconds that have passed since "2026-03-19 01:45:59", reinforcing the concept of a site that is constantly alive and machine-driven.
