**Overall Theme and Aesthetic**
The website should be designed as a retro-futuristic Apollo-era Mission Control dashboard viewed through a modern, elegant lens. The background is a deep void black (`#090a0f`) featuring a subtle, slowly translating navigational grid and a faint CRT monitor scanline overlay. The layout utilizes a "Bento-box" style asymmetrical CSS grid composed of brushed-metal charcoal panels with glowing cyan and amber telemetry accents. Typography must exclusively use monospace fonts (like IBM Plex Mono or JetBrains Mono) for headings and UI elements, paired with a highly legible sans-serif for body text. Interactions should feel mechanical—hover states should mimic illuminated toggle switches, and text should load with a blinking cursor typing effect.
