**Initial Load: The "Chaotic Desktop" State**
Upon page load, the user is immediately confronted with a deliberately overwhelming, full-screen "desktop" environment layered with chaotic, overlapping browser tabs and system windows. These windows feature anxiety-inducing UI elements: flashing terminal errors, overlapping code snippets, "Urgent: Fix Immediate" alerts, and cluttered stack traces. The z-index is a mess, creating a sense of digital claustrophobia. However, each chaotic window features a prominent, satisfyingly styled red "X" close button in its corner. To see the actual portfolio, the user must interact with the site by frantically clicking these "X" buttons. As each window is dismissed, it crumples, dissolves, or swiftly slides off-screen smoothly, slowly revealing the pristine portfolio beneath. Optional: A single "Force Quit All" button sits in the corner for impatient users.

**The Reveal: The Hero Interface**
Once the final chaotic tab is closed, a sweeping glassmorphic blur clears the screen, revealing an ultra-modern, deeply organized, dark-mode minimalist interface. At the top left, in elegant, cinematic typography, is the developer’s name: **Geva Kipper**. Right beside or beneath it, a sleek, muted text reads: "Currently engineering scalable solutions and systems at Google." Anchoring the header is the subtitle, rendered in a subtle, glowing, shifting generative font to emphasize the AI theme: *"This environment is re-imagined daily by AI; embrace the ephemeral."*

**The Layout: The "Organized Mind" Bento Grid**
The core of the site functions as an asymmetric "Bento Box" grid. Each of the eight sections is a beautifully spaced, softly rounded card with subtle lighting effects that respond to the user's mouse movements. When not hovered, the cards look matte and uniform; upon hover, they come alive with project-specific micro-animations.

**Card 1: GlglStats**
This card features a dark background with a subtly animated soundwave or data-compression visualization (representing the LZ77 algorithm). The text highlights the Python project and its analysis of repetitiveness in Israeli pop-chart songs. Interaction elements include a sleek globe icon linking to `https://glglstats.geekip.dev` and a glowing YouTube play-button icon linking to the PyData TLV conference presentation (`https://www.youtube.com/watch?v=jrKAtOsj1Lo`).

**Card 2: Ahuzat Dibuk**
This section utilizes a background that shifts between red and green gradients, mimicking a parking heat-map. It details the Python scraping and data visualization project for Tel-Aviv parking lot occupancies. The card features a localized cluster of news logos—Channel 13 (`https://13tv.co.il/item/news/domestic/internal/parking-902956407/`), Mako (`https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm`), and a classic blue Facebook icon pointing to the Secret Tel Aviv post (`https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/`).

**Card 3: 2D Part Orienting**
Designed with a computational, wireframe aesthetic, this card features a geometric polygon that gently rotates on hover, demonstrating a "push plan." The typography is slightly monospaced to reflect the C++ open-source library nature of the project. A stylized "Source & Info" button links directly to the computational geometry library page (`https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/`).

**Card 4: The Turing Jest**
This card has a split-screen "Versus" aesthetic, representing the LMSYS-like arena where LLMs compete in pun-making. Hovering reveals subtle theatrical masks or comedy iconography intertwined with digital nodes. A prominent, bold button invites the user to judge the puns, linking to `https://jest.geekip.dev`.

**Card 5: Rhymes of Israel**
The design of this card blends the look of a traditional, monochrome newspaper headline with colorful, nostalgic highschool yearbook scribbles in the margins. It explains the LLM Chrome extension that converts Israeli news into comedic rhymes. A classic Chrome Web Store badge sits at the bottom, linking to `https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb`.

**Card 6: Hatul Basak**
This card features a playful yet encyclopedic design, perhaps a stylized jigsaw puzzle piece or a subtle cat silhouette made of Wikipedia-style text. It describes the Hebrew port of catfishing.net, highlighting the AI daily generation and user-created themed Wikipedia trivia puzzles. A "Play & Create" call-to-action button links to `https://hatulbasak.geekip.dev`.

**Card 7: The Meta Portfolio**
A purely self-referential card featuring an "infinity mirror" visual effect or a wireframe miniature of the exact Bento grid the user is currently looking at. It jokingly points out the meta nature of showcasing the portfolio website project on the portfolio website itself, with a glowing loop icon linking back to `https://geekip.dev`.

**Card 8: The Social Node**
A condensed, highly polished control panel acting as the final card. It lacks heavy text, instead featuring three large, interactive, 3D-styled icons that light up with their respective brand colors when hovered: GitHub (`github.com/gevak`), LinkedIn (`linkedin.com/in/gevakip`), and Facebook (`facebook.com/geva.kipper`). 

**The Footer: AI Credit & Temporal Tracker**
At the very bottom of the page, floating outside the grid in a stark, technical monospaced font, is the footer. It reads: *"Architecture and logic synthesized by AI."* Beside it, serving as a live digital heartbeat for the site, is a ticking stopwatch. It features a glowing red or neon-blue counter displaying exact hours, minutes, and seconds, preceded by the text: *"Time elapsed since origin 2026-05-07 02:14:18: [HH:MM:SS]"*, constantly ticking upward as long as the page is open.
