**Overall Aesthetic and Concept**
The aesthetic is "Ethereal Cyber-Terminal." The interface merges the nostalgic, text-driven functionality of a command-line interface (CLI) with highly modern, elegant glassmorphism GUI elements. The background is a very dark deep-blue/matte black gradient (#0A0E17 to #000000). The layout is split: the left half (or a persistent floating window) acts as the active Terminal Prompt, while the right half (or the background) serves as a digital canvas where the "Boxes" (project cards) materialize as holographic, frosted-glass windows when summoned by user commands. Custom monospace fonts (like JetBrains Mono or Fira Code) control the terminal aspect, while sleek sans-serif typography is used inside the generated project boxes.

**Boot Sequence & Header Typography**
When the user arrives, the terminal executes a mock "boot-up" sequence, typing out the header information like a server message-of-the-day (MOTD). It types: `User: Geva Kipper`. Below it, in a slightly diminished neon-green text: `Status: Currently building scalable software solutions at Google.` Finally, a warning message flashes in soft magenta text: `System Notice: This interface is hallucinated and re-designed daily by AI. Expect permanent evolution.` This header remains pinned to the top of the terminal interface.

**The Interactive Prompt Mechanism**
Below the header is the active command line: `guest@geekip.dev:~ $ [blinking cursor]`. Users can freely type into this field. To ensure recruits and visitors know what to do, an auto-complete feature and a row of subtle "Suggested Commands" (styled like clickable tags, e.g., `[run glglstats]`, `[cat socials]`, `[ls projects]`) sit just above the input. When a user types a command (or clicks a suggestion) and hits Enter, the terminal prints a brief "Loading..." aesthetic, and the corresponding project box slides and fades into view on the right-hand canvas. 

**The Seven Dynamic Output Modules (The Boxes)**
When summoned via the terminal, these sections render as elegant, semi-transparent frosted glass cards with subtle border-glows corresponding to the project's theme. 
*   **Box 1 (GlglStats):** Summoned via `./glglstats`. The card features an abstract, animated SVG wave pattern representing audio compression. It highlights the use of Python and LZ77 algorithms to measure Israeli pop-chart repetitiveness. Two sleek, executable-style buttons link out: one to the live visualization site (https://glglstats.geekip.dev) and an embedded link to the PyData TLV conference video (https://www.youtube.com/watch?v=jrKAtOsj1Lo).
*   **Box 2 (Ahuzat Dibuk):** Summoned via `./ahuzat-dibuk`. Rendered with a background overlay of a glowing mini heat-map. The text describes the Python scraping and data visualization of Tel-Aviv parking lot occupancy. "Press & Media Logs" are presented as interactive terminal links connecting to Channel 13 (https://13tv.co.il/item/news/domestic/internal/parking-902956407/), Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), and the original viral Secret Tel-Aviv Facebook post (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).
*   **Box 3 (2D Part Orienting):** Summoned via `./orient-2d`. This box features a minimalist animation of a geometric polygon rotating and locking into place. It explains the C++ implementation in an open-source computational geometry library used for constructing algorithm push plans. A technical documentation button links to the TAU project page (https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/).
*   **Box 4 (The Turing Jest):** Summoned via `./turing-jest`. Designed to look like a split-screen versus battle layout. It details the LMSYS-style arena where LLMs compete to make and explain puns based on user ratings. A prominent hyperlink points to the live arena (https://jest.geekip.dev).
*   **Box 5 (Rhymes of Israel):** Summoned via `./rhymes`. The aesthetic of this card playfully mixes CLI harshness with a nostalgic, doodled high school yearbook vibe. It details the LLM-powered Chrome extension formatting Israeli news out of comedic rhymes. A stylized "Add to Chrome" button links to the web store (https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb).
*   **Box 6 (Meta Portfolio):** Summoned via `./portfolio`. This box creates a visual infinite-mirror or recursive feedback loop effect. It contains a self-referential joke about building the very site the user is looking at, with a cyclical link back to the domain (https://geekip.dev).
*   **Box 7 (Socials & Contact):** Summoned via `cd accounts`. A sleek connection-hub panel showing server-like nodes that link outwardly to external networks. Includes glowing icon-links for GitHub (github.com/gevak), LinkedIn (linkedin.com/in/gevakip), and Facebook (facebook.com/geva.kipper).

**Footer Panel**
At the very bottom of the viewport is a docked, full-width status bar, styled like a code editor’s footer (e.g., VSCode or Vim). On the left side, muted monospace text reads: `Design Architecture: LLM Core - Auto-generated iteration`. On the far right is the JavaScript-powered live stopwatch ticking up endlessly. It explicitly tracks the elapsed time, formatted as `Uptime: [X]hrs : [Y]mins : [Z]secs since 2026-03-08 01:27:56`.
