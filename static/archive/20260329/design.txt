**Overall Theme & Aesthetic**
The website operates as a full-viewport, interactive "brain dump." The dark-mode background (deep obsidian or midnight blue) is an HTML5 Canvas or WebGL space where glowing, semi-transparent neural pathways continuously pulse with light. Instead of traditional grid-based squares, the "boxes" are rendered as organic, floating glassmorphic "synaptic nodes." The UI marries cyberpunk elegance with modern web design—utilizing frosted glass textures, subtle neon drop-shadows, and smooth physics-based drifting animations. When a user hovers over any node, glowing lines (SVG or Canvas) dynamically draw themselves to connect that node to the central core, simulating a firing synapse.

**The Central Core (Hero Section)**
In the center of the viewport floats the largest, brightest node—the nucleus of the mind map. 
*   **Primary Typography:** "Geva Kipper" is written in a stark, modern sans-serif (like Inter or Roboto Mono) with a soft, pulsating back-glow.
*   **Google Line:** Directly underneath, in crisp white text: "Currently engineering intelligent systems and scalable solutions at Google."
*   **AI Subtitle:** A smaller, monochromatic terminal-style font sits below this, with a subtle glitch or raw-typing animation that reads: "Visual architecture generated by an artificial neural network. Re-designed daily, evolving endlessly."

**The Neural Nodes (Project Sections)**
Orbiting the central core are seven distinct nodes. Each acts as a clickable/expandable card. In their default state, they show a glowing icon and a title. On hover or click, the frosted glass expands smoothly via CSS transitions to reveal the description and links. The text within the nodes is clean and highly legible against dark, blurred backgrounds.

*   **Node 1: GlglStats** 
    *   *Visuals:* Emits a purple neon glow. The icon incorporates a soundwave or a compression zipper.
    *   *Content:* Describes the Python project measuring the repetitiveness of Israeli pop-chart songs using LZ77 compression and visualizing the patterns of famous tracks.
    *   *Links:* Contains two glowing, pill-shaped buttons. One linking to the live site (`https://glglstats.geekip.dev`) and a YouTube play-button icon linking to the PyData TLV presentation (`https://www.youtube.com/watch?v=jrKAtOsj1Lo`).

*   **Node 2: Ahuzat Dibuk**
    *   *Visuals:* Emits a warm orange/red glow (mimicking a heat-map).
    *   *Content:* Details the Python scraping and data visualization project for Tel-Aviv parking lot occupancy. 
    *   *Links:* Three minimalist external-link icons bridging to news coverage on 13tv (`https://13tv.co.il/item/news/domestic/internal/parking-902956407/`), Mako (`https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm`), and the original Secret Tel-Aviv Facebook post (`https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/`).

*   **Node 3: 2D Part Orienting**
    *   *Visuals:* Emits a cyan or electric blue glow. Features an abstract polygon geometric vector.
    *   *Content:* Explains the C++ implementation inside an open-source computational geometry library, focusing on the efficient algorithm for constructing push plans to orient polygons. 
    *   *Links:* A text link reading "View Academic & Code Repositories" pointing to `https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/`.

*   **Node 4: The Turing Jest**
    *   *Visuals:* Emits a bright yellow or gold glow. Features a minimalist jester hat woven into a microchip.
    *   *Content:* Presents the LMSYS-like arena where LLMs compete in crafting and explaining puns, rated by human users.
    *   *Links:* A primary call-to-action button linking directly to `https://jest.geekip.dev`.

*   **Node 5: Rhymes of Israel**
    *   *Visuals:* Emits a vibrant lime-green glow. Features a newspaper icon intertwined with a magic wand or sparkles.
    *   *Content:* Describes the LLM-powered Chrome extension formatting Israeli news headlines into comedic, high-school-yearbook-style rhymes.
    *   *Links:* A stylized Chrome Web Store button linking to `https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb`.

*   **Node 6: The Meta Portfolio**
    *   *Visuals:* Emits a magenta or pink neon glow. The icon is an infinite loop or a self-referencing code bracket `< />`.
    *   *Content:* Playfully addresses the meta-nature of the current site.
    *   *Links:* A self-referencing anchor linking back to `https://geekip.dev`.

*   **Node 7: Social Synapses**
    *   *Visuals:* A tighter cluster of three smaller, stark-white glowing nodes acting as output terminals.
    *   *Content:* Instead of descriptions, these are purely iconic representations (GitHub, LinkedIn, Facebook).
    *   *Links:* Connecting to `github.com/gevak`, `linkedin.com/in/gevakip`, and `facebook.com/geva.kipper`.

**HUD Footer (The Timekeeper)**
Fixed to the bottom of the viewport is a sleek, transparent bar styled like a sci-fi Heads-Up Display (HUD), spanning the width of the screen. 
*   On the left side, formatted in a monospace accent color (like cyan), text reads: "Architected by Google Gemini."
*   On the right side, a live JavaScript-driven stopwatch dynamically ticks up. It displays precisely how much time (Years, Months, Days, Hours, Minutes, Seconds) has elapsed since the temporal anchor: `2026-03-29 01:51:45`. The numbers should pulse slightly upon every passing second, grounding the fluid, ever-changing AI aesthetic in precise, rigorous timekeeping.
