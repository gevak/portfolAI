**Overall Visual Concept & Layout**
The website utilizes a modern, elegant "Faux OS Desktop" interface built on principles of Glassmorphism. The background is a fluid, slow-moving abstract gradient that visually represents the AI's "daily redesign" process. The UX is fully interactive; visitors can drag, drop, minimize, and bring to focus various floating "windows" or "folders" scattered across the canvas. The design eschews clunky retro-OS aesthetics in favor of a sleek, futuristic, and highly polished spatial-computing UI with subtle drop shadows and frosted glass blur effects. 

**System Taskbar (The Header)**
Spanning the top of the browser viewport is a translucent OS menu bar. 
*   **Left Corner:** Bold, minimalist typography reading "Geva Kipper". Beneath it or beside it in a lighter, smaller font: "Software Engineer currently solving complex problems at Google."
*   **Center:** A subtly pulsating, glowing notification tag that reads: "System Notice: Interface re-designed daily by AI. Expect constant evolution."

**Interactive Desktop & The Project Windows (Boxes 1-6)**
By default, the desktop displays six open, freely floating, Draggable Windows. Each window features a title bar (for dragging) with distinct "close" and "minimize" macOS-style dots, and uniquely themed interior content. When a user clicks a window, it fluidly transitions its Z-index to come to the front. 

*   **Window 1: GlglStats.py**
    *   *Visuals:* The interior features a stylized, vibrant line-graph background symbolizing LZ77 compression and music repetition.
    *   *Content:* Text explaining the Python project that measures repetitiveness of Israeli pop-chart songs and visualizes the patterns. 
    *   *Links:* A prominent primary button reading "Launch glglstats.geekip.dev". Below it, an embedded minimalist media player or a sleek video-thumbnail hyperlink pointing to the PyData TLV presentation (https://www.youtube.com/watch?v=jrKAtOsj1Lo).

*   **Window 2: Ahuzat_Dibuk.exe**
    *   *Visuals:* Features an ambient, glowing heat-map graphic fading into the background.
    *   *Content:* A description of the Python scraping and data visualization tool for calculating Tel Aviv parking lot occupancy for local residents.
    *   *Links:* Formatted as a "Press & Media" sub-folder list containing links to Channel 13 (https://13tv.co.il/item/news/domestic/internal/parking-902956407/) and Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm). Next to it, a "Community" link pointing to the original Facebook Secret Tel Aviv publication (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).

*   **Window 3: 2D_Part_Orienting.cpp**
    *   *Visuals:* A stark, elegant geometric aesthetic—perhaps rotating, wireframe polygons softly animating behind the text.
    *   *Content:* Details the C++ computational geometry project utilizing an efficient algorithm for constructing a "push plan" to orient given polygons.
    *   *Links:* A terminal-style hyperlink branching to the computational geometry library project page (https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/).

*   **Window 4: Turing_Jest.run**
    *   *Visuals:* A split-screen, red-vs-blue subtle tint inside the window to mimic an LMSYS combat arena. 
    *   *Content:* Describes the arena where LLMs compete at making and explaining puns, subsequently rated by human users.
    *   *Links:* An arcade-like, glowing "ENTER ARENA" button linking to https://jest.geekip.dev.

*   **Window 5: Rhymes_of_Israel.crx**
    *   *Visuals:* A playful UI layer resembling a highschool yearbook page with subtle doodles framing the text. 
    *   *Content:* Outlines the Chrome extension that utilizes an LLM to rewrite Israeli news headlines into comedic, yearbook-style rhymes.
    *   *Links:* A standard, recognized "Available in Chrome Web Store" badge linking directly to the install page (https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb).

*   **Window 6: index.html (The Meta Project)**
    *   *Visuals:* A recursive visual trick—this window displays a miniature, infinite "mirror reflection" of the current website design. 
    *   *Content:* A tongue-in-cheek explanation that the user is currently interacting with the very portfolio project being described.
    *   *Links:* A self-referential hyperloop button pointing back to https://geekip.dev.
