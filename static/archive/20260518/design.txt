The overall layout centers on a dark-mode, minimalist 3D-effect canvas featuring a massive, sleek, retro-futuristic digital alarm clock. This clock interface serves as the entire single-page application, taking up 80% of the viewport. It is rendered using modern CSS glassmorphism and subtle drop shadows to make it feel tactile. At the top of the clock casing, engraved on a metallic "manufacturer’s plate," is the header: "Geva Kipper – Software Engineer at Google, building intelligent, large-scale systems." Just below this plaque, a flickering LED-style subtitle reads: "Notice: This interface is re-designed daily by AI and is constantly evolving."

The primary user interaction is driven by the clock's physical-looking dials and buttons. Along the side or bottom of the clock face are tactile HTML/CSS buttons labeled `< HOUR`, `HOUR >`, and a large `SNOOZE / NEXT` button on top. Instead of scrolling down a page, the user clicks these buttons to change the "Alarm Time." The clock features a large central digital LED display. When the user changes the time, the glowing numbers (e.g., "01:00 AM") briefly flash, dissolve smoothly via CSS transitions, and morph into project content boxes. There are eight specific times the user can tune into, each unveiling one of the required sections styled as a glowing dashboard inside the clock face.

**01:00 AM: GlglStats** 
The LED display glows neon cyan, showing a dynamic, pixelated equalizer background to mimic pop-chart music. The content panel introduces 'GlglStats', an LZ77 compression-based Python project measuring the repetitiveness of Israeli pop songs with data visualizations. The panel features two retro digital buttons: "Launch Live App" linked to `https://glglstats.geekip.dev`, and a "PyData TLV Presentation" button linked directly to the YouTube video `https://www.youtube.com/watch?v=jrKAtOsj1Lo`.

**02:00 AM: Ahuzat Dibuk** 
The display shifts to a deep crimson hue. A low-resolution heatmap gradient gently pulses in the background. The text details the Python scraping and data visualization tool used to generate Tel-Aviv parking lot occupancy heat-maps. Below the text, three blinking indicator lights act as external links to the media coverage: one pointing to Channel 13 (`https://13tv.co.il/item/news/domestic/internal/parking-902956407/`), one to Mako (`https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm`), and one linking to the original Secret Tel Aviv Facebook release (`https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/`).

**03:00 AM: 2D Part Orienting** 
The display transitions into a neon-green geometric grid wireframe. A slow-rotating polygon sits behind the text, representing the computational geometry algorithm. The text highlights the C++ open-source library implementation for generating a 'push plan' to orient a polygon. A glowing "Execute Code" terminal-style button directs users to `https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/`.

**04:00 AM: The Turing Jest** 
The interface splits into an 8-bit boxing ring aesthetic. Bold typography introduces an LMSYS-style arena where LLMs compete in making and explaining puns, rated by human judges. A prominent, flashing "Enter Arena" button points users directly to `https://jest.geekip.dev`.

**05:00 AM: Rhymes of Israel** 
The screen illuminates with bright yellow LED text using a pixelated, high-school yearbook nostalgic font. The background features subtle animations of floating graduation caps and newspaper icons. The section describes the Chrome extension utilizing an LLM to rewrite Israeli news headlines into comedic rhymes. A stylized "Add to Browser" button links to the Chrome Web Store extension page (`https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb`).

**06:00 AM: Hatul Basak** 
The clock face takes on a synthwave, neon-purple aesthetic, complete with a tiny retro pixel-art cat traversing the bottom of the screen. The text explains the Hebrew port of catfishing.net—a Wikipedia trivia game featuring AI-generated daily challenges and custom user-themed puzzles. Two neon-bordered buttons, "Play Daily Challenge" and "Create Custom Puzzle", point to `https://hatulbasak.geekip.dev`.

**07:00 AM: The Meta Project** 
The interface playfully creates a "Droste effect," showing a wireframe rendering of the very alarm clock the user is interacting with. The text humorously highlights the meta-nature of presenting the portfolio site on the portfolio site itself. A pulsing self-referential link directs the user back to the root domain `https://geekip.dev`. 

**08:00 AM: Social Network Connectivity** 
The final time slot transforms the display into a sleek, digital rolodex. Three large, minimalist vector icons dominate the space for GitHub, LinkedIn, and Facebook. Each icon glows brightly when hovered over, cleanly hyperlinking out to `github.com/gevak`, `linkedin.com/in/gevakip`, and `facebook.com/geva.kipper` respectively.

Integrated into the bottom bezel of the physical alarm clock design is a scrolling ticker tape made of tiny red LEDs. It reads: "Site architecture and design generated by an AI model." Right next to this credit is a live JavaScript-driven digital stopwatch. The stopwatch actively ticks in real-time, displaying the exact Hours:Minutes:Seconds that have elapsed since the precise timestamp of `2026-05-18 02:41:37`, cementing the theme of time mechanics and dynamic, ever-changing AI presence.
