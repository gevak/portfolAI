The design is built around a "tmux-multiplexer" or "ncurses" Text User Interface (TUI) aesthetic, simulating an advanced, high-tech retro CRT computer terminal. The entire single-page layout utilizes a deep black background (#050505) with glowing, high-contrast matrix-green (#00FF41) or amber monospace typography (like VT323 or Fira Code). CSS will be used to apply a subtle curvature to the "screen," a faint moving scanline overlay, and a text-shadow to mimic phosphor glow. Hover states across the site will invert colors (black text on a green/amber background) and trigger a blinking block cursor (`█`) next to the hovered element. 

The top of the screen serves as the "Boot Sequence" and "Message of the Day" (MOTD) header. It slowly types out (via JavaScript) "Geva Kipper" as a logged-in user or root directory. Directly beneath it, styled as a system process log, is the sentence: `> system.user.current_assignment: Building [REDACTED/AWESOME] software as an Engineer at Google.` Below this, bordered by warning symbols, is the subtitle: `[SYS_NOTICE]: This environment is re-compiled and re-designed daily by artificial intelligence. Structural volatility is expected.`

The core layout consists of a CSS Grid displaying the requested items as individual "Terminal Panes." Despite the prompt mentioning "six boxes," all eight items are structured into an asynchronous grid (e.g., a 4x2 or 3-3-2 layout on desktop, stacking to a single column on mobile). Each pane is bordered by ASCII box-drawing characters (┌─┐└─┘│) and features a title bar resembling a file path (e.g., `~/projects/glglstats.exe`).
