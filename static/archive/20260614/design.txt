The visual concept is a sleek, "Neo-Retro" arcade cabinet built using modern CSS 3D transforms, glassmorphism, and subtle viewport framing. The user is positioned as if standing right in front of a high-end, futuristic arcade machine. The background is a dark, ambient room with dynamic neon purple and cyan lighting that reflects off the cabinet's polished acrylic surfaces. The typography blends clean, modern sans-serif fonts for readability with glowing, retro-wave accents for headers and interactive elements. 

The top of the screen features the arcade's "Marquee." Wrapped in a glowing, backlit glass panel, the primary logo reads "GEVA KIPPER" in an elegant, modern neon script. Just beneath this, nestled in an LED dot-matrix style crawling text banner, is the developer’s bio: a single, punchy sentence detailing his current work at Google. Below this banner, a subtitle pulses gently like an active system warning: "Caution: This reality is re-designed day-by-day by AI and is ever-changing."

The central viewport is the "Screen" of the cabinet, styled with a subtle, elegant CRT monitor effect—featuring a faint, warm glow, imperceptible scanlines, and slightly curved glass borders that do not compromise readability. This screen functions as a "Select Your Game" interface. The layout is a dynamic, horizontal CSS scrolling snap-grid displaying eight glowing "Cartridges" or "Game Levels" that expand and illuminate via CSS hover animations, accompanied by custom "Start" buttons representing the external links.

The eight interactive project boxes are designed as individual arcade game splash screens:
**Box 1 (GlglStats):** Themed with a pixelated audio equalizer. It features the description outlining the LZ77 Israeli pop-song repetitiveness project. It contains two glowing, arcade-style buttons: "Play Game" (linking to https://glglstats.geekip.dev) and "View Walkthrough" (linking to the PyData TLV video at https://www.youtube.com/watch?v=jrKAtOsj1Lo).
**Box 2 (Ahuzat Dibuk):** Themed with a glowing thermal heatmap over a grid. It details the Python scraping and Tel-Aviv parking data visualization. Four interactive icons act as "Power-Ups," linking respectively to the TV coverage (https://13tv.co.il/item/news/domestic/internal/parking-902956407/), the Mako article (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), and the original Facebook reveal (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).
**Box 3 (2D Part Orienting):** Displays a slowly spinning, wireframe geometric polygon traversing a 3D grid. The text details the C++ computational geometry push-plan algorithm. An "Access Source Code" button links precisely to https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/.
**Box 4 (The Turing Jest):** Features split-screen pixel-art robot jesters locked in a boss-battle. The text outlines the LMSYS-like LLM pun arena, with an "Enter Arena" coin-slot button linking to https://jest.geekip.dev.
**Box 5 (Rhymes of Israel):** Showcases an animated, comedic high-school yearbook interacting with the news. The text explains the LLM-powered Chrome extension, accompanied by an "Install Mod" button linking to https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb.
**Box 6 (Hatul Basak):** Displays an AI-generated, trivia-themed cat avatar. The text describes the Wikipedia-based trivia port and user-generated puzzles. Two adjacent arcade buttons read "Daily Challenge" and "Create Puzzle," both linking to https://hatulbasak.geekip.dev.
**Box 7 (The Meta Portfolio):** An "infinite mirror" graphic, visually representing the recursion of a portfolio inside a portfolio. The text points out the meta nature of the build. It features an "Insert Coin to Restart" link pointing strictly to https://geekip.dev.
**Box 8 (High Scores / Leaderboard):** Designed like a classic arcade high-score screen containing the developer's socials. The top three "Initials" on the leaderboard function as direct links: github.com/gevak, linkedin.com/in/gevakip, and facebook.com/geva.kipper.

Below the main screen lies the digital "Coin Return" slot and control deck, serving as the website's footer. It consists of a sleek, brushed metal panel featuring a glowing engraving that reads: "Level Generated by ChatGPT AI Model." Beside this engraving is a digital, 7-segment red LED stopwatch written in vanilla JavaScript that calculates and permanently increments the exact hours, minutes, and seconds elapsed precisely since the timestamp 2026-06-14 02:57:09, emphasizing the shifting, time-bound nature of the AI generation theme.
