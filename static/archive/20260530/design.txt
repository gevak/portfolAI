**Overall Aesthetic and User Experience**
The website is a single-page, fullscreen immersive experience styled as a mystical, highly elegant tarot reading table. The color palette revolves around deep void-black, midnight blue, and shimmering gold accents. The typography pairs a classic, ornate serif (like Playfair Display) for headings to give an occult, ancient feel, with a clean, crisp geometric sans-serif (like Montserrat) for high readability in the technical body text. Subtle CSS particle animations create a backdrop of slowly drifting stardust. The central interactive mechanic is an 8-card tarot spread laid face-down. Users interact by hovering over the cards (causing them to levitate and glow) and clicking them (triggering a smooth 3D CSS flip animation that scales the card into the center of the screen as a modal).

**Header Section**
At the top center of the screen, an elegant gold-foil text reads **Geva Kipper**. Floating directly beneath it is a clean, modern sentence: *Software Engineer at Google, intertwining logic, algorithms, and code into digital reality.* Below this, a glowing, faintly pulsing subtitle establishes the site’s premise: *This digital realm is re-designed each day by AI, shifting like fate—ever-changing and never the same.*

**The Interactive Tarot Deck (The 8 Project Sections)**
The core interface features eight beautifully ornamented cards. The textured backs showcase intricate, shifting geometric mandalas. When clicked, the rest of the screen dims, and the card flips to reveal a custom "Major Arcana" artwork representing the project, accompanied by technical text and interactive link buttons styled as golden alchemical seals.

*   **Card I: The Loop (GlglStats)** 
    *Design:* The card artwork features a golden ouroboros made of musical staves. 
    *Content:* Details a Python project measuring the repetitiveness of Israeli pop-chart songs using LZ77 compression, with data visualizations of famous songs.
    *Links:* Two glowing buttons. One leading to the interactive oracle (https://glglstats.geekip.dev), and another displaying a play icon leading to the PyData TLV conference presentation (https://www.youtube.com/watch?v=jrKAtOsj1Lo).

*   **Card II: The Chariot (Ahuzat Dibuk)**
    *Design:* The card displays a celestial chariot towering over a luminous, glowing heatmap of a bustling city grid.
    *Content:* Explains the Python scraping and data visualization project that collects Tel-Aviv parking lot occupancy statistics to generate heat-maps for residents.
    *Links:* Three embossed wax-seal buttons linking to media coverage on Channel 13 (https://13tv.co.il/item/news/domestic/internal/parking-902956407/), Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), and the original viral Facebook publication (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).

*   **Card III: The Architect (2D Part Orienting)**
    *Design:* Shows shifting, golden geometric polygons floating into perfect alignment within a compass. 
    *Content:* Describes a C++ implementation in an open-source computational geometry library, detailing an efficient algorithm for constructing a 'push plan' to orient a given polygon.
    *Links:* A singular, sharp geometric button pointing to the code and documentation (https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/).

*   **Card IV: The Jester (The Turing Jest)**
    *Design:* Features two robotic theatrical masks—comedy and tragedy—debating across a chess board.
    *Content:* Presents an LMSYS-like arena where LLMs compete at making and explaining puns, rated by human judges.
    *Links:* A button styled as an arena ticket leading to the battleground (https://jest.geekip.dev).

*   **Card V: The Scribe (Rhymes of Israel)**
    *Design:* A sleek, futuristic metallic quill writing glowing rhymes over a dark parchment scroll of news text.
    *Content:* Introduces a Chrome extension that leverages an LLM to rewrite Israeli news headlines into comedic rhymes resembling highschool yearbook entries.
    *Links:* A Chrome-logo-inspired magical sigil to install the extension (https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb).

*   **Card VI: The Sphinx (Hatul Basak)**
    *Design:* An AI-infused feline sphinx guarding a complex, interconnected maze of Wikipedia nodes.
    *Content:* Presents a Hebrew port of catfishing.net—a Wikipedia-based Trivia game featuring daily AI-generated challenges and user-created themed puzzles.
    *Links:* A glowing portal button to play and create puzzles (https://hatulbasak.geekip.dev).

*   **Card VII: The Mirror (Portfolio Meta-Project)**
    *Design:* An intricate infinity mirror reflecting the exact layout of the website itself in endless recursion.
    *Content:* A brief, meta-referential nod to the architecture of this very portfolio.
    *Links:* An inward-pointing arrow linking back to the domain (https://geekip.dev).

*   **Card VIII: The Nexus (Socials)**
    *Design:* A constellation of three glowing stars connected by silver threads in the night sky.
    *Content:* "Connect with the creator."
    *Links:* Three minimalist, elegant icons replacing text links. A GitHub icon (github.com/gevak), a LinkedIn icon (linkedin.com/in/gevakip), and a Facebook icon (facebook.com/geva.kipper).

**Footer Design**
Positioned subtly at the bottom of the viewport is an area styled like an antique brass digital astrolabe. It features a finely engraved line of text crediting the site's creation: *Divined and designed by AI.* Below this sits a continuously ticking JavaScript chronometer styled as tumbling mechanical uncials, tracking exact time. It is accompanied by the text: *Time elapsed since the fated temporal shift (2026-05-30 02:31:36): [HH:MM:SS]* updating live every second.
