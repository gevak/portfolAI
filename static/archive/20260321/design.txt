**Overall Visual Concept & Architecture**
The site is a single-page immersive experience built on a "Z-axis Time Machine" mechanic using modern HTML, CSS, and JavaScript. Instead of traditional scrolling, the user interacts with a glowing, horizontal "Temporal Scrubber" anchored at the bottom of the screen. As the user drags the slider back and forth through "time," the background dynamically shifts ambient gradients, and the project boxes (designed as frosted glassmorphic portals) pull forward from the deep background along a 3D Z-axis, snapping into focus at the center of the viewport before zooming past and blurring out. The aesthetic is sleek, dark, and futuristic—think a blend of a high-tech Google operations dash and aesthetic sci-fi. 

**Header & Typography**
Anchored to the top-left is a minimalist, elegant header. The title reads "Geva Kipper" in a bold, modern sans-serif typeface with a subtle metallic sheen. Beneath it is the primary subtitle: "Currently engineering scalable solutions and shaping the future of tech at Google." Underneath, accompanied by a softly pulsing AI icon, is the secondary subtitle: "Temporal anomaly detected: This portfolio is completely re-imagined and re-designed every single day by Artificial Intelligence, ensuring an ever-changing digital footprint." 

**The Temporal Nodes (The 7 Boxes)**
The main content exists as seven distinct, interactive "holographic" cards that emerge as the user scrubs the timeline. When a card is active in the central viewport, it illuminates with a thematic color accent, and its interactive elements become clickable. 

*   **Node 1: GlglStats (The Compression Era)**
    Features an ambient background animation of pulsing soundwaves built from 1s and 0s. The typography details the Python project measuring the repetitiveness of Israeli pop-chart songs using LZ77 compression and visualizing repetition patterns. Include a glowing live-site button linking to https://glglstats.geekip.dev, and an embedded "play" button linking to the PyData TLV presentation on YouTube (https://www.youtube.com/watch?v=jrKAtOsj1Lo).
*   **Node 2: Ahuzat Dibuk (The Urban Data Epoch)**
    Thematically styled with a subtle background grid displaying shifting heat-maps. The text details the Python scraping and data visualization project tracking Tel-Aviv parking lot occupancy. To the right, include a cluster of distinct media badges linking to the press coverage: Channel 13 (https://13tv.co.il/item/news/domestic/internal/parking-902956407/), Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), and the original viral Secret Tel-Aviv Facebook post (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).
*   **Node 3: 2D Part Orienting (The Geometric Shift)**
    Features a wireframe 2D polygon slowly rotating and orienting itself in the background. The copy outlines the C++ implementation within an open-source computational geometry library, detailing the efficient algorithm for constructing push plans. Actionable text links directly to the documentation and code at https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/.
*   **Node 4: The Turing Jest (The Syntactic Comedy Phase)**
    Visually playful, incorporating subtle neon theater-mask motifs. It describes the LMSYS-like arena where LLMs battle over generating and explaining puns, subject to human rating. A prominent neon-bordered button links directly to the battleground: https://jest.geekip.dev.
*   **Node 5: Rhymes of Israel (The Satire Timeline)**
    Styled like a glowing, holographic high school yearbook page overlaid with newsprint. It introduces the Chrome extension utilizing an LLM to transform Israeli news headlines into comedic rhymes. A stylized "Add to Chrome" button links directly to the installation page at https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb.
*   **Node 6: The Infinite Loop (The Meta Paradox)**
    A visually trippy card utilizing CSS recursive mirror effects (an infinity mirror illusion). It cheekily presents the very portfolio website the user is currently viewing, acknowledging the meta-engineering of building an AI-designed portfolio mechanism. Include a self-referential link snapping back to https://geekip.dev.
*   **Node 7: Communication Array (Present Day)**
    The final destination on the timeline scrubber. This card acts as a sleek digital business card grid. It removes project descriptions in favor of three large, interactive, glowing icons for seamless networking: GitHub (github.com/gevak), LinkedIn (linkedin.com/in/gevakip), and Facebook (facebook.com/geva.kipper).

**Footer & the Temporal Chronometer**
Anchored to the absolute bottom right of the screen, mirroring the header's aesthetic, is the system readout. It clearly displays: "Design Generation Origin: AI Entity." Below this credit sits the "Temporal Disruptor Uptime"—a live JavaScript stopwatch ticking upward in real-time, displaying the exact Hours, Minutes, and Seconds that have elapsed since the precise timestamp of 2026-03-21 01:22:41, utilizing a monospace, digital-clock font to reinforce the time-machine immersion.
