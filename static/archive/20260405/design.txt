**Overall Aesthetic and Concept**
The website adopts a modern interpretation of a retro 16-bit / 8-bit arcade boxing game (akin to *Punch-Out!!*), merged with sleek, modern UI containers. The visual style uses a CRT scanline overlay, glowing neon UI elements against a dark, bruised-purple and pixelated gym background, and smooth parallax scrolling. The user’s cursor is styled as a red, pixelated boxing glove, which "punches" (animates a quick jab) when clicking. The overall narrative frames the user as a rookie boxer stepping into the ring, with the developer acting as the veteran coach teaching them how to KO software bugs.

**Hero Section: The Training Camp**
The top of the page features a large, dynamic pixel-art animation of a boxing gym. On the left side stands an 8-bit avatar of the developer wearing a coach's track jacket and a subtle Google "G" baseball cap, rhythmically blowing a whistle. Beside him is a dialogue box (styled like classic JRPG text bubbles) that types out: *"I'm Geva Kipper, Software Engineer at Google, engineering scalable solutions and wrestling planetary-scale data."* 
Directly below the text box is a blinking, arcade-style insert-coin subtitle indicating the site’s dynamic nature: *"WARNING: Arena layout procedurally re-designed daily by AI. Expect an ever-changing environment."* 
At the bottom center, an animated "Scroll Down to Enter the Ring" arrow flashes in neon yellow.

**Main Content Grid: The 6-Round Bout (Project Boxes)**
As the user scrolls down, the background transitions into a boxing ring under bright overhead pixel-art spotlights. The six projects are presented as a masonry grid of interactive "Bout Cards" or "Rounds." Each card looks like an arcade VS. screen, featuring a specific "Bug" the coach is teaching the user to defeat. When the user hovers over a box, the card gains a glowing animated border, and an 8-bit punch sound visual effect (like a "POW!" comic burst) appears.

*   **Round 1: The Repetition Slime (GlglStats)**
    The card illustrates an 8-bit battle against a slime monster made of musical notes. The text describes a Python project measuring the repetitiveness of Israeli pop-chart songs using LZ77 compression and visualizing the patterns of famous tracks. The card features two glowing, pill-shaped buttons: one directing to the live dashboard (https://glglstats.geekip.dev) and another styled as a retro VHS tape leading to the PyData TLV presentation (https://www.youtube.com/watch?v=jrKAtOsj1Lo).
*   **Round 2: The Gridlock Golem (Ahuzat Dibuk)**
    This box depicts a fight against a traffic-jam monster composed of stacked pixel-art cars. The text details a Python scraping and data visualization project generating heat-maps of Tel-Aviv parking lot occupancy. Links to media coverage sit below the description, represented by retro newspaper and television icons for Channel 13 (https://13tv.co.il/item/news/domestic/internal/parking-902956407/), Mako (https://www.mako.co.il/nexter-news/Article-ae43964b891bf71027.htm), and the original viral Secret Tel-Aviv Facebook post (https://www.facebook.com/groups/secrettelaviv/posts/10159636301195943/).
*   **Round 3: The Geometry Phantom (2D Part Orienting)**
    The artwork here involves a boss fight against a constantly shifting, misshapen polygon phantom. The project description highlights a C++ implementation within an open-source computational geometry library to construct "push plans" for orienting polygons. A blue glowing button, accompanied by a C++ logo sprite, links to the code and research (https://www.cgl.cs.tau.ac.il/projects/2d-part-orienting/).
*   **Round 4: The Jester Bot (The Turing Jest)**
    An LMSYS arena-style matchup is shown on the card, where two LLMs, depicted as rival boxing robots wearing court jester hats, fight each other. The text explains the project as an arena where LLMs compete to make and explain puns based on user ratings. A bold "Enter the Arena" button links to the live site (https://jest.geekip.dev).
*   **Round 5: The Yearbook Scribe (Rhymes of Israel)**
    The enemy here is an angry, animated newspaper that shoots letters. The description details a Chrome extension powered by an LLM that transforms standard Israeli news headlines into comedic, high-school-yearbook-style rhymes. An icon of an 8-bit Chrome logo points users directly to the Chrome Web Store installation page (https://chromewebstore.google.com/detail/eainginjgdkpgolpjniijdihfbeeeklb).
*   **Round 6: The Meta-Mirror (Portfolio Redesign)**
    The final boss card shows the boxer fighting a shadowy clone of themselves, representing the meta-nature of the project. The description notes the engineering behind the very site the user is exploring. It contains a self-referential link (https://geekip.dev) that, when hovered over, triggers a localized screen-distorting glitch effect CSS animation.

**Section 7: The Hall of Fame (Social and Contact Box)**
Placed below the project grid, this section takes the aesthetic of a retro arcade "High Score" screen. The title reads "HALL OF FAME." Instead of high scores, neon-tinted pixel typography spells out GitHub (github.com/gevak), LinkedIn (linkedin.com/in/gevakip), and Facebook (facebook.com/geva.kipper). Each text line reacts to hover by changing from white to flashing gold, accompanied by a small spinning coin sprite.

**Footer: Match Stats**
The bottom edge of the screen contains a dark, heavy border holding green, monospaced terminal text. On the left side, it reads: *"RING CONSTRUCTED BY Gemini (AI Model) - Chief Architect & Prompt Handler."* On the right side, there is a prominent red digital stopwatch that visually ticks up in real-time using continuous JavaScript calculation, formatted exactly as `UPTIME: [HH:MM:SS]`. The timestamp anchor for this running stopwatch calculation is permanently set to count upwards from `2026-04-05 01:53:10`.
